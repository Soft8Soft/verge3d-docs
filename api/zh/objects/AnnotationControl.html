[name]

AnnotationControl是一个负责注释移动的类。通常，您不需要直接使用这个类—要创建自定义注释，最好重写后代[page:Annotation Annotation]类。

构造函数

[name](container)

container - 将包含注释的 HTML 元素或 HTML 元素本身的 id。

属性

通用属性查看[page:AnnotationControl]基类。

[property:null container]

容器ID或HTML元素。

[property:Vector3 projected]

投影到注释坐标的向量。

[property:function updatedCallback]

注释更新回调。输入参数：投影的X和Y坐标，注释可见度标签。

[property:Boolean isAnnotationControl]

用于检查这个类或派生类是否是注释控制器。
不要更改此设置，因为它在内部用于优化。

方法

通用方法查看 [page:Object3D]基类。

[method:AnnotationControl clone]()

返回此 AnnotationControl 对象和任何后代的克隆。

[method:AnnotationControl update]([param:Camera camera])

更新注释控制器位置和可见性，如果提供，则执行 [page:AnnotationControl.updatedCallback updatedCallback]。此方法用于 [page:WebGLRenderer] 类内部。
