[name]

包含有用渲染方法的类。

方法

[method:WebGLCubeRenderTarget renderSceneToCubemap]([param:Scene scene], [param:WebGLRenderer renderer], [param:WebGLCubeRenderTarget cubeRenderTarget], [param:Object cameraOptions])

将给定的场景渲染成立方体贴图。参数是：
  • scene - 要渲染成立方体贴图的场景[page:Scene]
  • renderer - [page:WebGLRenderer]
  • cubeRenderTarget - [page:WebGLCubeRenderTarget] 包含要渲染到的立方体贴图的渲染目标
  • cameraOptions - 一个包含立方体相机选项的对象，用于将场景渲染到立方体贴图中；选项及其默认值是： { camPos: new Vector3(), clipStart: 0.1, clipEnd: 100, } camPos - 盒子相机的位置
    clipStart - 近处平面的值
    clipEnd - 远处平面的值

[method:WebGLCubeRenderTarget renderWorldNodeMatToCubemap]([param:WebGLRenderer renderer], [param:MeshNodeMaterial material], [param:Integer cubeRes], [param:Object options])

将世界节点材质渲染到立方体贴图。参数是：
  • renderer - [page:WebGLRenderer]
  • material - 世界节点材质，应是 [page:MeshNodeMaterial]类型
  • cubeRes - 渲染的立方体贴图的分辨率
  • （可选）保存自动生成的立方体渲染目标的纹理参数的对象；与 [page:WebGLCubeRenderTarget] 构造函数的 *options* 参数格式相同
生成的带有立方体纹理的[page:WebGLCubeRenderTarget render target]可以用作场景背景。
