[name]

此约束允许根据3D视口宽度、高度或方向设置显示/隐藏对象。

此约束用于通过影响对象的位置， 旋转和缩放来建立对象之间的各种关系。 可与 [page:Object3D.constraints Object3D.constraints] 属性共用。

构造函数

[name]()

Properties

参阅 [page:Constraint] 基类了解其公共属性。

[property:App appInstance]

指当前 应用 。 默认为 *undefined* 。

[property:Boolean isCanvasBreakpointsConstraint]

用于检查此类或派生类是否为画布断点约束。 默认为 *true* 。

[property:Number maxHeight]

对象保持可见的最大画布高度。 默认为 *Infinity* 。

[property:Number maxWidth]

对象保持可见的最大画布宽度。 默认为 *Infinity* 。

[property:Number minHeight]

对象保持可见的最小画布高度。 默认为 *0* 。

[property:Number minWidth]

对象保持可见的最小画布宽度。 默认为 *0* 。

[property:String orientation]

对象保持可见时的屏幕方向。 默认为 *"ALL"* 。

  • "ALL" - 全部
  • "LANDSCAPE" - 横向模式
  • "PORTRAIT" - 纵向模式

[property:String type]

约束类型。等同于 "CanvasBreakpointsConstraint" 。

方法

[method:Camera findAlternativeCamera]([param:Camera camera], [param:Scene scene], [param:Number width], [param:Number height])

camera — 场景中的当前摄影机
scene — 正在查找的摄影机所对应的场景
width — 画布宽度
height — 画布高度

返回场景中的备用摄影机（具有可接受的可见性断点），如果未找到备用摄像机，则返回 null。

参阅 [page:Constraint] 基类了解其常用方法。
