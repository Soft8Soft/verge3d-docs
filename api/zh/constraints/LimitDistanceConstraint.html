[name]

此约束强制被约束对象原理目标、更接近目标或敲好保持在给定距离处。

约束用于通过影响对象的位置， 旋转和缩放来建立对象之间的各种关系。 可与 [page:Object3D.constraints Object3D.constraints] 属性一起使用。

构造函数

[name]([page:Object3D target])

属性

请参阅基础 [page:TargetConstraint] 类了解公共属性。

[property:Float distance]

与目标的距离。默认为 *0.0* 。

[property:Float influence]

约束对对象的影响百分比。默认值为 *1.0* （100%）。

[property:String limitMode]

指定距离钳制模式。默认值为 *"LIMITDIST_ONSURFACE"* 。

"LIMITDIST_INSIDE"
被约束对象与目标保持更远的给定距离。
"LIMITDIST_OUTSIDE"
被约束对象保持更接近其目标的给定距离
"LIMITDIST_ONSURFACE"
被约束对象与目标保持确切的给定距离。

[property:Boolean isLimitDistanceConstraint]

用于检查此类或派生类是否为限制距离约束。默认值为 *true* 。

[property:String type]

约束的类型。等同于 "LimitDistanceConstraint" 。
