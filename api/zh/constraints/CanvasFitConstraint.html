[name]

此约束可将面板或文本附加到您期望的位置，无论屏幕分辨率或设备如何。必须将对象父级设置为摄影机。

此约束通过影响对象的位置、旋转 和缩放来建立对象之间的各种关系， 可与 [page:Object3D.constraints Object3D.constraints] 属性共用。

构造函数

[name]([page:Object3D target])

属性

参阅 [page:TargetConstraint] 基类了解其公共属性。

[property:Number edgeH]

需适应的水平画布边界。默认为 *0* 。
-1 — 左边界。
1 — 右边界。
0 — 无边界。

[property:Number edgeV]

需适应的画布垂直边界。默认为 *0* 。
-1 — 底边。
1 — 顶边。
0 — 无边界。

[property:Float offset]

画布适应偏移值，必须 >= 0 。 默认为 *0.0* 。

[property:Boolean isCanvasFitConstraint]

用于检查此类或派生类是否为画布拟合约束。 默认为 *true* 。

[property:String type]

约束类型。等同于 "CanvasFitConstraint" 。

[property:Boolean useBoundingBox]

画布适合形状。默认为 *true*.
false — 球体。
true — 立方体。
