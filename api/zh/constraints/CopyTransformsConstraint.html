[name]

此约束强制其被约束对象具有与其目标一致的变换。

此约束用于通过影响对象的位置， 旋转和缩放来建立对象之间的各种关系。 可与 [page:Object3D.constraints Object3D.constraints] 属性一起使用。

构造函数

[name]([page:Object3D target])

属性

请查阅 [page:TargetConstraint] 基类了解其公共属性。

[property:Boolean fixCameraLightRotation]

在Verge3D中，我们使用右手坐标系，Y轴指向向上方向，但Blender和3ds Max使用Z轴向上的右手坐标系(见 Verge3D wiki) 。因此，要校正摄影机和灯光的旋转，请设置 *fixCameraLightRotation = true* 。默认为 *false* 。

[property:Boolean isCopyTransformsConstraint]

用于检查此类或派生类是否为复制变换约束。默认值为 *true* 。

[property:Float influence]

约束对对象的影响百分比。默认值为 *1.0* （100%） 。

[property:String mixMode]

指定如何将新变换与现有变换组合在一起。默认为 *"REPLACE"* 。

"AFTER"
新变换将添加到现有变换之后。
"BEFORE"
新变换将添加到现有变换之前。
"REPLACE"
此变换值将替换现有值。

[property:String type]

约束的类型。 等同于 "CopyTransformsConstraint" 。
