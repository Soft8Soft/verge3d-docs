[name]

此约束限定了被约束对象的轴向始终指向一个方向。此约束使用一个纯粹的摆动旋转。例如，最短的可能的单轴旋转。在其他软件中，您可以找到名为 "look at" 的约束。 约束被用于通过影响对象的位置，旋转和缩放来建立对象之间的各种关系。可与 [page:Object3D.constraints Object3D.constraints] 属性一起使用。

构造函数

属性

参阅 [page:TargetConstraint] 基类了解其公共属性。

[property:Boolean fixCameraLightRotation]

在Verge3D中，我们使用右手坐标系，Y轴指向向上方向，但Blender和3ds Max使用Z轴向上的右手坐标系(见 Verge3D wiki) 。因此，要校正摄影机和灯光的旋转，请设置 *fixCameraLightRotation = true* 。默认为 *false* 。

[property:Float influence]

约束对对象的影响百分比。默认为 *1.0* (100%) 。

[property:Boolean isDampedTrackConstraint]

用于检查此类或派生类是否为阻尼跟踪约束。默认值为 *true* 。

[property:String trackAxis]

始终指向目标的轴。可以是"X", "Y", "Z", "-X", "-Y", "-Z" 。默认为 "X" 。

[property:String type]

约束的类型。等同于 "DampedTrackConstraint" 。
