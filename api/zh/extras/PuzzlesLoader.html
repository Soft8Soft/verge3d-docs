[name]

一个能够加载拼图编辑器和可视化逻辑代码的类。

构造函数

[name]()

属性

[property:String editorCSSFile]

已弃用。 相对于拼图的源目录(通常是SDK的"puzzles/"文件夹)，拼图编辑器的CSS文件的路径。 默认是 "puzzles.css"。

[property:String editorJSFile]

已弃用。 相对于拼图的源目录(通常是SDK的"puzzles/"文件夹)，拼图编辑器的JS文件的路径。 默认是 "puzzles.min.js"。

[property:String editorMediaDir]

已弃用。 相对于拼图的源目录(通常是SDK的"puzzles/"文件夹)，拼图编辑器的媒体文件的路径。默认是 "media/"。

[property:String libraryXML]

已弃用。 相对于拼图的源目录(通常是SDK的"puzzles/"文件夹)，拼图编辑器的库文件的路径。默认是"library.xml"。

方法

[method:null loadEditorWithLogic](editorDir, logicURL, onLoad, onError)

加载拼图编辑器和逻辑文件。此方法需要提供拼图编辑器的源文件夹的路径(通过editorDir参数指定，通常是SDK的"puzzles/"文件夹)和逻辑JS文件的路径(logicURL参数)。onLoad 回调可以在编辑器和逻辑文件被加载后加载一个场景。

[method:null loadLogic](logicURL, onLoad, onError)

加载逻辑文件。onLoad回调可以用于加载逻辑文件后加载场景。
