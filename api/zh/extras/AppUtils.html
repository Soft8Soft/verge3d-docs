[name]

一个包含3D应用程序实用函数的类。

方法

[method:Object getPageParams]

解析页获取参数并返回包含解析后的键值数据的对象。

[method:null loadScript](url, appendElem, callback)

url - 加载的脚本地址。
appendElem - 要将加载的脚本附加到的HTML元素。
callback - 加载事件侦听器。
加载和执行一个JS文件。

[method:null drawWatermark](elem, text)

elem -- 要添加水印的目标元素
text -- 要添加的消息
在应用程序中添加水印（版权和/或可能的信息消息）。
