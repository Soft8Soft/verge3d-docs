[name]

包含用于绘制线条几何体的特殊网格。旨在与 [page:MeshLineMaterial] 材质一起使用。能够从 [page:Geometry] 和 Array/Float32Array 数据创建几何体，对于 [page:BufferGeometry] 使用 [page:MeshLineIndexed] 代替。

构造函数

[name]()

属性

[property:BufferGeometry geometry]

要传递到 [page:Mesh] 构造函数的线条几何体。在调用 [page:.setGeometry setGeometry] 方法后创建。

方法

[method:null setGeometry](geometry, widthCallback)

从指定的 [page:Geometry] 或 Array/Float32Array 数据创建线条几何体。 widthCallback 可用于控制沿线每个点的宽度。它的参数 p 沿着这条线从 0 到 1 变化： line.setGeometry(geometry, function(p) { return p; });
