[name]

Ограничение, которое заставляет ограниченный объект отслеживать цель, а также фиксирует оси объекта.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]([page:Object3D target])

Свойства

Общие свойства смотрите в базовом классе [page:TargetConstraint].

[property:Boolean isLockedTrackConstraint]

Используется для проверки того, является ли данный или производные классы заблокированными ограничениями пути. По умолчанию *true*.

[property:String lockAxis]

Ось ограниченного объекта, которая заблокирована. Может быть одной из "X", "Y", "Z". По умолчанию - "Y".

[property:String trackAxis]

Ось ограниченного объекта, которая отслеживает цель. Может быть одной из "X", "-X", "Y", "-Y", "Z", "-Z". По умолчанию - "X".

[property:String type]

Тип ограничения. Равен "LockedTrackConstraint".
