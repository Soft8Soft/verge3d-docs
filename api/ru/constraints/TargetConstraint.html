[name]

Абстрактный базовый класс для ограничений, которые зависят от целевого объекта.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]([page:Object3D target])

Это создает общее целевое ограничение.

Свойства

Общие свойства смотрите в базовом классе [page:Constraint].

[property:Boolean isTargetConstraint]

Используется для проверки того, является ли данный или производные классы целевыми ограничениями. По умолчанию *true*.

[property:Object3D target]

Цель, от которой зависит данное ограничение. Должно быть [page:Object3D].

[property:String type]

Тип ограничения. Равен "TargetConstraint".
