[name]

Ограничение, которое ограничивает положение ограниченного объекта в пространстве его родителя.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints]..

Конструктор

[name]()

Свойства

Общие свойства смотрите в базовом классе [page:Constraint].

[property:Boolean isLimitLocationConstraint]

Используется для проверки того, являются ли данный или производные классы ограничениями местоположения. По умолчанию *true*.

[property:Vector3 max]

[page:Vector3], содержащий максимальные значения по соответствующим осям для ограничения положения ограниченного объекта. По умолчанию - Vector3(Infinity, Infinity, Infinity).

[property:Vector3 min]

[page:Vector3], содержащий минимальные значения по соответствующим осям для ограничения положения ограниченного объекта. По умолчанию - Vector3(-Infinity, -Infinity, -Infinity).

[property:String type]

Тип ограничения. Равен "LimitLocationConstraint".
