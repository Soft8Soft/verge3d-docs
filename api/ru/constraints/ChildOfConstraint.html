[name]

Ограничение, которое заставляет ограниченный объект вести себя как дочерний объект целевого без фактического родительства.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]([page:Object3D target])

Свойства

Общие свойства смотрите в базовом классе [page:TargetConstraint].

[property:Boolean isChildOfConstraint]

Используется для проверки того, являются ли данный или производные классы дочерними по отношению к ограничениям childOf. По умолчанию *true*.

[property:Matrix4 offsetMatrix]

Матрица смещения для ограниченного объекта. Применяется в родительском пространстве. По умолчанию - матрица тождества.

[property:String type]

Тип ограничения. Равен "ChildOfConstraint".
