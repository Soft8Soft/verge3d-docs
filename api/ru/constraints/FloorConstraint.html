[name]

Ограничение, которое ограничивает положение ограничиваемого объекта на основе положения цели. Таким образом, целевой объект ведет себя как "inivisible wall", "floor" и т.д....

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]([page:Object3D target])

Свойства

Общие свойства смотрите в базовом классе [page:TargetConstraint].

[property:String floorLocation]

Ось, вдоль которой будет применяться ограничение. Может быть одной из "X", "-X", "Y", "-Y", "Z", "-Z". Положительное значение означает, что положение ограниченного объекта вдоль этой оси может быть только выше значения [page:.offset]. Отрицательное означает - только под значением [page:.offset]. По умолчанию - "Y".

[property:Boolean isFloorConstraint]

Используется для проверки того, являются ли данный или производные классы ограничениями пола. По умолчанию *true*.

[property:Number offset]

Предельное значение по оси [page:.floorLocation]. По умолчанию равно 0.

[property:String type]

Тип ограничения. Равен "FloorConstraint".
