[name]

AnimationMixer - это проигрыватель для анимации определенного объекта в сцене. Когда несколько объектов в сцене анимируются независимо друг от друга, для каждого объекта может использоваться один AnimationMixer.

Обзор различных элементов системы анимации Verge3D смотрите в статье "Animation System" в разделе "Next Steps" руководства.

Конструктор

[name]([param:Object3D rootObject])

[page:Object3D rootObject] - объект, анимация которого будет воспроизводиться этим микшером.

Свойства

[property:Number time]

Глобальное время микшера (в секундах; начинается с 0 при создании микшера).

[property:Number timeScale]

Масштабный коэффициент для глобального [page:.time mixer time].

Примечание: Установка TimeScale микшера на 0, а затем обратно на 1 - это возможность приостановить/отменить паузу все действия, управляемые этим микшером.

Методы

[method:AnimationAction clipAction]([param:AnimationClip clip], [param:Object3D optionalRoot])

Возвращает [page:AnimationAction] для переданного клипа, по желанию используя корневой объект, отличный от корневого объекта микшера по умолчанию. Первым параметром может быть либо объект [page:AnimationClip], либо имя AnimationClip.

Если действие, соответствующее параметрам клипа и корня, еще не существует, оно будет создано этим методом. Вызов этого метода несколько раз с одними и теми же параметрами клипа и корня всегда возвращает один и тот же экземпляр клипа.

[method:AnimationAction existingAction]([param:AnimationClip clip], [param:Object3D optionalRoot])

Возвращает существующий [page:AnimationAction] для переданного клипа, по желанию используя корневой объект, отличный от корневого объекта микшера по умолчанию.

Первым параметром может быть либо объект [page:AnimationClip], либо имя AnimationClip.

[method:Object3D getRoot]()

Возвращает корневой объект этого микшера.

[method:AnimationMixer stopAllAction]()

Останавливает все ранее запланированные действия на этом микшере.

[method:AnimationMixer update]([param:Number deltaTimeInSeconds])

Продвигает глобальное время микшера и обновляет анимацию.

Обычно это делается в цикле рендеринга, передавая [page:Clock.getDelta clock.getDelta], масштабированную по [page:.timeScale timeScale] микшера).

[method:AnimationMixer setTime]([param:Number timeInSeconds])

Устанавливает глобальный микшер на определенное время и соответствующим образом обновляет анимацию.

Это полезно, когда нужно перейти к точному времени в анимации. Входной параметр будет масштабирован по [page:.timeScale timeScale] микшера.

[method:null uncacheClip]([param:AnimationClip clip])

Распределяет все ресурсы памяти для клипа.

[method:null uncacheRoot]([param:Object3D root])

Распределяет все ресурсы памяти для корневого объекта.

[method:null uncacheAction]([param:AnimationClip clip], [param:Object3D optionalRoot])

Распределяет все ресурсы памяти для действия.
