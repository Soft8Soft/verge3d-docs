[name]

Дорожка значений ключевых кадров, которые представляют изменения цвета.

В самой базовой реализации этого подкласса пока нет ничего особенного. Однако, это место для параметризации цветового пространства.

Конструктор

[name]([param:String name], [param:Array times], [param:Array values])

[page:String name] — (требуется) идентификатор для KeyframeTrack.
[page:Array times] — (требуется) массив времен ключевых кадров.
[page:Array values] — значения для ключевых кадров в указанное время.
[page:Constant interpolation] — тип используемой интерполяции. Смотрите [page:Animation Animation Constants] для возможных значений. По умолчанию [page:Animation InterpolateLinear].
