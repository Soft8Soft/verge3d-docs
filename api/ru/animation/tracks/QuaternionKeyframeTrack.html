[name]

Дорожка значений ключевых кадров кватерниона.

Конструктор

[name]([param:String name], [param:Array times], [param:Array values])

[page:String name] (требуется) идентификатор для KeyframeTrack.
[page:Array times] (требуется) массив времен ключевых кадров.
[page:Array values] значения для ключевых кадров в указанное время.
[page:Constant interpolation] тип интерполяции для использования. Смотрите [page:Animation Animation Constants] для возможных значений. По умолчанию [page:Animation InterpolateLinear].

Свойства

Унаследованные свойства смотрите в [page:KeyframeTrack].

[property:Constant DefaultInterpolation]

Тип интерполяции, используемый по умолчанию, [page:Animation InterpolateLinear].

[property:String ValueTypeName]

Строка 'quaternion'.

Методы

Унаследованные методы смотрите в [page:KeyframeTrack].

[method:null InterpolantFactoryMethodLinear]()

Возвращает новый [page:QuaternionLinearInterpolant QuaternionLinearInterpolant] на основе [page:KeyframeTrack.values values], [page:KeyframeTrack.times times] и [page:KeyframeTrack.valueSize valueSize] ключевых кадров.
