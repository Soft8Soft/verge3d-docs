[name]

Объект с различными функциями для помощи в анимации, используемый внутри.

Methods

[method:Array arraySlice](array, from, to)

То же самое, что и Array.prototype.slice, но также работает с типизированными массивами.

[method:Array convertArray](array, type, forceClone)

Преобразует массив к определенному типу.

[method:Array flattenJSON](jsonKeys, times, values, valuePropertyName )

Используется для парсинга форматов ключевых кадров AOS.

[method:Array getKeyframeOrder](times)

Возвращает массив, по которому могут быть отсортированы времена и значения.

[method:Boolean isTypedArray](object)

Возвращает *true*, если объект является типизированным массивом.

[method:Array sortedArray](values, stride, order)

Сортирует массив, ранее возвращенный [page:AnimationUtils.getKeyframeOrder getKeyframeOrder].

[method:AnimationClip subclip]([param:AnimationClip clip], [param:String name], [param:Number startFrame], [param:Number endFrame], [param:Number fps])

Создает новый клип, содержащий только сегмент исходного клипа между заданными кадрами.
