[name]

Класс, способный загружать редактор пазлов и код визуальной логики.

Конструктор

[name]()

Конструктор не принимает параметров.

Методы

[method:null loadEditorWithLogic](editorDir, logicURL, onLoad, onError)

Загружает редактор пазлов и файл логики. Методу необходимо указать путь к исходной папке редактора пазлов (задается параметром `editorDir`, обычно это подпапка puzzles установчной папки «Вердж3Д») и путь к JS-файлу логики (параметр `logicURL`). Обработчик `onLoad` можно использовать для загрузки сцены после того, как редактор и файл логики были загружены.

[method:null loadLogic](logicURL, onLoad, onError)

Загружает файл логики. Обработчик `onLoad` можно использовать для загрузки сцены после загрузки файла логики.
