[name]

Специальный меш, содержащий геометрию для рисования линий. Предназначен для использования с материалом [page:MeshLineMaterial]. Может создавать геометрию из данных Array/Float32Array, для [page:BufferGeometry] используйте [page:MeshLineIndexed].

Конструктор

[name]()

Свойства

[property:BufferGeometry geometry]

Геометрия линии для передачи в конструктор [page:Mesh]. Создается после вызова метода [page:.setGeometry setGeometry].

Методы

[method:null setGeometry](geometry, widthCallback)

Постройте геометрию линии из данных Array/Float32Array. widthCallback может быть использован для управления шириной в каждой точке вдоль линии. Его параметр p изменяется от 0 до 1 вдоль линии: line.setGeometry(geometry, function(p) { return p; });
