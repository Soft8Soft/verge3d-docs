[name]

Этот класс содержит ряд функций, которые могут быть полезны при создании 3Д-приложений.

Методы

[method:undefined addToAppList]([param:App app])

Добавляет `app` в глобальный массив `v3d.apps` array. Обычно этот метод вызывается в конструкторе [page:App App].

[method:Object getPageParams]

Разбирает GET-параметры страницы и возвращает объект с разобранными данными ключ-значение.

[method:Boolean isJS]([param:String url])

Проверяет, является ли параметр `url` «Яваскрипт»-файлом.

[method:Boolean isXML]([param:String url])

Проверяет, является ли параметр `url` XML-файлом.

[method:null loadScript](url, appendElem, callback)

  • url — URL загруженного скрипта.
  • appendElem — HTML-элемент для добавления загруженного сценария.
  • callback — обработчик события загрузки.

Загружает и выполняет заданный «Яваскрипт»-файл.

[method:undefined printPerformanceInfo]([param:App scope], [param:Float delta])

Собирает и выводит профиль производительности. Смотрите [page:App.printPerformanceInfo] для получения подробной инфомрации.

[method:undefined registerServiceKeys]([param:App app])

Регистрирует сервисные клавиатурные комбинации (такие как тройная тильда) для запущенного приложения.

[method:undefined removeFromAppList]([param:App app])

Удаляет `app` из глобального массива `v3d.apps`. Обычно вызывается методом [page:App.dispose].

[method:undefined unregisterServiceKeys]([param:App app])

Удаляет сервисные клавиатурные комбинации для запущенного приложения

[method:String updatePageParam]([param:String url], [param:String param], [param:Any paramVal])

  • [page:String url] — входной УРЛ.
  • [page:String param] — имя параметра.
  • [page:Any paramVal] — значение параметра.

Обновляет GET-параметр `url` и возвращает обновлённый УРЛ-адрес.
