Реализует простой, но настраиваемый прелоадер. Смотрите стандартные шаблоны приложений «Вердж3Д» чтобы увидеть как он работает.

Конструктор

[name]([param:Object parameters])

[page:Object parameters] — объект со свойствами, определяющими поведение прелоадера.

Допустимо использовать только один параметр:

[page:String container] — ID элемента или сам элемент, используемый для хранения создаваемого прелоадера.

Properties

[property:HTMLElement bar]

Элемент <div>, используемый для отображения полосы прелоадера.

[property:HTMLElement logo]

Элемент <div>, используемый для отображения логотипа прелоадера.

[property:HTMLElement logoCont]

Элемент <div>, используемый как контейнер для логотипа прелоадера.

[property:HTMLElement background]

Элемент <div>, используемый для фона прелоадера.

[property:HTMLElement container]

Контейнерный элемент, переданный в конструкторе.

[property:Clock clock]

Экземпляр класса [page:Clock clock]. Предназначен для внутреннего использования.

Методы

Простой прелоадер переопределяет методы [page:Preloader Preloader], реализующие анимацию изображения и логику очистки.

Customization

Обратитесь к соответствующему руководству за инструкциями по настройке прелоадера с помощью CSS.
