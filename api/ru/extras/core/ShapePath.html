[name]

Этот класс используется для преобразования серии фигур в массив [page:Path], например, SVG-фигуры в контур (смотрите пример ниже). Он используется внутри [page:Font] для преобразования шрифта в формате JSON в серию контуров.

Конструктор

[name]()

Создает новый ShapePath. В отличие от [page:Path], никакие точки не передаются, поскольку ShapePath предназначен для генерации после создания.

Совойства

[property:Array subPaths]

Массив [page:Path]s.

[property:Array currentPath]

Текущий [page:Path], который генерируется.

[property:Color color]

[page:Color] фигуры, по умолчанию установлен на белый цвет (0xffffffff).

Методы

[method:this moveTo]([param:Float x], [param:Float y])

Создает новую [page:Path] и вызывает [page:Path.moveTo](x, y) на этой [page:Path]. Также указывает [page:ShapePath.currentPath currentPath] на эту [page:Path].

[method:this lineTo]([param:Float x], [param:Float y])

Это создает линию от смещения [page:ShapePath.currentPath currentPath] до X и Y и обновляет смещение до X и Y.

[method:this quadraticCurveTo]([param:Float cpX], [param:Float cpY], [param:Float x], [param:Float y])

Это создает квадратичную кривую от смещения [page:ShapePath.currentPath currentPath] к x и y с cpX и cpY в качестве контрольных точек и обновляет смещение [page:ShapePath.currentPath currentPath] к x и y.

[method:this bezierCurveTo]([param:Float cp1X], [param:Float cp1Y], [param:Float cp2X], [param:Float cp2Y], [param:Float x], [param:Float y])

Это создает кривую безье от смещения [page:ShapePath.currentPath currentPath] по x и y с cp1X, cp1Y и cp1X, cp1Y в качестве контрольных точек и обновляет смещение [page:ShapePath.currentPath currentPath currentPath] по x и y.

[method:this splineThru] ([param:Array points])

points - массив [page:Vector2]s

Подключает новую [page:SplineCurve] к [page:ShapePath.currentPath currentPath].

[method:Array toShapes]([param:Boolean isCCW], [param:Boolean noHoles])

isCCW — Изменяет способ создания твердых тел и отверстий
noHoles — Генерировать или не генерировать отверстия

Преобразует массив [page:ShapePath.subPaths subPaths] в массив Shapes. По умолчанию сплошные фигуры определяются по часовой стрелке (CW), а отверстия - против часовой стрелки (CCW). Если параметр isCCW имеет значение true, то они будут перевернуты. Если параметр noHoles имеет значение true, то все пути будут заданы как сплошные фигуры и isCCW будет игнорироваться.
