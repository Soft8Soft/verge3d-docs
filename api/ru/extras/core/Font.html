[name]

Создаёт набор [page:Shape Shapes], представляющий шрифт, загруженный в формате JSON.

Используется внутри [page:FontLoader].

Конструктор

[name](data)

data — Данные JSON, представляющие шрифт.

Этот конструктор создает новый [name], который представляет собой массив [page:Shape Shapes].

Свойства

[property:Array data]

Данные JSON, переданные в конструкторе.

Методы

[method:null generateShapes]([param:String text], [param:Float size])

[page:String text] — строка текста.
[page:Float size] — (необязательно) масштаб для [page:Shape Shapes]. По умолчанию *100*.
Создает массив [page:Shape Shapes], представляющий текст в шрифте.
