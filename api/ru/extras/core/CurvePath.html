[name]

Абстрактный базовый класс, расширяющий [page:Curve]. CurvePath - это просто массив связанных кривых, но сохраняет api кривой.

Конструктор

[name]()

Конструктор не принимает никаких параметров.

Свойства

Общие свойства смотрите в базовом классе [page:Curve].

[property:Array curves]

Массив [page:Curve Curves].

[property:Boolean autoClose]

Следует ли автоматически закрывать путь или нет.

Методы

Общие методы смотрите в базовом классе [page:Curve].

[method:null add]([param:Curve curve])

Добавьте кривую в массив [page:.curves].

[method:null closePath]()

Добавляет [page:LineCurve lineCurve] для закрытия пути.

[method:Array getCurveLengths]()

Получить список кумулятивных длин кривых в массиве [page:.curves].

[method:Vector getPoint]([param:Float t])

[page:Float t] -Положение на кривой. Должно находиться в диапазоне [0, 1]..

Возвращает вектор для заданной позиции на траектории кривой.

[method:Array getPoints]([param:Integer divisions])

divisions — количество частей, на которые нужно разделить кривую. По умолчанию *12*.

Возвращает массив точек, представляющих последовательность кривых. Параметр *division* определяет количество частей, на которые делится каждая кривая. Однако, в целях оптимизации и качества, фактическое разрешение выборки для каждой кривой зависит от ее типа. Например, для [page:LineCurve] возвращаемое количество точек всегда равно 2.

[method:Array getSpacedPoints]([param:Integer divisions])

divisions — количество частей, на которые нужно разделить кривую. По умолчанию *40*.

Возвращает набор делений + 1 равноотстоящих друг от друга точек с помощью getPointAt(u).
