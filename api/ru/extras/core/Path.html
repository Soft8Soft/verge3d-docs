[name]

Двумерное представление пути. Класс предоставляет методы для создания путей и контуров двумерных фигур, аналогичные API 2D Canvas.

Пример Кода

const path = new v3d.Path(); path.lineTo(0, 0.8); path.quadraticCurveTo(0, 1, 0.2, 1); path.lineTo(1, 1); const points = path.getPoints(); const geometry = new v3d.BufferGeometry().setFromPoints(points); const material = new v3d.LineBasicMaterial({ color: 0xffffff }); const line = new v3d.Line(geometry, material); scene.add(line);

Конструктор

[name]([param:Array points])

points -- (необязательный) массив [page:Vector2 Vector2s].

Создает контур из точек. Первая точка определяет смещение, затем последующие точки добавляются в массив [page:CurvePath.curves curves] как [page:LineCurve LineCurves].

Если точки не указаны, создается пустой контур, а [page:.currentPoint] устанавливается в начало координат.

Свойства

Общие свойства смотрите в базовом классе [page:CurvePath].

[property:Array currentPoint]

Текущее смещение контура. Любая новая добавленная [page:Curve] будет начинаться здесь.

Методы

Общие методы смотрите в базовом классе [page:CurvePath].

[method:this absarc]([param:Float x], [param:Float y], [param:Float radius], [param:Float startAngle], [param:Float endAngle], [param:Boolean clockwise])

x, y -- Абсолютный центр дуги.
radius -- Радиус дуги.
startAngle -- Начальный угол в радианах.
endAngle -- Конечный угол в радианах.
clockwise -- Развернуть дугу по часовой стрелке. По умолчанию *false*.

Добавляет абсолютно позиционированную [page:EllipseCurve EllipseCurve] к пути.

x, y -- Абсолютный центр эллипса.
xRadius -- Радиус эллипса по оси x.
yRadius -- Радиус эллипса по оси y.
startAngle -- Начальный угол в радианах.
endAngle -- Конечный угол в радианах.
clockwise -- Развернуть эллипс по часовой стрелке. По умолчанию имеет значение false.
rotation -- Угол поворота эллипса в радианах, против часовой стрелки от положительной оси X. Необязателен, по умолчанию равен 0.

[method:this lineTo]([param:Float x], [param:Float y])

Соединяет [page:LineCurve] из [page:.currentPoint] с x, y на пути.

[method:this moveTo]([param:Float x], [param:Float y])

Переместите [page:.currentPoint] к x, y.

[method:this quadraticCurveTo]([param:Float cpX], [param:Float cpY], [param:Float x], [param:Float y])

Создает квадратичную кривую из [page:.currentPoint] с cpX и cpY в качестве контрольных точек и обновляет [page:.currentPoint] до x и y.

[method:this setFromPoints]([param:Array vector2s])

points - массив [page:Vector2 Vector2s]
