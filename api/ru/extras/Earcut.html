[name]

Реализация алгоритма триангуляции полигонов. Код представляет собой порт [link:https://github.com/mapbox/earcut mapbox/earcut].

Методы

[method:Array triangulate](data, holeIndices, dim)

data — Плоский массив координат вершин.
holeIndices — Массив индексов отверстий, если таковые имеются.
dim — Количество координат на вершину во входном массиве.

Триангулирует заданное определение фигуры, возвращая массив треугольников. Треугольник определяется тремя последовательными целыми числами, представляющими индексы вершин.
