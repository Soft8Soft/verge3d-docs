[name]

Создает двухмерную кривую в форме эллипса. При установке [page:Number xRadius] равным [page:Number yRadius] получится круг.

Пример

const curve = new v3d.EllipseCurve( 0, 0, // ax, aY 10, 10, // xRadius, yRadius 0, 2 * Math.PI, // aStartAngle, aEndAngle false, // aClockwise 0 // aRotation ); const points = curve.getPoints(50); const geometry = new v3d.BufferGeometry().setFromPoints(points); const material = new v3d.LineBasicMaterial({ color : 0xff0000 }); // Create the final object to add to the scene const ellipse = new v3d.Line(geometry, material);

Конструктор

[name]([param:Float aX], [param:Float aY], [param:Float xRadius], [param:Float yRadius], [param:Radians aStartAngle], [param:Radians aEndAngle], [param:Boolean aClockwise], [param:Radians aRotation])

[page:Float aX] – X центр эллипса. По умолчанию *0*.
[page:Float aY] – Y центр эллипса. По умолчанию *0*.
[page:Float xRadius] – Радиус эллипса в направлении x. По умолчанию *1*.
[page:Float yRadius] – Радиус эллипса в направлении y. По умолчанию *1*.
[page:Radians aStartAngle] – Начальный угол кривой в радианах, начиная с положительной оси X. По умолчанию *0*.
[page:Radians aEndAngle] – Конечный угол кривой в радианах, начиная с положительной оси X. По умолчанию *2 x Math.PI*.
[page:Boolean aClockwise] – Будет ли эллипс рисоваться по часовой стрелке. По умолчанию *false*.
[page:Radians aRotation] – Угол поворота эллипса в радианах, против часовой стрелки от положительной оси X (необязательно). По умолчанию *0*.
