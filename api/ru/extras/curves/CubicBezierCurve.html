[name]

Create a smooth 2d cubic bezier curve, defined by a start point, endpoint and two control points.

Example

var curve = new v3d.CubicBezierCurve( new v3d.Vector2(-10, 0), new v3d.Vector2(-5, 15), new v3d.Vector2(20, 15), new v3d.Vector2(10, 0) ); var points = curve.getPoints(50); var geometry = new v3d.BufferGeometry().setFromPoints(points); var material = new v3d.LineBasicMaterial({ color : 0xff0000 }); // Create the final object to add to the scene var curveObject = new v3d.Line(geometry, material);
