[name]

Специальный меш, содержащий геометрию для рисования линий. Предназначен для использования с материалом [page:MeshLineMaterial]. Может создавать геометрию из [page:BufferGeometry]. Для данных Array/Float32Array используйте [page:MeshLine].

Конструктор

[name]()

Свойства

[property:BufferGeometry geometry]

Геометрия линии для передачи в конструктор [page:Mesh]. Создается после вызова метода [page:.fromBufferGeometry fromBufferGeometry].

Методы

[method:null fromBufferGeometry]([page:BufferGeometry geometry])

Постройте геометрию линии из данных [page:BufferGeometry].
