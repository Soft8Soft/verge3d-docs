[name]

Серия линий, проведенных между парами вершин.

Это почти то же самое, что [page:Line]; разница лишь в том, что она отрендерена с использованием [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/drawElements gl.LINES] вместо [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/drawElements gl.LINE_STRIP].

Конструктор

[name]([param:Geometry geometry], [param:Material material])

[page:Geometry geometry] — Пара(ы) вершин, представляющих каждый сегмент(ы) линии.
[page:Material material] — Материал для линии. По умолчанию это [page:LineBasicMaterial LineBasicMaterial].
