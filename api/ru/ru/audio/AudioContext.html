[name]

Здесь содержатся методы для настройки [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext].

Используется внутри классов [page:AudioListener AudioListener] и [page:AudioLoader AudioLoader].

Использует [link:https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API Web Audio API].

Методы

[method:AudioContext getContext]()

Возвращает значение переменной *context* во внешней области видимости, если она определена, иначе установите ее в новый [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext].

[method:AudioContext setContext]([param:AudioContext value])

Установите переменную *context* во внешней области видимости в значение *value*.
