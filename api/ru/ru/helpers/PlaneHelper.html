[name]

Объект-помощник для визуализации [page:Plane].

Пример Кода

const plane = new v3d.Plane(new v3d.Vector3(1, 1, 0.2), 3); const helper = new v3d.PlaneHelper(plane, 1, 0xffff00); scene.add(helper);

Конструктор

[name]([param:Plane plane], [param:Float size], [param:Color hex])

[page:Plane plane] -- плоскость для визуализации.
[page:Float size] -- (опционально) длина стороны вспомогательной плоскости. По умолчанию равна 1.
[page:Color color] -- (опционально) цвет помощника. По умолчанию 0xffff00.

Создает новую репрезентацию сетки переданной плоскости.

Свойства

Общие свойства см. в базовом классе [page:LineSegments].

[property:Plane plane]

Визуализируемая [page:Plane plane].

[property:Float size]

Длина стороны плоскости помощника.

Методы

Общие методы см. в базовом классе [page:LineSegments].

[method:void updateMatrixWorld]([param:Boolean force])

Этот метод переопределяет метод в базовом классе [page:Object3D], так что он также обновляет объект-помощник в соответствии со свойствами [page:PlaneHelper.plane .plane] и [page:PlaneHelper.size .size].
