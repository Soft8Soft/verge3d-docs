[page:Object3D] → [page:Line] → [page:LineSegments] →

[name]

Помогает визуализировать то, что камера содержит в фруструме.
Он визуализирует фрустум камеры с помощью [page:LineSegments]..

Пример Кода

const camera = new v3d.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000); const helper = new v3d.CameraHelper(camera); scene.add(helper);

Примеры

[example:webgl_camera WebGL / camera]
[example:webgl_geometry_extrude_splines WebGL / extrude / splines]

Конструктор

[name]([param:Camera camera])

[page:Camera camera] -- Камера для визуализации.

Это создаст новый [name] для указанной камеры.

Свойства

Общие свойства см. в базовом классе [page:LineSegments].

[property:Camera camera]

Визуализируемая камера.

[property:Object pointMap]

Здесь содержатся точки, используемые для визуализации камеры.

[property:Object matrix]

Референс на [page:Object3D.matrixWorld camera.matrixWorld].

[property:Object matrixAutoUpdate]

См. [page:Object3D.matrixAutoUpdate]. Здесь установлено значение *false*, так как помощник использует [page:Object3D.matrixWorld matrixWorld] камеры.

Методы

Общие методы см. в базовом классе [page:LineSegments].

[method:null update]()

Обновляет помощник на основе projectionMatrix камеры.
