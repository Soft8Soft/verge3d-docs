[name]

Объект-помощник для визуализации [page:Box3].

Пример Кода

const box = new v3d.Box3(); box.setFromCenterAndSize(new v3d.Vector3(1, 1, 1), new v3d.Vector3(2, 1, 3)); const helper = new v3d.Box3Helper(box, 0xffff00); scene.add(helper);

Конструктор

[name]([param:Box3 box], [param:Color color])

[page:Box3 box] -- Box3 для отображения.
[page:Color color] -- (опционально) цвет бокса. По умолчанию 0xffff00.

Создает новый вайрфреймовый бокс, представляющий заданный Box3.

Свойства

Общие свойства см. в базовом классе [page:LineSegments].

[property:Box3 box]

Визуализируемый Box3.

Методы

Общие методы см. в базовом классе [page:LineSegments].

[method:void updateMatrixWorld]([param:Boolean force])

Этот метод переопределяет метод в базовом классе [page:Object3D], так что он также обновляет рамку каркаса в пределах свойства [page:Box3Helper.box .box].
