[name]

Класс, представляющий четырехмерный [link:https://en.wikipedia.org/wiki/Vector_space вектор]. Четырехмерный вектор - это упорядоченная четверка чисел (обозначенных x, y, z и w), которая может быть использована для представления ряда вещей, таких как:

  • Точка в четырехмерном пространстве.
  • Направление и длина в четырехмерном пространстве. В Verge3D длина всегда будет равна [link:https://en.wikipedia.org/wiki/Euclidean_distance Евклидово расстояние] (расстояние по прямой) от (0, 0, 0, 0) до (x, y, z, w), а направление также измеряется от (0, 0, 0, 0) в сторону (x, y, z, w).
  • Любая произвольная упорядоченная четверка чисел.

Есть и другие вещи, для представления которых можно использовать 4D-вектор, однако это самые распространенные варианты использования в Verge3D.

Пример Кода

const a = new v3d.Vector4(0, 1, 0, 0); //без аргументов; инициализируется в (0, 0, 0, 0, 1) const b = new v3d.Vector4(); const d = a.dot(b);

Конструктор

[page:Float y] — значение y этого вектора. По умолчанию *0*.
[page:Float z] — значение z этого вектора. По умолчанию *0*.
[page:Float w] — значение w этого вектора. По умолчанию *0*.

Методы

[method:this add]([param:Vector4 v])

Добавляет [page:Vector4 v] к данному вектору.

Добавляет скалярное значение s к значениям [page:.x x], [page:.y y], [page:.z z] и [page:.w w] этого вектора.

Устанавливает этот вектор в [page:Vector4 a] + [page:Vector4 b]..

[method:this applyMatrix4]([param:Matrix4 m])

Умножает данный вектор на 4 x 4 [page:Matrix4 m].

[method:this ceil]()

Компоненты [page:.x x], [page:.y y], [page:.z z] и [page:.w w] этого вектора округляются до ближайшего целого значения.

[method:this clamp]([param:Vector4 min], [param:Vector4 max])

[page:Vector4 min] — минимальные значения [page:.x x], [page:.y y], [page:.z z] и [page:.w w].
[page:Vector4 max] — максимальные значения [page:.x x], [page:.y y], [page:.z z] и [page:.w w] в нужном диапазоне

Если значение x, y, z или w этого вектора больше, чем значение x, y, z или w вектора max, оно заменяется соответствующим значением.

Если значение x, y, z или w этого вектора меньше, чем значение x, y, z или w вектора min, оно заменяется соответствующим значением.

[method:this clampLength]([param:Float min], [param:Float max])

[page:Float min] — минимальное значение, до которого будет зажиматься длина
[page:Float max] — максимальное значение, до которого будет зажиматься длина

Если длина этого вектора больше значения max, то он заменяется значением max.

Если длина этого вектора меньше минимального значения, он заменяется минимальным значением.

[method:this clampScalar]([param:Float min], [param:Float max])

[page:Float min] — минимальное значение, до которого будут зажиматься компоненты
[page:Float max] — максимальное значение, на которое будут зажиматься компоненты

Если значения x, y, z или w этого вектора больше максимального значения, они заменяются максимальным значением.

Если значения x, y, z или w этого вектора меньше минимального значения, они заменяются минимальным значением.

[method:Vector4 clone]()

Возвращает новый Vector4 с теми же значениями [page:.x x], [page:.y y], [page:.z z] и [page:.w w], что и данный.

[method:this copy]([param:Vector4 v])

Копирует значения свойств [page:.x x], [page:.y y], [page:.z z] и [page:.w w] переданного Vector4 в этот Vector4.

[method:this divideScalar]([param:Float s])

Делит данный вектор на скаляр [page:Float s].
Устанавливает вектор в *(0, 0, 0, 0, 0)*, если *[page:Float s] = 0*.

[method:Float dot]([param:Vector4 v])

Вычисляет [link:https://en.wikipedia.org/wiki/Dot_product скалярное произведение] данного вектора и [page:Vector4 v].

[method:Boolean equals]([param:Vector4 v])

Проверяет строгое равенство данного вектора и [page:Vector4 v].

[method:this floor]()

Компоненты этого вектора округляются до ближайшего целого значения.

[method:this fromArray]([param:Array array], [param:Integer offset])

[page:Array array] — исходный массив.
[page:Integer offset] — (необязательно) смещение в массиве. По умолчанию равно 0.

Устанавливает значение [page:.x x] этого вектора в массив[offset + 0], [page:.y y] в массив[offset + 1] [page:.z z] в массив[offset + 2] и [page:.w w] в массив[offset + 3].

[method:this fromBufferAttribute]([param:BufferAttribute attribute], [param:Integer index])

[page:BufferAttribute attribute] — атрибут источника.
[page:Integer index] — индекс в атрибуте.

Устанавливает значения [page:.x x], [page:.y y], [page:.z z] и [page:.w w] этого вектора из атрибута [page:BufferAttribute].

[method:Float getComponent]([param:Integer index])

[page:Integer index] — 0, 1, 2 или 3.

Если индекс равен 0, возвращается значение [page:.x x].
Если индекс равен 1, возвращается значение [page:.x x].
Если индекс равен 2, возвращается значение [page:.x x].
Если индекс равен 3, возвращается значение [page:.x x].

[method:Float length]()

Вычисляет [link:https://en.wikipedia.org/wiki/Euclidean_distance Евклидову длину] (длину прямой линии) от (0, 0, 0, 0, 0) до (x, y, z, w).

[method:Float manhattanLength]()

Вычисляет [link:http://en.wikipedia.org/wiki/Taxicab_geometry расстояние городских кварталов] этого вектора.

[method:Float lengthSq]()

Вычисляет квадрат [link:https://en.wikipedia.org/wiki/Euclidean_distance Евклидовой длины] (длины прямой) от (0, 0, 0, 0, 0) до (x, y, z, w). Если вы сравниваете длины векторов, то вместо этого следует сравнивать квадрат длины, так как он вычисляется немного эффективнее.

[method:this lerp]([param:Vector4 v], [param:Float alpha])

[page:Vector4 v] - [page:Vector4] для интерполяции в направлении.
[page:Float alpha] — коэффициент интерполяции, обычно в замкнутом интервале [0, 1].

Линейно интерполирует между этим вектором и [page:Vector4 v], где alpha - процентное расстояние вдоль линии - alpha = 0 будет этот вектор, а alpha = 1 будет [page:Vector4 v].

[method:this lerpVectors]([param:Vector4 v1], [param:Vector4 v2], [param:Float alpha])

[page:Vector4 v1] — начальный [page:Vector4].
[page:Vector4 v2] - [page:Vector4] для интерполяции в направлении.
[page:Float alpha] — коэффициент интерполяции, обычно в замкнутом интервале [0, 1].

Устанавливает этот вектор как вектор, линейно интерполированный между [page:Vector4 v1] и [page:Vector4 v2], где alpha - процентное расстояние вдоль линии, соединяющей два вектора - alpha = 0 будет [page:Vector4 v1], а alpha = 1 будет [page:Vector4 v2].

[method:this negate]()

Инвертирует этот вектор - то есть устанавливает x = -x, y = -y, z = -z и w = -w.

[method:this normalize]()

Преобразует этот вектор в [link:https://en.wikipedia.org/wiki/Unit_vector единичный вектор] — то есть устанавливает его равным вектору с тем же направлением, что и этот, но [page:.length length] 1.

Если значение x, y, z или w этого вектора меньше, чем значение x, y, z или w [page:Vector4 v], замените это значение соответствующим значением max.

[method:this min]([param:Vector4 v])

Если значение x, y, z или w этого вектора больше, чем значение x, y, z или w [page:Vector4 v], замените это значение соответствующим значением min.
