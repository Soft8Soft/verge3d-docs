[name]

Цилиндрические координаты точки [link:https://en.wikipedia.org/wiki/Cylindrical_coordinate_system cylindrical coordinates].

Конструктор

[name]([param:Float radius], [param:Float theta], [param:Float y])

[page:Float radius] — distance from the origin to a point in the x-z plane. Default is *1.0*.
[page:Float theta] — counterclockwise angle in the x-z plane measured in radians from the positive z-axis. Default is *0*.
[page:Float y] — height above the x-z plane. Default is *0*.

Свойства

[property:Float radius]

[property:Float theta]

[property:Float y]

Методы

[method:Cylindrical clone]()

Возвращает новую цилиндрическую систему координат с теми же свойствами [page:.radius radius], [page:.theta theta] и [page:.y y], что у данной.

[method:Cylindrical copy]([param:Cylindrical other])

Копирует значения свойств [page:.radius radius], [page:.theta theta] и [page:.y y] переданного цилиндрической системой координат в эту цилиндрическую систему координат.

[method:Cylindrical set]([param:Float radius], [param:Float theta], [param:Float y])

Устанавливает значения свойств [page:.radius radius], [page:.theta theta] и [page:.y y] этой цилиндрической системы координат.

[method:Cylindrical setFromVector3]([param:Vector3 vec3])

Устанавливает значения свойств [page:.radius radius], [page:.theta theta] и [page:.y y] этой цилиндрической системы координат из [page:Vector3 Vector3].

[method:Cylindrical setFromCartesianCoords]([param:Float x], [param:Float y], [param:Float z])

Устанавливает значения свойств [page:.radius radius], [page:.theta theta] и [page:.y y] этой цилиндрической системы координат из декартовых координат.
