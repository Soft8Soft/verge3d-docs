[name]

Фрустумы [link:http://en.wikipedia.org/wiki/Frustum Frustums] используются для определения того, что находится в поле зрения камеры. Они помогают ускорить процесс рендеринга - объекты, находящиеся за пределами фрустума камеры, могут быть исключены из рендеринга.

Этот класс в основном предназначен для внутреннего использования рендерером для вычисления фрустума [page:Camera camera camera] или [page:LightShadow.camera shadowCamera].

Конструктор

[page:Plane p0] — (необязательно) по умолчанию используется новый [page:Plane].
[page:Plane p1] — (необязательно) по умолчанию используется новый [page:Plane].
[page:Plane p2] — (необязательно) по умолчанию используется новый [page:Plane].
[page:Plane p3] — (необязательно) по умолчанию используется новый [page:Plane].
[page:Plane p4] — (необязательно) по умолчанию используется новый [page:Plane].
[page:Plane p5] — (необязательно) по умолчанию используется новый [page:Plane].

Создает новый [name].

Свойства

[property:Array planes]

Массив из 6 [page:Plane planes].

Методы

[method:Frustum clone]()

Возвращает новый Frustum с теми же параметрами, что и этот.

[method:Boolean containsPoint]([param:Vector3 point])

[page:Vector3 point] - [page:Vector3] для проверки.

Проверяет, содержит ли фрустум точку [page:Vector3 point].

[method:Frustum copy]([param:Frustum frustum])

[page:Frustum frustum] — Фрустум для копирования.

Копирует свойства переданного [page:Frustum frustum] в этот.

[method:Boolean intersectsBox]([param:Box3 box])

[page:Box3 box] - [page:Box3] для проверки пересечения.

Возвращает true, если [page:Box3 box] пересекается с этим фрустом.

[method:Boolean intersectsObject]([param:Object3D object])

Проверяет, пересекает ли фрустум ограничительная сфера [page:Geometry.boundingSphere bounding sphere] объекта [page:Object3D object].

Обратите внимание, что объект должен иметь [page:Geometry] или [page:BufferGeometry], чтобы можно было рассчитать ограничительную сферу.

[method:Boolean intersectsSphere]([param:Sphere sphere])

[page:Sphere sphere] - [page:Sphere] для проверки пересечения.

Возвращает true, если [page:Sphere sphere] пересекается с этим фрустом.

[method:Boolean intersectsSprite]([param:Sprite sprite])

Проверяет, пересекает ли [page:Sprite sprite] фрустум.

[method:this set]([param:Plane p0], [param:Plane p1], [param:Plane p2], [param:Plane p3], [param:Plane p4], [param:Plane p5])

Устанавливает фрустум из переданных плоскостей. Порядок следования плоскостей не подразумевается.
Обратите внимание, что этот метод копирует только значения из заданных объектов.

[method:this setFromProjectionMatrix]([param:Matrix4 matrix])

[page:Matrix4 matrix] — Проекция [page:Matrix4] используемая для установки [page:.planes planes]

Устанавливает плоскости фрустума из матрицы проекции.
