[name]

Класс, представляющий цвет(Color).

Примеры Кода

Цвет может быть инициализирован любым из следующих способов:

//пустой конструктор - по умолчанию будет белым const color1 = new v3d.Color(); //Шестнадцатеричный цвет (рекомендуется) const color2 = new v3d.Color(0xff0000); //RGB-строка const color3 = new v3d.Color("rgb(255, 0, 0)"); const color4 = new v3d.Color("rgb(100%, 0%, 0%)"); //Имя цвета X11 - поддерживаются все 140 имен цветов. //Обратите внимание на отсутствие CamelCase в названии const color5 = new v3d.Color('skyblue'); //Строка HSL const color6 = new v3d.Color("hsl(0, 100%, 50%)"); //Разделение значений RGB между 0 и 1 const color7 = new v3d.Color(1, 0, 0);

Конструктор

[name]([param:Color_Hex_or_String r], [param:Float g], [param:Float b])

[page:Color_Hex_or_String r] — (необязательно) Если определены аргументы [page:Float g] и [page:Float b], красная составляющая цвета. Если они не определены, это может быть [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet hexadecimal triplet] (рекомендуется), строка в стиле CSS или другой экземпляр Color.
[page:Float g] — (необязательно) Если он определен, зеленый компонент цвета.
[page:Float b] — (необязательно) Если он определен, синий компонент цвета.

Обратите внимание, что стандартным методом указания цвета в Verge3D является [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet hexadecimal triplet], и этот метод используется во всей остальной документации.

Если все аргументы определены, то [page:Color_Hex_or_String r] — это красный компонент, [page:Float g] — зеленый компонент и [page:Float b] — синий компонент цвета.
Когда определен только [page:Color_Hex_or_String r]:

  • Это может быть [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet hexadecimal triplet] представляющий цвет (рекомендуется).
  • Это может быть другой экземпляр Color
  • Это может быть строка в стиле CSS. Например:
    • 'rgb(250, 0,0)'
    • 'rgb(100%,0%,0%)'
    • 'hsl(0, 100%, 50%)'
    • '#ff0000'
    • '#f00'
    • 'red'

Свойства

[property:Float r]

Значение красного канала между 0 и 1. По умолчанию 1.

[property:Float g]

Значение зеленого канала от 0 до 1. По умолчанию 1.

[property:Float b]

Значение синего канала от 0 до 1. По умолчанию 1.

Методы

[method:Color add]([param:Color color])

Добавляет значения RGB из [page:Color color] к значениям RGB этого цвета.

[method:Color addColors]([param:Color color1], [param:Color color2])

Устанавливает значения RGB этого цвета в сумму значений RGB [page:Color color1] и [page:Color color2].

[method:Color addScalar]([param:Number s])

Добавляет [page:Number s] к значениям RGB этого цвета.

[method:Color clone]()

Возвращает новый Цвет с теми же значениями [page:.r r], [page:.g g] и [page:.b b], что и данный.

[method:Color copy]([param:Color color])

Копирует параметры [page:.r r], [page:.g g] и [page:.b b] из [page:Color color] в этот цвет.

[method:Color convertGammaToLinear]([param:Float gammaFactor])

[page:Float gammaFactor] — (необязательно). По умолчанию *2.0*.

Преобразует данный цвет из гамма-пространства в линейное пространство, взяв [page:.r r], [page:.g g] и [page:.b b] в степени [page:Float gammaFactor].

[method:Color convertLinearToGamma]([param:Float gammaFactor])

[page:Float gammaFactor] — (необязательно). По умолчанию *2.0*.

Преобразует данный цвет из линейного пространства в гамма-пространство, взяв [page:.r r], [page:.g g] и [page:.b b] в степени 1 / [page:Float gammaFactor].

[method:Color convertLinearToSRGB]()

Преобразует данный цвет из линейного пространства в пространство sRGB.

[method:Color convertRGBEToLinear]([param:Float expFactor])

[page:Float expFactor] — Общая экспонента RGBE.

Преобразует данный цвет из формата RGBE в линейный.

[method:Color convertSRGBToLinear]()

Преобразует данный цвет из пространства sRGB в линейное пространство.

[method:Color copyGammaToLinear]([param:Color color], [param:Float gammaFactor])

[page:Color color] — Цвет для копирования.
[page:Float gammaFactor] — (необязательно). По умолчанию *2.0*.

Копирует заданный цвет в этот цвет, а затем преобразует этот цвет из гамма-пространства в линейное пространство, взяв [page:.r r], [page:.g g] и [page:.b b] в степени [page:Float gammaFactor].

[method:Color copyLinearToGamma]([param:Color color], [param:Float gammaFactor])

[page:Color color] — Цвет для копирования.
[page:Float gammaFactor] — (необязательно). По умолчанию *2.0*.

Копирует заданный цвет в этот цвет, а затем преобразует этот цвет из линейного пространства в гамма-пространство, взяв [page:.r r], [page:.g g] и [page:.b b] в степени 1 / [page:Float gammaFactor].

[method:Color copyLinearToSRGB]([param:Color color]])

[page:Color color] — Цвет для копирования.
Копирует заданный цвет в этот цвет, а затем преобразует этот цвет из линейного пространства в пространство sRGB.

[method:Color copyRGBEToLinear]([param:Color color], [param:Float expFactor]])

[page:Color color] — Цвет для копирования.
[page:Float expFactor] — Общая экспонента RGBE.

Копирует заданный цвет в этот цвет, а затем преобразует этот цвет из формата RGBE в линейный.

[method:Color copySRGBToLinear]([param:Color color])

[page:Color color] — Цвет для копирования.
Копирует заданный цвет в этот цвет, а затем преобразует этот цвет из пространства sRGB в линейное пространство.

[method:Boolean equals]([param:Color color])

Сравнивает RGB-значения [page:Color color] со значениями данного объекта. Возвращает true, если они одинаковы, false в противном случае.

[method:Color fromArray]([param:Array array], [param:Integer offset])

[page:Array array] - [page:Array] поплавков в виде [[page:Float r], [page:Float g], [page:Float b]].
[page:Integer offset] — Необязательное смещение в массиве.

Устанавливает компоненты этого цвета на основе массива, отформатированного как [[page:Float r], [page:Float g], [page:Float b]].

[method:this fromBufferAttribute]([param:BufferAttribute attribute], [param:Integer index])

[page:BufferAttribute attribute] — атрибут источника.
[page:Integer index] — индекс в атрибуте.

Устанавливает компоненты этого цвета из атрибута [page:BufferAttribute].

[method:Integer getHex]()

Устанавливает компоненты этого цвета из атрибута [page:BufferAttribute].

[method:String getHexString]()

Возвращает шестнадцатеричное значение этого цвета в виде строки (например, 'FFFFFF')..

[method:Object getHSL]([param:Object target])

[page:Object target] — результат будет скопирован в этот объект. Добавляет ключи h, s и l к объекту (если они еще не присутствуют).

Преобразует значения [page:.r r], [page:.g g] и [page:.b b] этого Color в формат [link:https://en.wikipedia.org/wiki/HSL_and_HSV HSL] и возвращает объект вида: { h: 0, s: 0, l: 0 }

[method:String getStyle]()

Возвращает значение этого цвета в виде строки стиля CSS. Пример: 'rgb(255,0,0)' формы:

[method:Color lerp]([param:Color color], [param:Float alpha])

[page:Color color] — цвет для сходства.
[page:Float alpha] — коэффициент интерполяции в замкнутом интервале [0, 1].

Линейно интерполирует значения RGB этого цвета к значениям RGB переданного аргумента. Аргумент alpha можно представить как соотношение между двумя цветами, где 0.0 - данный цвет, а 1.0 - первый аргумент.

[method:Color lerpHSL]([param:Color color], [param:Float alpha])

[page:Color color] — цвет для сходства.
[page:Float alpha] — коэффициент интерполяции в замкнутом интервале [0, 1].

Линейно интерполирует значения HSL этого цвета к значениям HSL переданного аргумента. Он отличается от классического [page:.lerp] тем, что не интерполирует прямо от одного цвета к другому, а проходит через все оттенки между этими двумя цветами. Аргумент alpha можно представить как соотношение между двумя цветами, где 0.0 - этот цвет, а 1.0 - первый аргумент.

[method:Color multiply]([param:Color color])

Умножает значения RGB этого цвета на значения RGB данного [page:Color color].

[method:Color multiplyScalar]([param:Number s])

Умножает значения RGB этого цвета на [page:Number s].

[method:Color offsetHSL]([param:Float h], [param:Float s], [param:Float l])

Добавляет заданные значения [page:Float h], [page:Float s] и [page:Float l] к значениям этого цвета. Внутренне это преобразует значения [page:.r r], [page:.g g] и [page:.b b] цвета в HSL, добавляет [page:Float h], [page:Float s] и [page:Float l], а затем преобразует цвет обратно в RGB.

[method:Color set]([param:Color_Hex_or_String value])

[page:Color_Hex_or_String value] — Значение для установки этого цвета.

Смотрите Конструктор выше для получения полной информации о том, чем может быть [page:Color_Hex_or_String value]. Делегирует [page:.copy], [page:.setStyle] или [page:.setHex] в зависимости от типа ввода.

[method:Color setHex]([param:Integer hex])

[page:Integer hex] — [link:https://en.wikipedia.org/wiki/Web_colors#Hex_triplet hexadecimal triplet] формат.

Устанавливает этот цвет из шестнадцатеричного значения.

[method:Color setHSL]([param:Float h], [param:Float s], [param:Float l])

[page:Float h] — значение оттенка в диапазоне от 0,0 до 1,0
[page:Float s] — значение насыщенности от 0,0 до 1,0
[page:Float l] — значение освещенности в диапазоне от 0,0 до 1,0

Устанавливает цвет из значений HSL.

[method:Color setRGB]([param:Float r], [param:Float g], [param:Float b])

[page:Float r] — значение красного канала между 0,0 и 1,0.
[page:Float g] — значение зеленого канала между 0,0 и 1,0.
[page:Float b] — значение синего канала между 0,0 и 1,0.

Устанавливает данный цвет из значений RGB.

[method:Color setScalar]([param:Float scalar])

[page:Float scalar] — значение между 0.0 и 1.0.

Устанавливает все три цветовых компонента в значение [page:Float scalar].

[method:Color setStyle]([param:String style])

[page:String style] — цвет как строка CSS-стиля.

Устанавливает этот цвет из строки CSS-стиля. Например, "rgb(250, 0,0)", "rgb(100%, 0%, 0%)", "hsl(0, 100%, 50%)", "#ff0000", "#f00", or "red" (или любого имени цвета X11 [link:https://en.wikipedia.org/wiki/X11_color_names#Color_name_chart] - поддерживаются все 140 названий цветов).
Полупрозрачные цвета, такие как "rgba(255, 0, 0, 0, 0.5)" и "hsla(0, 100%, 50%, 0.5)" также принимаются, но координата альфа-канала будет отброшена.

Обратите внимание, что для названий цветов X11 несколько слов, таких как Dark Orange, превращаются в строку 'darkorange' (все строчные буквы).

[method:Color setColorName]([param:String style])

[page:String style] — название цвета (from [link:https://en.wikipedia.org/wiki/X11_color_names#Color_name_chart X11 названий цветов]).

Устанавливает этот цвет из имени цвета. Быстрее, чем метод [page:.setStyle], если вам не нужны другие форматы CSS-стилей.

Для удобства список имен представлен в Color.NAMES в виде хэша: Color.NAMES.aliceblue // returns 0xF0F8FF

[method:Color sub]([param:Color color])

Вычитает компоненты RGB данного цвета из компонентов RGB этого цвета. Если в результате получается отрицательный компонент, он устанавливается равным нулю.

[method:Array toArray]([param:Array array], [param:Integer offset])

[page:Array array] — Необязательный массив для хранения цвета.
[page:Integer offset] — Необязательное смещение в массиве.

Возвращает массив вида [r, g, b].
