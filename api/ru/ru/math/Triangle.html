[name]

Геометрический треугольник, определяемый тремя [page:Vector3 Vector3s], представляющими три его угла.

Конструктор

[name]([param:Vector3 a], [param:Vector3 b], [param:Vector3 c])

[page:Vector3 a] — первый угол треугольника. По умолчанию это [page:Vector3] в точке (0, 0, 0).
[page:Vector3 b] — второй угол треугольника. По умолчанию это [page:Vector3] в точке (0, 0, 0)..
[page:Vector3 c] — последний угол треугольника. По умолчанию это [page:Vector3] в точке (0, 0, 0)..

Создает новый [name].

Свойства

[property:Vector3 a]

Первый угол треугольника. По умолчанию это [page:Vector3] в точке (0, 0, 0).

[property:Vector3 b]

Второй угол треугольника. По умолчанию это [page:Vector3] в точке (0, 0, 0).

[property:Vector3 c]

Последний угол треугольника. По умолчанию это [page:Vector3] в точке (0, 0, 0).

Методы

[method:Triangle clone]()

Возвращает новый треугольник с теми же свойствами [page:.a a], [page:.b b] и [page:.c c], что и данный.

[method:Vector3 closestPointToPoint]([param:Vector3 point], [param:Vector3 target])

[page:Vector3 point] - [page:Vector3]
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает ближайшую точку треугольника к [page:Vector3 point].

[method:Boolean containsPoint]([param:Vector3 point])

[page:Vector3 point] - [page:Vector3] для проверки.

Возвращает true, если переданная точка, спроецированная на плоскость треугольника, лежит внутри треугольника.

[method:Triangle copy]([param:Triangle triangle])

Копирует значения свойств [page:.a a], [page:.b b] и [page:.c c] переданного треугольника в данный треугольник.

[method:Boolean equals]([param:Triangle triangle])

Возвращает true, если два треугольника имеют одинаковые свойства [page:.a a], [page:.b b] и [page:.c c].

[method:Float getArea]()

Возвращает площадь треугольника.

[method:Vector3 getBarycoord]([param:Vector3 point], [param:Vector3 target])

[page:Vector3 point] - [page:Vector3]
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает [link:https://en.wikipedia.org/wiki/Barycentric_coordinate_system barycentric coordinate] из заданного вектора.

[link:http://commons.wikimedia.org/wiki/File:Barycentric_coordinates_1.png Picture of barycentric coordinates]

[method:Vector3 getMidpoint]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Вычислите среднюю точку треугольника.

[method:Vector3 getNormal]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Вычисляет [link:https://en.wikipedia.org/wiki/Normal_(geometry) normal vector] треугольника.

[method:Plane getPlane]([param:Plane target])

[page:Plane target] — результат будет скопирован в эту плоскость.

Вычислите [page:Plane plane] на основе треугольника.

[method:Boolean intersectsBox]([param:Box3 box])

[page:Box3 box] — Куб для проверки пересечения.

Определяет, пересекает ли данный треугольник [page:Box3 box].

[method:Triangle set]([param:Vector3 a], [param:Vector3 b], [param:Vector3 c]) [param:Triangle this]

Устанавливает свойства [page:.a a], [page:.b b] и [page:.c c] треугольника на переданные [page:Vector3 vector3s].
Обратите внимание, что этот метод только копирует значения из заданных объектов.

[method:Triangle setFromPointsAndIndices]([param:Array points], [param:Integer i0], [param:Integer i1], [param:Integer i2]) [param:Triangle this]

точки - [page:Array] of [page:Vector3]s
i0 - [page:Integer] индекс
i1 - [page:Integer] индекс
i2 - [page:Integer] индекс

Устанавливает векторы треугольника в векторы в массиве.
