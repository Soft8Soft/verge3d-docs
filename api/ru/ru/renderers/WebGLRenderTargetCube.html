Используется [page:CubeCamera] в качестве [page:WebGLRenderTarget].

Примеры

См. [page:CubeCamera] для примеров.

Конструктор

[name]([param:Number width], [param:Number height], [param:Object options])

[page:Float width] — Ширина renderTarget.
[page:Float height] — Высота renderTarget.
options - (необязательный) объект, содержащий параметры текстуры для автогенерируемой цели текстуры и булевые depthBuffer/stencilBuffer. Объяснение параметров текстуры см. в [page:Texture Texture]. Ниже перечислены допустимые параметры:

[page:Constant wrapS] — по умолчанию [page:Textures ClampToEdgeWrapping].
[page:Constant wrapT] — по умолчанию [page:Textures ClampToEdgeWrapping].
[page:Constant magFilter] — по умолчанию [page:Textures .LinearFilter].
[page:Constant minFilter] — по умолчанию [page:Textures LinearFilter].
[page:Constant format] — по умолчанию [page:Textures RGBAFormat].
[page:Constant type] — по умолчанию [page:Textures UnsignedByteType].
[page:Number anisotropy] — по умолчанию *1*. См. [page:Texture.anistropy]
[page:Constant encoding] — по умолчанию [page:Textures LinearEncoding].
[page:Boolean depthBuffer] — по умолчанию *true*. Установите значение false, если оно вам не нужно.
[page:Boolean stencilBuffer] — по умолчанию *true*. Установите значение false, если оно вам не нужно.
