Добро пожаловать в справочник разработчика «Вердж3Д»!

Эта документация предназначена для программистов, которые хотели бы создавать интерактивные 3Д-веб-приложения с помощью языка программирования «ЯваСкрипт» (англ. JavaScript). Если вы собираетесь создавать эти приложения без программирования, вам потребуется лишь информация из руководства пользователя.

Выберите тему слева. Если вы не знаете с чего начать — перейдите к разделу основы программирования.

Данное руководство доступно также на английском и китайском языках.
