[name]

Класс для загрузки [page:CubeTexture CubeTexture]. Для загрузки файлов используется внутренний [page:ImageLoader].

Пример Кода

const scene = new v3d.Scene(); scene.background = new v3d.CubeTextureLoader() .setPath('textures/cubeMaps/') .load([ 'px.png', 'nx.png', 'py.png', 'ny.png', 'pz.png', 'nz.png' ]);

[method:CubeTexture load]([param:String urls], [param:Function onLoad], [param:Function onProgress], [param:Function onError])

[page:String urls] — массив из 6 ссылок на изображения, по одной для каждой стороны CubeTexture. Ссылки должны быть указаны в следующем порядке: pos-x, neg-x, pos-y, neg-y, pos-z, neg-z. Они также могут быть [link:https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs Data URIs].
Обратите внимание, что по традиции кубомапы задаются в системе координат, в которой положительный угол x находится справа, если смотреть вверх по положительной оси z - другими словами, используется левосторонняя система координат. Поскольку Verge3D использует правостороннюю систему координат, карты окружения, используемые в Verge3D, поменяют местами pos-x и neg-x.
[page:Function onLoad] — Будет вызван после завершения загрузки. Аргументом будет загруженная [page:Texture texture].
[page:Function onProgress] — Вызывается в процессе загрузки. Аргументом будет экземпляр XMLHttpRequest, который содержит байты .[page:Integer total] и .[page:Integer loaded].
[page:Function onError] — Вызывается при ошибках загрузки.

Начните загрузку из url и передайте загруженную [page:CubeTexture texture] в onLoad.
