[name]

Ограничение, которое ограничивает вращение ограниченного объекта в пространстве его родителя.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]()

Свойства

Общие свойства смотрите в базовом классе [page:Constraint].

[property:String axis]

Ось, вокруг которой следует ограничить вращение. Может быть одной из "X", "Y", "Z". По умолчанию - "Y".

[property:Boolean isLimitRotationConstraint]

Используется для проверки того, являются ли данный или производные классы ограничениями поворота. По умолчанию *true*.

[property:Number max]

Максимально допустимый угол Эйлера вокруг соответствующей оси для ограничения вращения объекта. По умолчанию 2π.

[property:Number min]

Минимально допустимый угол Эйлера вокруг соответствующей оси для ограничения вращения объекта. По умолчанию равно 0.

[property:String type]

Тип ограничения. Равен "LimitRotationConstraint".
