[name]

Абстрактный базовый класс для ограничений.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]()

Это создает общее ограничение.

Свойства

[property:Boolean isConstraint]

Используется для проверки того, является ли данный или производные классы ограничениями. По умолчанию *true*.

[property:Boolean mute]

Отключает ограничение, чтобы оно не влияло на ограниченный объект. По умолчанию *false*.

[property:String name]

Имя ограничения. По умолчанию это пустая строка.

[property:String type]

Тип ограничения. Равен "Constraint".

Методы

[method:Constraint clone]()

Возвращает новое ограничение с теми же параметрами, что и данное ограничение.

[method:Constraint copy]([page:Constraint source])

Скопируйте параметры из переданного ограничения "source" в это ограничение.

[method:null update]([page:Object3D obj])

Обновляет преобразования переданного объекта, применяя к нему ограничение.
