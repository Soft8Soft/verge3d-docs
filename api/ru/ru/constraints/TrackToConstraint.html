[name]

Ограничение, которое заставляет ограниченный объект отслеживать цель.

Ограничения используются для установления различных отношений между объектами, влияя на их положение, вращение и масштаб. Могут использоваться со свойством [page:Object3D.constraints Object3D.constraints].

Конструктор

[name]([page:Object3D target])

Свойства

Общие свойства смотрите в базовом классе [page:TargetConstraint].

[property:Boolean isTrackToConstraint]

Используется для проверки того, являются ли данный или производные классы ограничениями TrackTo. По умолчанию *true*.

[property:String trackAxis]

Ось ограниченного объекта, по которой отслеживается цель. Может быть одной из "X", "-X", "Y", "-Y", "Z", "-Z". По умолчанию - "X".

[property:String type]

Тип ограничения. Равен "TrackToConstraint".

[property:String upAxis]

Ось ограниченного объекта, которая всегда максимально выровнена с осью Y мирового пространства. Может быть одной из "X", "-X", "Y", "-Y", "Z", "-Z". По умолчанию - "Y".
