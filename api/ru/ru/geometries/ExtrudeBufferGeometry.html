[name]

Создает экструдированную BufferGeometry из формы контура.

Пример Кода

const length = 12, width = 8; const shape = new v3d.Shape(); shape.moveTo(0,0); shape.lineTo(0, width); shape.lineTo(length, width); shape.lineTo(length, 0); shape.lineTo(0, 0); const extrudeSettings = { steps: 2, depth: 16, bevelEnabled: true, bevelThickness: 1, bevelSize: 1, bevelOffset: 0, bevelSegments: 1 }; const geometry = new v3d.ExtrudeBufferGeometry(shape, extrudeSettings); const material = new v3d.MeshBasicMaterial({ color: 0x00ff00 }); const mesh = new v3d.Mesh(geometry, material) ; scene.add(mesh);

Конструктор

[name]([param:Array shapes], [param:Object options])

shapes — Фигура или массив фигур.
options — Объект, который может содержать следующие параметры.

  • curveSegments — int. Количество точек на кривых. По умолчанию - 12.
  • steps — int. количество точек для разделения сегментов по глубине выдавленного сплайна.По умолчанию равно 1.
  • depth — float. глубина выдавливания (рельефа) фигуры. По умолчанию 100.
  • bevelEnabled — bool. Применить фаску к фигуре. По умолчанию - true.
  • bevelThickness — float. Насколько глубоко в исходную фигуру уходит фаска. По умолчанию - 6.
  • bevelSize — float. Расстояние от контура фигуры, на которое распространяется фаска. По умолчанию bevelThickness - 2.
  • bevelOffset — float. Расстояние от контура фигуры, на котором начинается фаска. По умолчанию равно 0.
  • bevelSegments — int. Количество слоев фаски. По умолчанию - 3.
  • extrudePath — v3d.Curve. 3D-сплайн контура, по которому должна быть экструдирована форма. Фаски не поддерживаются для экструзии контура.
  • UVGenerator — Object. Объект, предоставляющий функции генератора UV.

Этот объект экструдирует 2D-форму в 3D-геометрию.

При создании меша с этой геометрией, если вы хотите, чтобы отдельный материал использовался для ее лицевой и боковых сторон, вы можете использовать массив материалов. Первый материал будет применен к лицевой стороне; второй материал будет применен к боковым сторонам.
