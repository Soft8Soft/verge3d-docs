[page:BufferGeometry] →

[name]

Его можно использовать в качестве вспомогательного объекта для просмотра [page:Geometry Geometry] в виде сетки.

Пример Кода

const geometry = new v3d.SphereBufferGeometry(100, 100, 100); const wireframe = new v3d.WireframeGeometry(geometry); const line = new v3d.LineSegments(wireframe); line.material.depthTest = false; line.material.opacity = 0.25; line.material.transparent = true; scene.add(line);
