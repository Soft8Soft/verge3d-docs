[name]

Абстрактный базовый класс для светильников - все остальные типы светильников наследуют описанные здесь свойства и методы.

Конструктор

[name]([param:Integer color], [param:Float intensity])

[page:Integer color] — (необязательно) шестнадцатеричный цвет света. По умолчанию 0xffffffff (белый).
[page:Float intensity] — (необязательно) числовое значение силы/интенсивности света. По умолчанию 1.

Создает новый [name]. Обратите внимание, что он не предназначен для прямого вызова (вместо этого используйте один из производных классов).

Свойства

Общие свойства смотрите в базовом классе [page:Object3D Object3D].

[property:Color color]

Цвет света. По умолчанию используется новый [page:Color], установленный на белый, если не передан в конструкторе.

[property:Float intensity]

Интенсивность или сила света.
В режиме [page:WebGLRenderer.physicallyCorrectLights physically correct] произведение [page:.color color] * intensity интерпретируется как сила света, измеряемая в канделах.
По умолчанию - *1.0*.

Методы

Общие методы смотрите в базовом классе [page:Object3D Object3D].

[method:Light copy]([param:Light source])

Копирует значение [page:.color color] и [page:.intensity intensity] из света [page:Light source] в этот свет.
