[name]

RectAreaLight излучает свет равномерно по всей поверхности прямоугольной плоскости. Этот тип света можно использовать для имитации таких источников света, как яркие окна или полосатое освещение.

Важные замечания:

  • Отсутствует поддержка теней.
  • Поддерживаются только [page:MeshStandardMaterial MeshStandardMaterial] и [page:MeshPhysicalMaterial MeshPhysicalMaterial].

Пример кода

const width = 10; const height = 10; const intensity = 1; const rectLight = new v3d.RectAreaLight(0xffffff, intensity, width, height); rectLight.position.set(5, 5, 0); rectLight.lookAt(0, 0, 0); scene.add(rectLight) const rectLightHelper = new v3d.RectAreaLightHelper(rectLight); rectLight.add(rectLightHelper);

Конструктор

[name]([param:Integer color], [param:Float intensity], [param:Float width], [param:Float height])

[page:Integer color] — (необязательно) шестнадцатеричный цвет света. По умолчанию 0xffffffff (белый).
[page:Float intensity] — (необязательно) интенсивность света, или яркость. По умолчанию 1.
[page:Float width] — (необязательно) ширина света. По умолчанию 10.
[page:Float height] — (необязательно) высота света. По умолчанию 10.
