[name]

Этот свет одинаково глобально освещает все объекты в сцене.

Этот свет нельзя использовать для отбрасывания теней, так как он не имеет направления.

Пример

const light = new v3d.AmbientLight(0x404040); // soft white light app.scene.add(light);

Конструктор

[name]([param:Integer color], [param:Float intensity])

[page:Integer color] — (необязательно) Числовое значение RGB-компонента цвета. По умолчанию 0xffffffff.
[page:Float intensity] — (необязательно) Числовое значение силы/интенсивности света. По умолчанию - 1.
