[name]

Свет, который излучается из одной точки во всех направлениях. Чаще всего его используют для воспроизведения света, излучаемого голой лампочкой.

Этот свет может отбрасывать тени - подробности смотрите на странице [page:PointLightShadow].

Пример кода

const light = new v3d.PointLight(0xff0000, 1, 100); light.position.set(50, 50, 50); scene.add(light);

Конструктор

[name]([param:Integer color], [param:Float intensity], [param:Number distance], [param:Float decay])

[page:Integer color] — (необязательно) шестнадцатеричный цвет света. По умолчанию 0xffffffff (белый).
[page:Float intensity] — (необязательно) числовое значение силы/интенсивности света. По умолчанию 1.

[page:Number distance] — Максимальная дальность действия света. По умолчанию 0 (без ограничения).
[page:Float decay] — Величина затемнения света по расстоянию до него. По умолчанию - 1. Для освещения [page:WebGLRenderer.physicallyCorrectLights physically correct], установите значение 2.

Создает новый [name].

Свойства

Общие свойства смотрите в базовом классе [page:Light Light].

[property:Float decay]

Величина ослабления света в зависимости от расстояния до него
В режиме [page:WebGLRenderer.physicallyCorrectLights physically correct] затухание = 2 приводит к физически реалистичному падению света.
По умолчанию *1*.

[property:Float distance]

Default mode — Когда расстояние равно нулю, свет не ослабевает. Когда расстояние ненулевое, свет ослабевает линейно от максимальной интенсивности в положении света до нуля на данном расстоянии от света.

[page:WebGLRenderer.physicallyCorrectLights Physically correct] режим — Когда расстояние равно нулю, свет будет ослабевать по закону обратного квадрата до бесконечного расстояния. Когда расстояние ненулевое, свет будет ослабевать по обратно-квадратичному закону до тех пор, пока не приблизится к отсечке расстояния, где он будет быстро и плавно ослабевать до 0. По своей сути, отсечки не являются физически корректными.

По умолчанию *0.0*.

[property:Float power]

Сила света.
В режиме [page:WebGLRenderer.physicallyCorrectLights physically correct] световая мощность света, измеряемая в люменах. По умолчанию *4Math.PI*.

Это напрямую связано с [page:.intensity intensity] в соотношении power = intensity * 4π и изменение этого параметра также изменит интенсивность.

[property:PointLightShadow shadow]

[page:PointLightShadow], используемый для расчета теней для этого света.

Камера LightShadow [page:LightShadow.camera camera] установлена на [page:PerspectiveCamera] с [page:PerspectiveCamera.fov fov] равным 90, [page:PerspectiveCamera.aspect aspect] равным 1, [page:PerspectiveCamera.near near] плоскостью сечения 0,5 и [page:PerspectiveCamera.far far] плоскостью обрезания 500.
