[name]

Базовый класс для всех типов камер. При создании камеры следует всегда использовать один из унаследованных классов.

Конструктор

[name]()

Создает новый [name]. Обратите внимание, что этот класс не предназначен для прямого вызова; вероятно, вместо него вам понадобится [page:PerspectiveCamera] или [page:OrthographicCamera].

Свойства

Смотрите свойства базового класса [page:Object3D].

[property:ControlSettings controlSettings]

Настройки управления камерой, используемые для установки [page:App.controls контролов] приложения.

[property:Boolean ignoreShadowLayers]

Игнорировать слои камеры при рендеринге в карту теней. Используется внутри SSR пасса. По умолчанию `false`.

[property:Boolean isCamera]

Указывает, что объект является экземпляром класса [name]. Только для чтения.

[property:Layers layers]

[page:Layers Слои], которые использует камера. Это свойство наследуется от [page:Object3D].

Объекты должны иметь хотя бы один общий слой с камерой, чтобы быть видимыми при рендеринге.

[property:Matrix4 matrixWorldInverse]

Это обратная матрица матрицы `matrixWorld`. `matrixWorld` содержит матрицу мирового преобразования камеры.

[property:Matrix4 projectionMatrix]

Это матрица, которая содержит проекцию.

[property:Matrix4 projectionMatrixInverse]

Обратная матрица матрицы проекции `projectionMatrix`.

[property:Object viewportFit]

Настройки режима подгонки под вьюпорт:

[page:Cameras type]
Тип подгонки. Смотрите [page:Cameras тут] для получения дополнительной информации.
[page:Float initialAspect]
Начальное значение соотношения сторон.

Методы

Смотрите методы базового класса [page:Object3D].

[method:Camera clone]([param:Boolean recursive])

Возвращает новую камеру с теми же свойствами, что и текущая.

[method:Camera copy]([param:Camera source], [param:Boolean recursive])

Копирует свойства из исходной камеры в текущую.

[method:Vector3 getWorldDirection]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот вектор.

Возвращает [page:Vector3], представляющий направление в мировом пространстве, вдоль которого смотрит камера. Примечание: камера смотрит вниз по своей локальной, отрицательной оси Z.
