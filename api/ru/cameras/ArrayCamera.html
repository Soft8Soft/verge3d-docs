[name]

[name] можно использовать для эффективного рендеринга сцены с заранее определенным набором камер. Это важный аспект производительности при рендеринге VR-сцен.
Экземпляр [name] всегда имеет массив подкамер. Для каждой подкамеры обязательно нужно определить свойство *viewport*, определяющее часть видового экрана, которая рендерится этой камерой.

Конструктор

[name]([param:Array array])

Массив камер.

Свойства

Общие свойства смотрите в базовом классе [page:PerspectiveCamera].

[property:Array cameras]

Массив камер.
