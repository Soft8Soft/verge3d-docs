[name]

Простая система кэширования, используемая внутри [page:FileLoader].

Пример

Чтобы включить кэширование для всех загрузчиков, использующих [page:FileLoader], установите значение

v3d.Cache.enabled = true.

Свойства

[property:Boolean enabled]

Включено ли кэширование. По умолчанию *false*.

[property:Object files]

[page:Object object] в котором хранятся кэшированные файлы.

Методы

[method:null add]([param:String key], [param:Object file])

[page:String key] — [page:String key], чтобы ссылаться на кэшированный файл.
[page:Object file] — Файл для кэширования.

Добавляет запись в кэш с ключом для ссылки на файл. Если этот ключ уже содержит файл, он перезаписывается.

[method:null get]([param:String key])

[page:String key] — Ключевая строка

Получение значения [page:String key]. Если ключ не существует, возвращается *undefined*.

[method:null remove]([param:String key])

[page:String key] — Ключевая строка, ссылающаяся на кэшированный файл.

Удаляет кэшированный файл, связанный с ключом.

[method:null clear]()

Удаляет все значения из кэша.
