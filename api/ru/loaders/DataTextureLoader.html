[name]

Абстрактный базовый класс для загрузки общих бинарных форматов текстур (rgbe, hdr, ...). Он использует внутренний [page:FileLoader] для загрузки файлов и создает новую [page:DataTexture].

Примеры

См. [link:https://github.com/Soft8Soft/verge3d-code-examples/blob/master/jsm/loaders/RGBELoader.js RGBELoader] для примера производного класса.

Конструктор

[name]([param:LoadingManager manager])

[page:LoadingManager manager] — [page:LoadingManager loadingManager] для использования загрузчиком. По умолчанию [page:LoadingManager v3d.DefaultLoadingManager].

Создает новый [name].

Свойства

Общие свойства см. в базовом классе [page:Loader].

Методы

Общие методы см. в базовом классе [page:Loader].

[method:DataTexture load]([param:String url], [param:Function onLoad], [param:Function onProgress], [param:Function onError])

[page:String url] — путь или URL-адрес к файлу. Это также может быть [link:https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs Data URI].
[page:Function onLoad] — Будет вызван после завершения загрузки. Аргументом будет загруженная текстура.
[page:Function onProgress] — Вызывается в процессе загрузки. Аргументом будет экземпляр XMLHttpRequest, который содержит байты .[page:Integer total] и .[page:Integer loaded].
[page:Function onError] — Вызывается при ошибках загрузки.

Начните загрузку из url и передайте загруженную текстуру в onLoad.
