[page:Loader] →

[name]

Загрузчик для загрузки [page:Image]. Используется модулями [page:CubeTextureLoader] и [page:TextureLoader].

Пример

// instantiate a loader const loader = new v3d.ImageLoader(); // load a image resource loader.load( // resource URL 'textures/skyboxsun25degtest.png', // onLoad callback function(image) { // use the image, e.g. draw part of it on a canvas const canvas = document.createElement('canvas'); const context = canvas.getContext('2d'); context.drawImage(image, 100, 100); }, // onProgress callback currently not supported undefined, // onError callback function() { console.error('An error happened.'); } );

Конструктор

[name]([param:LoadingManager manager])

[page:LoadingManager manager] — [page:LoadingManager loadingManager] для использования загрузчиком. По умолчанию [page:LoadingManager v3d.DefaultLoadingManager].

Создает новый [name].

Свойства

Общие свойства см. в базовом классе [page:Loader].

Методы

Общие методы см. в базовом классе [page:Loader].

[method:HTMLImageElement load]([param:String url], [param:Function onLoad], [param:Function onProgress], [param:Function onError])

[page:String url] — путь или URL-адрес к файлу. Это также может быть [link:https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs Data URI].
[page:Function onLoad] — Будет вызван после завершения загрузки. Аргументом будет загруженный [page:Image image].
[page:Function onProgress] — Эта функция обратного вызова в настоящее время не поддерживается.
[page:Function onError] — Вызывается при ошибках загрузки.

Начните загрузку из url и верните объект [page:Image image], который будет содержать данные.
