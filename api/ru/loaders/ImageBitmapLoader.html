[name]

Загрузчик для загрузки [page:Image] в качестве [link:https://developer.mozilla.org/de/docs/Web/API/ImageBitmap ImageBitmap]. ImageBitmap обеспечивает асинхронный и ресурсоэффективный способ подготовки текстур для рендеринга в WebGL.
В отличие от [page:FileLoader], [name] не прекращает несколько одновременных запросов к одному и тому же URL.

Обратите внимание, что [page:Texture.flipY] и [page:Texture.premultiplyAlpha] с [link:https://developer.mozilla.org/de/docs/Web/API/ImageBitmap ImageBitmap] игнорируются. [link:https://developer.mozilla.org/de/docs/Web/API/ImageBitmap ImageBitmap] нуждается в этих настройках при создании растрового изображения в отличие от обычных изображений, которым они нужны при загрузке на GPU. Вам нужно установить эквивалентные параметры через [page:ImageBitmapLoader.setOptions] вместо этого. Подробности см. в [ссылка:https://www.khronos.org/registry/webgl/specs/latest/1.0/#6.10 WebGL specification].

Пример

// instantiate a loader const loader = new v3d.ImageBitmapLoader(); // set options if needed loader.setOptions({ imageOrientation: 'flipY' }); // load a image resource loader.load( // resource URL 'textures/skyboxsun25degtest.png', // onLoad callback function(imageBitmap) { const texture = new v3d.CanvasTexture(imageBitmap); const material = new v3d.MeshBasicMaterial({ map: texture }); }, // onProgress callback currently not supported undefined, // onError callback function(err) { console.log('An error happened'); } );

Конструктор

[name]([param:LoadingManager manager])

[page:LoadingManager manager] — [page:LoadingManager loadingManager] для использования загрузчиком. По умолчанию [page:LoadingManager v3d.DefaultLoadingManager].

Создает новый [name].

Свойства

Общие свойства см. в базовом классе [page:Loader].

[property:String options]

Необязательный объект, задающий параметры для используемого внутри фабричного метода [link:https://developer.mozilla.org/en-US/docs/Web/API/WindowOrWorkerGlobalScope/createImageBitmap createImageBitmap]. По умолчанию *undefined*.

Методы

Общие методы см. в базовом классе [page:Loader].

[method:null load]([param:String url], [param:Function onLoad], [param:Function onProgress], [param:Function onError])

[page:String url] — путь или URL-адрес к файлу. Это также может быть [link:https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs Data URI].
[page:Function onLoad] — Будет вызван после завершения загрузки. Аргументом будет загруженный [page:Image image].
[page:Function onProgress] — Эта функция обратного вызова в настоящее время не поддерживается.
[page:Function onError] — Вызывается при ошибках загрузки.

Начните загрузку из url и верните объект [page:ImageBitmap image], который будет содержать данные.

[method:ImageBitmapLoader setOptions]([param:Object options])

Устанавливает объект опций для [link:https://developer.mozilla.org/en-US/docs/Web/API/WindowOrWorkerGlobalScope/createImageBitmap createImageBitmap].
