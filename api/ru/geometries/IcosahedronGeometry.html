[name]

Класс для генерации геометрии икосаэдра.

Конструктор

[name]([param:Float radius], [param:Integer detail])

radius — По умолчанию 1.
detail — По умолчанию 0. При установке этого значения больше 0 добавляется больше вершин, в результате чего он перестает быть икосаэдром. Когда значение параметра больше 1, это фактически сфера.

Свойства

Общие свойства см. в базовом классе [page:Geometry].

[property:Object parameters]

Объект со свойством для каждого из параметров конструктора. Любая модификация после построения не изменяет геометрию.
