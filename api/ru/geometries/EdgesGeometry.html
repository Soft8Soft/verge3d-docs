[page:BufferGeometry] →

[name]

Его можно использовать в качестве вспомогательного объекта для просмотра граней объекта [page:Geometry Geometry].

Пример Кода

const geometry = new v3d.BoxBufferGeometry(100, 100, 100); const edges = new v3d.EdgesGeometry(geometry); const line = new v3d.LineSegments(edges, new v3d.LineBasicMaterial({ color: 0xffffff })); scene.add(line);

Примеры

[example:webgl_helpers helpers]

Конструктор

[name]([param:Geometry geometry], [param:Integer thresholdAngle])

geometry — Любой геометрический объект.
thresholdAngle — Грань отображается только в том случае, если угол (в градусах) между нормалями смежных граней превышает это значение. По умолчанию = 1 градус.

Свойста

Общие свойства см. в базовом классе [page:BufferGeometry].

[property:Object parameters]

Объект со свойством для каждого из параметров конструктора. Любая модификация после построения не изменяет геометрию.
