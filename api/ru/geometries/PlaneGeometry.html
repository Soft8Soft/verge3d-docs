[name]

Класс для создания плоских геометрий

Пример

const geometry = new v3d.PlaneGeometry(5, 20, 32); const material = new v3d.MeshBasicMaterial({color: 0xffff00, side: v3d.DoubleSide}); const plane = new v3d.Mesh(geometry, material); app.scene.add(plane);

Конструктор

[name]([param:Float width], [param:Float height], [param:Integer widthSegments], [param:Integer heightSegments])

width — Ширина по оси X. По умолчанию 1.
height — Высота по оси Y. По умолчанию 1.
widthSegments — Опционально. По умолчанию 1.
heightSegments — Опционально. По умолчанию 1.

Свойства

Общие свойства см. в базовом классе [page:BufferGeometry].

[property:Object parameters]

Объект со свойством для каждого из параметров конструктора. Любая модификация после построения не изменяет геометрию.
