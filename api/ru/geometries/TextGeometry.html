[name]

Класс для генерации текста в виде единой геометрии. Он создается путем предоставления строки текста и хэша из параметров, состоящих из загружаемого [page:Font] и настроек для родительской геометрии [page:ExtrudeGeometry]. Дополнительные сведения см. на страницах [page:Font], [page:FontLoader] и [page:Creating-Text].

Пример Кода

const loader = new v3d.FontLoader(); loader.load('fonts/helvetiker_regular.typeface.json', function(font) { const geometry = new v3d.TextGeometry('Hello Verge3D!', { font: font, size: 80, height: 5, curveSegments: 12, bevelEnabled: true, bevelThickness: 10, bevelSize: 8, bevelOffset: 0, bevelSegments: 5 }); });

Примеры

[example:webgl_geometry_text geometry / text]

Конструктор

[name]([param:String text], [param:Object parameters])

text — Текст, который должен быть показан.
parameters — Объект, который может содержать следующие параметры.

  • font — экземпляр v3d.Font.
  • size — Float. Размер текста. По умолчанию равен 100.
  • height — Float. Толщина для экструзии текста. По умолчанию - 50.
  • curveSegments — Integer. Количество точек на кривых. По умолчанию - 12.
  • bevelEnabled — Boolean. Включает фаску. По умолчанию - False.
  • bevelThickness — Float. Глубина фаски текста. По умолчанию - 10.
  • bevelSize — Float. На каком расстоянии от контура текста находится фаска. По умолчанию - 8.
  • bevelOffset — Float. На каком расстоянии от контура текста начинается фаска. По умолчанию 0.
  • bevelSegments — Integer. Количество сегментов фаски. По умолчанию - 3.
