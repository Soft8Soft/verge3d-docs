[name]

Спрайт - это плоскость, всегда обращенная к камере, обычно с частично прозрачной текстурой.

Спрайты не отбрасывают тени, настройка castShadow = true не будет иметь никакого эффекта.

Пример

const map = new v3d.TextureLoader().load('sprite.png'); const material = new v3d.SpriteMaterial({ map: map }); const sprite = new v3d.Sprite(material); app.scene.add(sprite);

Конструктор

[name]([param:Material material])

[page:Material material] — (необязательно) экземпляр [page:SpriteMaterial]. По умолчанию это белый [page:SpriteMaterial].

Создает новый [name].

Свойства

Общие свойства см. в базовом классе [page:Object3D].

[property:SpriteMaterial material]

Экземпляр [page:SpriteMaterial], определяющий внешний вид объекта. По умолчанию это белый [page:SpriteMaterial].

[property:Vector2 center]

Точка привязки спрайта и точка, вокруг которой вращается спрайт. Значение (0,5, 0,5) соответствует средней точке спрайта. Значение (0, 0) соответствует левому нижнему углу спрайта. По умолчанию используется значение (0,5, 0,5).

Методы

Общие методы см. в базовом классе [page:Object3D].

[method:Sprite clone]()

Возвращает клон этого Sprite-объекта и всех его преемников.

[method:Sprite copy]([param:Sprite sprite])

Копирует свойства переданного спрайта в этот спрайт.

[method:null raycast]([param:Raycaster raycaster], [param:Array intersects])

Получить пересечения между отбрасываемым лучом и данным спрайтом. [page:Raycaster.intersectObject]() вызовет этот метод. Перед отбрасыванием лучей на спрайты необходимо инициализировать рэйкастер вызовом [page:Raycaster.setFromCamera]().
