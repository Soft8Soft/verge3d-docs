[name]

Класс, представляющий триангулированные полигональные меши. Также служит базой для других классов, таких как [page:SkinnedMesh].

Пример

const geometry = new v3d.BoxBufferGeometry(1, 1, 1); const material = new v3d.MeshBasicMaterial({ color: 0xffff00 }); const mesh = new v3d.Mesh(geometry, material); app.scene.add(mesh);

Конструктор

[name]([param:BufferGeometry geometry], [param:Material material])

  • [page:BufferGeometry geometry] — (необ.) экземпляр [page:BufferGeometry]. По умолчанию это новый экземпляр класса [page:BufferGeometry].
  • [page:Material material] — (необ.) один или массив [page:Material]. По умолчанию это новый экземпляр класса [page:MeshBasicMaterial].

Свойства

См. свойства базового класса [page:Object3D].

[property:BufferGeometry geometry]

Экземпляр [page:BufferGeometry] (или производных классов), определяющий структуру объекта.

[property:Boolean isMesh]

Указывает, что данный объект является экземпляром класса [name].

[property:Material material]

Экземпляр материала, полученный из базового класса [page:Material] или массив материалов, определяющий внешний вид объекта. По умолчанию это [page:MeshBasicMaterial].

[property:Array morphTargetInfluences]

Массив весов, обычно от 0 до 1, который определяет, насколько сильно применяется морфинг. По умолчанию не определен, но сбрасывается в пустой массив командой [page:Mesh.updateMorphTargets updateMorphTargets].

[property:Object morphTargetDictionary]

Список morphTargets основанный на свойстве morphTarget.name. По умолчанию не определен, но перестраивается [page:Mesh.updateMorphTargets updateMorphTargets].

Методы

См. методы базового класса [page:Object3D].

[method:Mesh clone]()

Возвращает клон данного объекта [name] и его потомков.

[method:Vector3 getVertexPosition]([param:Integer vert], [param:Vector3 target])

Получает позицию вертекса по указанному индексу в локальной системе координат. Учитывает состояние анимации морфинга и скиннинга.

[method:null raycast]([param:Raycaster raycaster], [param:Array intersects])

Получить пересечения между отбрасываемым лучом и этим мешем. [page:Raycaster.intersectObject] вызовет этот метод, но результаты не будут упорядочены.

[method:null updateMorphTargets]()

Обновляет morphTargets так, чтобы они не оказывали влияния на объект. Сбрасывает свойства [page:Mesh.morphTargetInfluences morphTargetInfluences] и [page:Mesh.morphTargetDictionary morphTargetDictionary].
