[name]

Непрерывная линия.

Это почти то же самое, что и [page:LineSegments]; разница лишь в том, что она отрендерена с использованием [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/drawElements gl.LINE_STRIP] вместо [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/drawElements gl.LINES]

Пример

const material = new v3d.LineBasicMaterial({ color: 0x0000ff }); const points = []; points.push(new v3d.Vector3(- 10, 0, 0)); points.push(new v3d.Vector3(0, 10, 0)); points.push(new v3d.Vector3(10, 0, 0)); const geometry = new v3d.BufferGeometry().setFromPoints(points); const line = new v3d.Line(geometry, material); app.scene.add(line);

Конструктор

[name]([param:BufferGeometry geometry], [param:Material material])

[page:BufferGeometry geometry] — вершины, представляющие сегмент(ы) линии. По умолчанию это новый [page:BufferGeometry].
[page:Material material] — материал для линии. По умолчанию это новый [page:LineBasicMaterial].

Свойства

Общие свойства см. в базовом классе [page:Object3D]..

[property:Geometry geometry]

Вершины, представляющие сегмент(ы) линии.

[property:Material material]

Материал для линии.

[property:Array morphTargetInfluences]

Массив весов, обычно от 0 до 1, которые определяют, насколько сильно применяется морфинг. По умолчанию не определен, но сбрасывается в пустой массив с помощью [page:.updateMorphTargets]().

[property:Object morphTargetDictionary]

Список morphTargets, основанный на свойстве morphTarget.name. По умолчанию не определен, но перестраивается [page:.updateMorphTargets]().

Методы

Общие методы см. в базовом классе [page:Object3D].

[method:Line computeLineDistances]()

Вычисляет массив значений расстояний, которые необходимы для [page:LineDashedMaterial]. Для каждой вершины в геометрии метод вычисляет суммарную длину от текущей точки до самого начала линии.

[method:null raycast]([param:Raycaster raycaster], [param:Array intersects])

Получить пересечения между отбрасываемым [page:Ray] и этой линией. [page:Raycaster.intersectObject] вызовет этот метод.

[method:Line clone]()

Возвращает клон данного Line-объекта и его преемников.

[method:null updateMorphTargets]()

Обновляет morphTargets так, чтобы они не оказывали влияния на объект. Сбрасывает [page:.morphTargetInfluences] и [page:.morphTargetDictionary].
