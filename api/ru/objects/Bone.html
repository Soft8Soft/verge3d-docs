[name]

Кость, которая является частью [page:Skeleton]. Скелет, в свою очередь, используется [page:SkinnedMesh]. Кости почти идентичны пустым [page:Object3D].

Пример

const root = new v3d.Bone(); const child = new v3d.Bone(); root.add(child); child.position.y = 5;

Конструктор

[name]()

Создает новое [name].

Свойста

Общие свойства см. в базовом классе [page:Object3D].

[property:String type]

Устанавливает значение 'Bone', оно может быть использовано для поиска всех костей в сцене..

Общие методы см. в базовом классе [page:Object3D].
