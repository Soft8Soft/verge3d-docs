[name]

Класс для отображения точек. Точки отображаются с помощью [page:WebGLRenderer], с использованием [link:https://developer.mozilla.org/en-US/docs/Web/API/WebGLRenderingContext/drawElements gl.POINTS].

Конструктор

[name]([param:BufferGeometry geometry], [param:Material material])

  • [page:BufferGeometry geometry] — (необязательный) экземпляр [page:BufferGeometry].
  • [page:Material material] — (необязательный) [page:Material]. По умолчанию это новый [page:PointsMaterial].

Свойства

Общие свойства см. в базовом классе [page:Object3D].

[property:Geometry geometry]

Экземпляр [page:BufferGeometry] (или производных классов), определяющий структуру объекта.

[property:Material material]

Экземпляр [page:Material], определяющий внешний вид объекта. По умолчанию это [page:PointsMaterial].

[property:Array morphTargetInfluences]

Массив весов, обычно от 0 до 1, который определяет, насколько сильно применяется морфинг. По умолчанию не определен, но сбрасывается в пустой массив командой [page:Points.updateMorphTargets updateMorphTargets].

[property:Object morphTargetDictionary]

Список morphTargets, основанный на свойстве morphTarget.name. По умолчанию не определена, но перестраивается [page:Points.updateMorphTargets updateMorphTargets].

Методы

Общие методы см. в базовом классе [page:Object3D]..

[method:null raycast]([param:Raycaster raycaster], [param:Array intersects])

Получить пересечения между отбрасываемым лучом и данными точками. [page:Raycaster.intersectObject] вызовет этот метод.

[method:Points clone]()

Возвращает клон этого Points-объекта и его преемников.

[method:null updateMorphTargets]()

Обновляет morphTargets так, чтобы они не оказывали влияния на объект. Сбрасывает свойства [page:Points.morphTargetInfluences morphTargetInfluences] и [page:Points.morphTargetDictionary morphTargetDictionary].
