[name]

Класс практически идентичный [page:Object3D Object3D]. Его цель - сделать работу с группами объектов синтаксически более понятной.

Пример

const geometry = new v3d.BoxBufferGeometry(1, 1, 1); const material = new v3d.MeshBasicMaterial({color: 0x00ff00}); const cubeA = new v3d.Mesh(geometry, material); cubeA.position.set(100, 100, 0); const cubeB = new v3d.Mesh(geometry, material); cubeB.position.set(-100, -100, 0); //create a group and add the two cubes //These cubes can now be rotated / scaled etc as a group const group = new v3d.Group(); group.add(cubeA); group.add(cubeB); app.scene.add(group);
