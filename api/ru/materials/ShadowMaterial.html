[name]

Этот материал может принимать тени, но в остальном полностью прозрачен.

Пример Кода

const geometry = new v3d.PlaneBufferGeometry(2000, 2000); geometry.rotateX(- Math.PI / 2); const material = new v3d.ShadowMaterial(); material.opacity = 0.2; const plane = new v3d.Mesh(geometry, material); plane.position.y = -200; plane.receiveShadow = true; scene.add(plane);

Примеры

[example:webgl_geometry_spline_editor geometry / spline / editor]

Конструктор

[name]([param:Object parameters])

[page:Object parameters] - (опционально) объект с одним или несколькими свойствами, определяющими внешний вид материала. Сюда можно передать любое свойство материала (включая любое свойство, унаследованное от [page:Material]).

Свойства

Общие свойства см. в базовых классах [page:Material].

[property:Boolean transparent]

Определяет, является ли этот материал прозрачным. По умолчанию *true*.
