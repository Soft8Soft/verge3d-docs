[page:Material] →

[name]

Материал для использования с [page:Sprite].

Пример

const map = new v3d.TextureLoader().load('textures/sprite.png'); const material = new v3d.SpriteMaterial({ map: map, color: 0xffffff }); const sprite = new v3d.Sprite(material); sprite.scale.set(200, 200, 1) app.scene.add(sprite);

Конструктор

[name]([param:Object parameters])

[page:Object parameters] — (необ.) объект с одним или несколькими свойствами, определяющими внешний вид материала. Сюда можно передать любое свойство материала (включая любое свойство, унаследованное от [page:Material]).

Исключением является свойство [page:Hexadecimal color], которое может быть передано в виде шестнадцатеричной строки и по умолчанию имеет значение *0xffffff* (белый). [page:Color.set](color) вызывается внутренне. SpriteMaterials не обрезаются при использовании [page:Material.clippingPlanes].

Свойства

Общие свойства см. в базовом классе [page:Material].

[property:Texture alphaMap]

Альфа-карта - это черно-белая текстура, которая контролирует непрозрачность поверхности. (черный: полностью прозрачный; белый: полностью непрозрачный). По умолчанию - null.

Используется только цвет текстуры, игнорируя альфа-канал, если он существует. Для текстур RGB и RGBA рендерер [page:WebGLRenderer WebGL] будет использовать зеленый канал при выборке этой текстуры из-за дополнительного бита точности, предоставляемого для зеленого цвета в DXT-сжатом и несжатом форматах RGB 565. Текстуры только с яркостью и яркость/альфа также будут работать, как и ожидалось.

[property:Color color]

[page:Color] материала, по умолчанию установлен на белый (0xffffffff). Значение [page:.map] умножается на цвет.

[property:Texture map]

Карта текстуры. По умолчанию null.

[property:Radians rotation]

Вращение спрайта в радианах. По умолчанию 0.

[property:Boolean sizeAttenuation]

Зависит ли размер спрайта от глубины камеры. (Только для камеры перспективы.) По умолчанию *true*.
