[name]

Создает текстуру на основе данных в сжатом виде, например, из файла [link:https://en.wikipedia.org/wiki/DirectDraw_Surface DDS] file.

Для использования с [page:CompressedTextureLoader CompressedTextureLoader].

Конструктор

[name]([param:Array mipmaps], [param:Number width], [param:Number height], [param:Constant format], [param:Constant type], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Number anisotropy])

[page:Array mipmaps] — Массив mipmaps должен содержать объекты с данными, шириной и высотой. Мипмапы должны иметь правильный формат и тип.
[page:Number width] — Ширина самого большого мипмапа.
[page:Number height] — Высота самого большого мипмапа.
[page:Constant format] — Формат, используемый в мипмапах. См. [page:Textures ST3C Compressed Texture Formats], [page:Textures PVRTC Compressed Texture Formats] и [page:Textures ETC Compressed Texture Format] для других вариантов.
[page:Constant type] — По умолчанию [page:Textures v3d.UnsignedByteType]. См. [page:Textures type constants] для других вариантов.
[page:Constant mapping] — Способ нанесения изображения на объект. Тип объекта [page:Textures v3d.UVMapping]. См. [page:Textures mapping constants] для других вариантов.
[page:Constant wrapS] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant wrapT] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant magFilter] — Как сэмплируется текстура, если тексель занимает более одного пикселя. По умолчанию [page:Textures v3d.LinearFilter]. См. [page:Textures magnification filter constants] для других вариантов.
[page:Constant minFilter] — Как сэмплируется текстура, если тексель занимает менее одного пикселя. По умолчанию [page:Textures v3d.LinearMipmapLinearFilter]. См. [page:Textures minification filter constants] для других вариантов.
[page:Number anisotropy] — Количество семплов, взятых вдоль оси, проходящей через пиксель, имеющий наибольшую плотность текселей. По умолчанию это значение равно 1. Большее значение дает менее размытый результат, чем базовый мипмап, ценой использования большего количества сэмплов текстуры. Используйте renderer.getMaxAnisotropy(), чтобы найти максимальное допустимое значение анизотропии для GPU; это значение обычно равно 2.

Свойства

Общие свойства см. в базовом классе [page:Texture Texture].

[property:Boolean flipY]

False по умолчанию. Переворачивание текстур не работает для сжатых текстур.

[property:Boolean generateMipmaps]

False по умолчанию. Мипмапы не могут быть сгенерированы для сжатых текстур.
