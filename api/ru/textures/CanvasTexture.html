[page:Texture] →

[name]

Создает текстуру из элемента canvas.

Это почти то же самое, что и базовый класс [page:Texture Texture], за исключением того, что он сразу устанавливает [page:Texture.needsUpdate needsUpdate] в *true*.

Конструктор

[name]([param:HTMLElement canvas], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Constant format], [param:Constant type], [param:Number anisotropy])

[page:HTMLElement canvas] — Элемент HTML canvas, из которого загружается текстура.
[page:Constant mapping] — Способ нанесения изображения на объект. Тип объекта [page:Textures v3d.UVMapping]. См. [page:Textures mapping constants] для других вариантов.
[page:Constant wrapS] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant wrapT] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant magFilter] — Способ дискретизации текстуры, когда тексель занимает более одного пикселя. По умолчанию [page:Textures v3d.LinearFilter]. См. [page:Textures magnification filter constants] для других вариантов.
[page:Constant minFilter] — Способ дискретизации текстуры, когда тексель занимает менее одного пикселя. По умолчанию [page:Textures v3d.LinearMipmapLinearFilter]. См. [page:Textures minification filter constants] для других вариантов.
[page:Constant format] — Формат, используемый в текстуре. См. [page:Textures format constants] для других вариантов.
[page:Constant type] --По умолчанию [page:Textures v3d.UnsignedByteType]. См. [page:Textures type constants] для других вариантов.
[page:Number anisotropy] — Количество семплов, взятых вдоль оси, проходящей через пиксель, имеющий наибольшую плотность текселей. По умолчанию это значение равно 1. Большее значение дает менее размытый результат, чем базовый мипмап, ценой использования большего количества сэмплов текстуры. Используйте [page:WebGLrenderer.getMaxAnisotropy renderer.getMaxAnisotropy](), чтобы найти максимальное допустимое значение анизотропии для GPU; обычно это значение равно 2.

Свойства

Общие свойства см. в базовом классе [page:Texture Texture].

[property:Boolean needsUpdate]

True по умолчанию. Это необходимо для того, чтобы данные canvas были загружены.
