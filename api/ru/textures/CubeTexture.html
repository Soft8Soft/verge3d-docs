[name]

Создает текстуру куба, состоящую из шести изображений.

Пример кода

const loader = new v3d.CubeTextureLoader(); loader.setPath('textures/cube/pisa/'); const textureCube = loader.load([ 'px.png', 'nx.png', 'py.png', 'ny.png', 'pz.png', 'nz.png' ]); const material = new v3d.MeshBasicMaterial({ color: 0xffffff, envMap: textureCube });

Конструктор

[name](images, mapping, wrapS, wrapT, magFilter, minFilter, format, type, anisotropy)

CubeTexture почти эквивалентен по функциональности и использованию [page:Texture]. Единственные различия заключаются в том, что изображения представляют собой массив из 6 изображений, а не одно изображение, и опциями отображения являются [page:Textures v3d.CubeReflectionMapping] (по умолчанию) или [page:Textures v3d.CubeRefractionMapping].
