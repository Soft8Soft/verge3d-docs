[page:Texture] →

[name]

Создает текстуру для использования с видеотекстурой.

Это почти то же самое, что и базовый класс [page:Texture Texture], за исключением того, что он постоянно устанавливает [page:Texture.needsUpdate needsUpdate] в *true*, чтобы текстура обновлялась по мере воспроизведения видео. Автоматическое создание [page:Texture.mipmaps mipmaps] также отключено.

Пример кода

// assuming you have created a HTML video element with id="video" const video = document.getElementById('video'); const texture = new v3d.VideoTexture(video);

Конструктор

[name]([param:Video video], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Constant format], [param:Constant type], [param:Number anisotropy])

[page:Video video] — Элемент видео, который будет использоваться в качестве текстуры.
[page:Constant mapping] — Как изображение наносится на объект. Тип объекта [page:Textures v3d.UVMapping]. См. [page:Textures mapping constants] для других вариантов.
[page:Constant wrapS] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant wrapT] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant magFilter] — Способ дискретизации текстуры, когда тексель занимает более одного пикселя. По умолчанию [page:Textures v3d.LinearFilter]. См. [page:Textures magnification filter constants] для других вариантов.
[page:Constant minFilter] — Способ дискретизации текстуры, когда тексель занимает менее одного пикселя. По умолчанию [page:Textures v3d.LinearMipmapLinearFilter]. См. [page:Textures minification filter constants] для других вариантов.
[page:Constant format] — Формат, используемый в текстуре. См. [page:Textures format constants] для других вариантов.
[page:Constant type] — По умолчанию [page:Textures v3d.UnsignedByteType]. См. [page:Textures type constants] для других вариантов.
[page:Number anisotropy] — Количество образцов, взятых вдоль оси, проходящей через пиксель, имеющий наибольшую плотность текселей. По умолчанию это значение равно 1. Большее значение дает менее размытый результат, чем базовый мипмап, за счет использования большего количества образцов текстуры. Используйте [page:WebGLrenderer.getMaxAnisotropy renderer.getMaxAnisotropy](), чтобы найти максимальное допустимое значение анизотропии для GPU; это значение обычно равно степени 2.

Свойста

См. базовый класс [page:Texture Texture] для общих свойств.

[property:Boolean needsUpdate]

Здесь не нужно устанавливать этот параметр вручную, так как он обрабатывается методом [page:VideoTexture.update update update].

Методы

См. базовый класс [page:Texture Texture] для общих методов.

[method:null update]()

Вызывается автоматически и устанавливает [property:Boolean needsUpdate] в значение *true* каждый раз, когда когда доступен новый кадр.
