[name]

Этот класс можно использовать для автоматического сохранения информации о глубине рендеринга в текстуру. При использовании контекста рендеринга WebGL 1, [имя] требует поддержку расширения [link:https://www.khronos.org/registry/webgl/extensions/WEBGL_depth_texture/ WEBGL_depth_texture].

Конструктор

[name]([param:Number width], [param:Number height], [param:Constant type], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Number anisotropy], [param:Constant format])

[page:Number width] — ширина текстуры.
[page:Number height] — высота текстуры.
[page:Constant type] — По умолчанию [page:Textures v3d.UnsignedShortType] при использовании [page:Textures DepthFormat] и [page:Textures v3d.UnsignedInt248Type] при использовании [page:Textures DepthStencilFormat]. Другие варианты см. в [page:Textures type constants].
[page:Constant mapping] -- См. [page:Textures mapping mode constants] для подробностей.
[page:Constant wrapS] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant wrapT] — По умолчанию [page:Textures v3d.ClampToEdgeWrapping]. См. [page:Textures wrap mode constants] для других вариантов.
[page:Constant magFilter] — Как сэмплируется текстура, если тексель занимает более одного пикселя. По умолчанию [page:Textures v3d.NearestFilter]. См. [page:Textures magnification filter constants] для других вариантов.
[page:Constant minFilter] — Как сэмплируется текстура, если тексель занимает менее одного пикселя. По умолчанию [page:Textures v3d.NearestFilter]. См. [page:Textures minification filter constants] для других вариантов.
[page:Number anisotropy] — Количество сэмплов, взятых вдоль оси, проходящей через пиксель, имеющий наибольшую плотность текселей. По умолчанию это значение равно 1. Большее значение дает менее размытый результат, чем базовый мипмап, ценой использования большего количества текстурных сэмплов. Используйте [page:WebGLrenderer.getMaxAnisotropy renderer.getMaxAnisotropy](), чтобы найти максимальное допустимое значение анизотропии для GPU; обычно это значение равно 2.
[page:Constant format] — должен быть либо [page:Textures DepthFormat] (по умолчанию) или [page:Textures DepthStencilFormat]. См. [page:Textures format constants] для подробностей.

Свойства

Общие свойства смотрите в базовом классе [page:Texture Texture]. Следующие свойства также являются частью класса текстуры, но имеют другие значения по умолчанию.

[page:Texture.format .format]

Либо [page:Textures DepthFormat] (по умолчанию) или [page:Textures DepthStencilFormat]. См. [page:Textures format constants] для подробностей.

[page:Texture.type .type]

По умолчанию [page:Textures v3d.UnsignedShortType] при использовании [page:Textures DepthFormat] и [page:Textures v3d.UnsignedInt248Type] при использовании [page:Textures DepthStencilFormat]. См. [page:Textures format constants] для подробностей.

[page:Texture.magFilter .magFilter]

Способ дискретизации текстуры, когда тексель занимает более одного пикселя. По умолчанию [page:Textures v3d.NearestFilter]. См. [page:Textures magnification filter constants] для других вариантов.

[page:Texture.minFilter .minFilter]

Способ дискретизации текстуры, когда тексель занимает менее одного пикселя. По умолчанию [page:Textures v3d.NearestFilter]. См. [page:Textures magnification filter constants] для других вариантов.

[page:Texture.flipY .flipY]

Глубинные текстуры не нужно переворачивать, поэтому по умолчанию это значение *false*.

[page:Texture.generateMipmaps .generateMipmaps]

Глубинные текстуры не используют мипмапы.

Методы

Общие методы смотрите в базовом классе [page:Texture Texture].
