[page:Texture] →

[name]

Создает трехмерную текстуру. Этот тип текстуры можно использовать только в контексте рендеринга WebGL 2.

Конструктор

[name]([param:TypedArray data], [param:Number width], [param:Number height], [param:Number depth])

[page:Object data] -- данные текстуры.
[page:Number width] -- ширина текстуры.
[page:Number height] -- высота текстуры.
[page:Number depth] -- глубина текстуры.

Примеры

[example:webgl2_materials_texture3d WebGL2 / materials / texture3d]

Свойства

Общие свойства смотрите в базовом классе [page:Texture Texture].

[property:number wrapR]

Определяет, как текстура будет развернута в направлении глубины.
По умолчанию [page:Textures v3d.ClampToEdgeWrapping], где край зажимается в тексели внешнего края. Два других варианта - [page:Textures v3d.RepeatWrapping] и [page:Textures v3d.MirroredRepeatWrapping]. См. [page:Textures texture constants] для подробностей.
