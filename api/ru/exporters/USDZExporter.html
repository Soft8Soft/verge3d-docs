[name]

Осуществляет экспорт в формат USDZ.

Пример

const usdzExporter = new v3d.USDZExporter(); const dataURL = await new Promise(function(resolve, reject) { usdzExporter.parse(app.scene).then(function(value) { const dataUrl = URL.createObjectURL(new Blob([value], { type: 'application/octet-stream' })); resolve(dataUrl); }, function(reason) { console.error('Ошибка экспорта в USDZ: ' + reason); reject(reason); }); });

Constructor

[name]()

Создаёт новый [name].

Methods

[method:undefined parse]([param:Object3D|Scene scene], [param:Object options])

[page:Object3D scene]
Экспортируемая сцена или массив объектов.
[page:Options options]
Опции экспорта:
  • [page:Boolean quickLookCompatible] — включить режим максимальной совместимости с форматом Apple Quick Look. По умолчанию `true`.
  • [page:Integer maxTextureSize] — ограничивает размер экспортируемых изображений (используется одновременно для ширины и высоты). По умолчанию `1024`.

Создаёт новый *.usdz*-ресурс на основе поданной на вход сцены или массива объектов.
