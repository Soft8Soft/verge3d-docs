[name]

Сцены позволяют установить, что и где будет рендерить «Вердж3Д». Здесь вы размещаете объекты, освещение и камеры.

Конструктор

[name]()

Создаёт новый объект сцены.

Свойства

[property:Object background]

Задает фон, используемый при рендеринге сцены. По умолчанию — `null`. Возможные значения:

  • [page:Color] — задаёт однородный цвет фона.
  • [page:Texture] — задаёт плоскую текстуру фона.
  • Кубические карта в виде [page:CubeTexture], либо текстура, использующая равнопромежуточную проекцию — задаёт небесную сферу.

Настройки камеры `zoom` и `view` игнорируются.

[property:Float backgroundBlurriness]

Коэффициент размытия фона. Поддерживается только, если в качестве [page:Scene.background] подана текстура окружения. Разрешённые значения лежат в интервале от `0` до `1`. Значение по умолчанию — `0`.

[property:Float backgroundIntensity]

Коэффициент затухания фона. Поддерживаются только если в качестве фона задана текстура. Значение по умолчанию — `1`.

[property:Boolean backgroundToneMapped]

Использовать отображение тонов при рендеринге фона. Значение по умолчанию — `true`.

[property:Texture environment]

Если значение не равно `null`, данная текстура устанавливается в качестве карты окружения для всех физических материалов в сцене. Однако невозможно перезаписать существующую текстуру, назначенную [page:MeshStandardMaterial.envMap]. По умолчанию — `null`.

[property:Fog fog]

Экземпляр [page:Fog fog], определяющий тип тумана, который влияет на все отрисованное в сцене. По умолчанию — `null`.

[property:Boolean isScene]

Показывает, что данный объект является экземпляром класса [name]. Только для чтения.

[property:Material overrideMaterial]

Если значение не равно null, это заставит все в сцене рендериться с этим материалом. По умолчанию — `null`.

[property:Array postprocessing]

Массив с пассами постобработки, используемыми для рендеринга сцены. Массив используется как временное хранилище, используемое [page:GLTFLoader].

[property:CubeReflectionProbe worldEnvMapProbe]

Экземпляр зонда отраженного света, который добавляет IBL-окружение ко всем объектам сцены.

[property:Material worldMaterial]

Материал используемый для создания окружения.
