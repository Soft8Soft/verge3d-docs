[name]

Этот класс определяет линейный туман, т.е. туман, линейно увеличивающийся по плотности с расстоянием.

Конструктор

[name]([param:Integer color], [param:Float near], [param:Float far])

Параметр `color` передается в конструктор [page:Color] для установки цвета тумана. Цвет может быть шестнадцатеричным целым числом или строкой CSS-стиля.

Свойства

[property:Boolean isFog]

Флаг только для чтения, обозначающий экземпляр класса [name].

[property:String name]

Необязательное имя объекта (не обязательно уникальное). По умолчанию это пустая строка.

[property:Color color]

Цвет тумана. Например, если установить значение красного цвета, то дальние объекты будут рендериться красными.

[property:Float near]

Минимальное расстояние для начала применения тумана. Объекты, находящиеся на расстоянии менее `near` от активной камеры, не будут затронуты туманом.

По умолчанию `1`.

[property:Float far]

Максимальное расстояние, на котором туман перестает рассчитываться и применяться. Объекты, находящиеся на расстоянии более чем `far` от активной камеры, не будут затронуты туманом.

По умолчанию `1000`.

Методы

[method:Fog clone]()

Возвращает новый экземпляр тумана с теми же параметрами, что и данный экземпляр.

Пазлы

Используйте пазл add fog, чтобы включить эффект линейного тумана без программирования.
