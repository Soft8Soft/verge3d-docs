[name]

Этот класс определяет экспоненциальный квадратичный туман, который дает четкий вид вблизи камеры и быстрее сгущается дальше от камеры.

Конструктор

[name]([param:Integer color], [param:Float density])

Параметр `color` передается в конструктор [page:Color] для установки цвета тумана. Цвет может быть шестнадцатеричным целым числом или строкой в CSS-стиля.

Свойства

[property:Boolean isFogExp2]

Флаг только для чтения, обозначающий экземпляр класса [name].

[property:String name]

Необязательное имя объекта (необязательно уникальное). По умолчанию это пустая строка.

[property:Color color]

Цвет тумана. Например если установить значение красного цвета, то дальние объекты будут рендериться красными.

[property:Float density]

Определяет, как быстро туман будет становиться плотным.

По умолчанию `0.00025`.

Методы

[method:FogExp2 clone]()

Возвращает новый экземпляр [name] с теми же параметрами, что и данный экземпляр.

Пазлы

Используйте пазл add fog, чтобы включить эффект экспоненциального тумана без программирования.
