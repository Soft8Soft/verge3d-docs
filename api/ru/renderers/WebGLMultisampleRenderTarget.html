[name]

Специальный render target, который может быть использован для применения многосемплерных рендербуферов. Внимание: [name] можно использовать только с контекстом рендеринга WebGL 2.

Конструктор

[name]([param:Number width], [param:Number height], [param:Object options])

[page:Float width] — Ширина render target.
[page:Float height] — Высота render target.
[page:Object options] — (необязательный) объект, содержащий параметры текстуры для автоматически генерируемой целевой текстуры и булевы depthBuffer/stencilBuffer.

Свойства

[property:number samples]

Указывает количество семплов, которые будут использоваться для хранения рендербуфера. Однако, максимальный поддерживаемый размер мультисэмплинга зависит от платформы и определяется через *gl.MAX_SAMPLES*..
