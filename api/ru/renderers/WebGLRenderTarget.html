[name]

[link:https://webglfundamentals.org/webgl/lessons/webgl-render-to-texture.html Render target] это буфер куда видеокарта выводит пиксели для сцены, которая рендерится в фоновом режиме. Он используется в различных эффектах, например, для применения постобработки к отрисованному изображению перед его выводом на экран.

Конструктор

[name]([param:Number width], [param:Number height], [param:Object options])

[page:Float width] — Ширина renderTarget.
[page:Float height] — Высота renderTarget.
options - (необязательный объект, содержащий параметры текстуры для автогенерируемой целевой текстуры и булевы depthBuffer/stencilBuffer. Объяснение параметров текстуры см. в [page:Texture Texture]. Ниже перечислены допустимые параметры:

[page:Constant wrapS] — по умолчанию [page:Textures ClampToEdgeWrapping].
[page:Constant wrapT] — по умолчанию [page:Textures ClampToEdgeWrapping].
[page:Constant magFilter] — по умолчанию [page:Textures LinearFilter].
[page:Constant minFilter] — по умолчанию [page:Textures LinearFilter].
[page:Constant format] — по умолчанию [page:Textures RGBAFormat].
[page:Constant type] — по умолчанию [page:Textures UnsignedByteType].
[page:Number anisotropy] — по умолчанию *1*. См. [page:Texture.anisotropy]
[page:Constant encoding] — по умолчанию [page:Textures LinearEncoding].
[page:Boolean depthBuffer] — по умолчанию *true*. Установите значение false, если он вам не нужен.
[page:Boolean stencilBuffer] — по умолчанию *false*. Установите значение true, если он вам не нужен.

Creates a new [name]

Свойства

[property:number width]

Ширина render target.

[property:number height]

Высота render target.

[property:Vector4 scissor]

Прямоугольная область внутри вьюпорта объекта рендеринга. Фрагменты, находящиеся за пределами этой области, будут игнорированы.

[property:Boolean scissorTest]

Указывает, активен или нет ножницный тест.

[property:Vector4 viewport]

Вьюпорт render target.

[property:Texture texture]

Этот экземпляр текстуры содержит отрисованные пиксели. Используйте его в качестве исходного материала для дальнейшей обработки.

[property:Boolean depthBuffer]

Рендеринг в буфер глубины. По умолчанию - true.

[property:Boolean stencilBuffer]

Рендеринг в буфер стенсила. По умолчанию false.

[property:DepthTexture depthTexture]

Если установлено, то глубина сцены будет отрисовываться на эту текстуру. По умолчанию - null.
