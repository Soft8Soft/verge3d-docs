[name]

Используется [page:CubeCamera] в качестве [page:WebGLRenderTarget]..

Примеры

См. [page:CubeCamera] для примеров.

Конструктор

[name]([param:Number size], [param:Object options])

[page:Float size] — размер, в пикселях.
options - (необязательный) объект, содержащий параметры текстуры для автогенерируемой целевой текстуры и булевых значений depthBuffer/stencilBuffer. Описание параметров текстуры см. в [page:Texture Texture]. Ниже перечислены допустимые параметры:

[page:Constant wrapS] — по умолчанию [page:Textures ClampToEdgeWrapping].
[page:Constant wrapT] — по умолчанию [page:Textures ClampToEdgeWrapping].
[page:Constant magFilter] — по умолчанию [page:Textures .LinearFilter].
[page:Constant minFilter] — по умолчанию [page:Textures LinearFilter].
[page:Constant format] — по умолчанию [page:Textures RGBAFormat].
[page:Constant type] — по умолчанию [page:Textures UnsignedByteType].
[page:Number anisotropy] — по умолчанию *1*. См. [page:Texture.anistropy]
[page:Constant encoding] — по умолчанию [page:Textures LinearEncoding].
[page:Boolean depthBuffer] — по умолчанию *true*. Установите значение false, если он вам не нужен.
[page:Boolean stencilBuffer] — по умолчанию *false*. Установите значение true, если он вам не нужен.

Методы

[method:WebGLCubeRenderTarget fromEquirectangularTexture]([param:WebGLRenderer renderer], [param:Texture texture])

[page:WebGLRenderer renderer] — рендерер.
[page:Texture texture] — равносторонняя текстура.

Используйте этот метод, если вы хотите преобразовать равноугольную панораму в формат cubemap.

[method:undefined clear]([param:WebGLRenderer renderer], [param:Boolean color], [param:Boolean depth], [param:Boolean stencil])

Используйте эту функцию, чтобы очистить буферы цвета, глубины и/или стенсила render target. Цветовой буфер устанавливается в текущий чистый цвет рендерера. Аргументы по умолчанию имеют значение *true*.
