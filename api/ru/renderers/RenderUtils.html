[name]

Класс, содержащий полезные методы рендеринга.

Методы

[method:Integer assignAllLayersMask]([param:Object3D obj])

Назначить все используемые в «Вердж3Д» [page:Layers слои] для данного [page:Object3D объекта]. Данная функция возвращает маску слоёв, которая была назначена на объекте до переназначения.

[method:WebGLCubeRenderTarget renderSceneToCubemap]([param:Scene scene], [param:WebGLRenderer renderer], [param:WebGLCubeRenderTarget cubeRenderTarget], [param:Object cameraOptions])

Рендерит заданную сцены в кубическую карту. Параметры:

scene
[page:Scene Сцена] для рендеринга в кубическую карту
renderer
[page:WebGLRenderer]
cubeRenderTarget
[page:WebGLCubeRenderTarget] содержащий кубическую карту для рендеринга
cameraOptions

объект, содержащий параметры кубической камеры, используемой для рендеринга сцены в кубическую карту; параметры и их значения по умолчанию:

  • camPos — позиция кубической камеры
  • clipStart — значение ближней плоскости отсечения
  • clipEnd — значение дальней плоскости отсечения
{ camPos: new Vector3(), clipStart: 0.1, clipEnd: 100, }

[method:WebGLCubeRenderTarget renderWorldNodeMatToCubemap]([param:WebGLRenderer renderer], [param:MeshNodeMaterial material], [param:Integer cubeRes], [param:Object options])

Рендерит нодовый материал в кубическую карту. Параметры:

renderer
[page:WebGLRenderer]
material
материал ноды мира, должен иметь тип [page:MeshNodeMaterial]
cubeRes
разрешение кубической карты
options
(необ.) объект, содержащий параметры текстуры для автоматически генерируемого кубического рендер-таргета; имеет тот же формат что и параметр `options` конструктора [page:WebGLCubeRenderTarget]

Полученный [page:WebGLCubeRenderTarget render target] с текстурой куба можно использовать в качестве фона для сцены.
