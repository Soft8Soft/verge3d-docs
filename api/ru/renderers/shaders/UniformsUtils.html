[name]

Предоставляет полезные функции для управления переменными.

Методы

[method:Object clone]([param:Object src])

src -- Объект, представляющий определения переменных.

Клонирует заданные определения переменных, выполняя глубокую копию. Это означает, что если [page:Uniform.value value] переменной ссылается на объект, например [page:Vector3] или [page:Texture], то клонированная переменная будет ссылаться на новый объект.

[method:Object merge]([param:Array uniforms])

uniforms -- Массив объектов, содержащих определения переменных..

Объединяет заданные определения переменных в один объект. Поскольку метод внутренне использует [page:.clone](), он выполняет глубокую копию при создании объединенных определений переменных. [sourceHint] [sourceHint]
