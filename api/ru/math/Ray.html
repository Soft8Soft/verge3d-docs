[name]

Луч, который испускается из начала координат в определенном направлении. Он используется [page:Raycaster] для помощи [link:https://en.wikipedia.org/wiki/Ray_casting raycasting]. Raycasting используется для пикинга мыши (определение объектов в 3D-пространстве, над которыми находится мышь), а также для других целей.

Конструктор

[name]([param:Vector3 origin], [param:Vector3 direction])

[page:Vector3 origin] — (необязательно) начало координат [page:Ray]. По умолчанию это [page:Vector3] в точке (0, 0, 0)..
[page:Vector3 direction] - [page:Vector3] Направление [page:Ray]. Оно должно быть нормализовано (с помощью [page:Vector3.normalize]), чтобы методы работали правильно. По умолчанию это [page:Vector3] в точке (0, 0, -1).

Создает новый [name].

Свойства

[property:Vector3 origin]

Начало координат [page:Ray]. По умолчанию это [page:Vector3] в точке (0, 0, 0).

[property:Vector3 direction]

Направление [page:Ray]. Оно должно быть нормализовано (с помощью [page:Vector3.normalize]), чтобы методы работали правильно. По умолчанию это [page:Vector3] в точке (0, 0, -1).

Методы

[method:Ray applyMatrix4]([param:Matrix4 matrix4])

[page:Matrix4 matrix4] - [page:Matrix4] для применения к данному [page:Ray].

Преобразуйте этот [page:Ray] с помощью [page:Matrix4].

[method:Vector3 at]([param:Float t], [param:Vector3 target])

[page:Float t] — расстояние вдоль [page:Ray] для получения позиции.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Получить [page:Vector3], который находится на заданном расстоянии вдоль данного [page:Ray].

[method:Ray clone]()

Создает новый Ray c [page:.origin origin] и [page:.direction direction] идентичными данному.

[method:Vector3 closestPointToPoint]([param:Vector3 point], [param:Vector3 target])

[page:Vector3 point] — точка, к которой нужно подойти ближе всего.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Получите точку вдоль данного [page:Ray], которая ближе всего к заданному [page:Vector3].

[method:Ray copy]([param:Ray ray])

Копирует свойства [page:.origin origin] и [page:.direction direction direction] из [page:Ray ray] в этот луч.

[method:Float distanceSqToPoint]([param:Vector3 point])

[page:Vector3 point] - [page:Vector3] для вычисления расстояния до.

Получите квадратичное расстояние ближайшего приближения между [page:Ray] и [page:Vector3].

[method:Float distanceSqToSegment]([param:Vector3 v0], [param:Vector3 v1], [param:Vector3 optionalPointOnRay], [param:Vector3 optionalPointOnSegment])

[page:Vector3 v0] — начало отрезка линии.
[page:Vector3 v1] — конец отрезка линии.
optionalPointOnRay - (необязательно), если указано, получает точку на данном [page:Ray], которая ближе всего к отрезку.
optionalPointOnSegment - (необязательно), если указано, получает точку на отрезке прямой, которая ближе всего к данному [page:Ray].

Получите квадратичное расстояние между данным [page:Ray] и отрезком прямой.

[method:Float distanceToPlane]([param:Plane plane])

[page:Plane plane] - [page:Plane] для получения расстояния до.

Получите расстояние от [page:.origin origin] до [page:Plane], или *null*, если [page:Ray] не пересекает [page:Plane].

[method:Float distanceToPoint]([param:Vector3 point])

[page:Vector3 point] - [page:Vector3] для вычисления расстояния.

Получает расстояние ближайшего сближения между [page:Ray] и точкой [page:Vector3].

[method:Boolean equals]([param:Ray ray])

[page:Ray ray] - [page:Ray] для сравнения.

Возвращает true, если этот и другой [page:Ray ray] имеют одинаковые [page:.origin origin] и [page:.direction direction].

[method:Vector3 intersectBox]([param:Box3 box], [param:Vector3 target])

[page:Box3 box] - [page:Box3] для интерсекта.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Пересекает данный [page:Ray] с [page:Box3], возвращая точку пересечения или *null*, если пересечения нет.

[method:Vector3 intersectPlane]([param:Plane plane], [param:Vector3 target])

[page:Plane plane] - [page:Plane] для интерсекта.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Пересекает данный [page:Ray] с [page:Plane], возвращая точку пересечения или *null*, если пересечения нет.

[method:Vector3 intersectSphere]([param:Sphere sphere], [param:Vector3 target])

[page:Sphere sphere] - [page:Sphere] для интерсекта.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Пересекает данный [page:Ray] с [page:Sphere], возвращая точку пересечения или *null*, если пересечения нет.

[method:Vector3 intersectTriangle]([param:Vector3 a], [param:Vector3 b], [param:Vector3 c], [param:Boolean backfaceCulling], [param:Vector3 target])

[page:Vector3 a], [page:Vector3 b], [page:Vector3 c] — Точки [page:Vector3], составляющие треугольник.
[page:Boolean backfaceCulling] — использовать ли backface culling.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Пересекает данный [page:Ray] с треугольником, возвращая точку пересечения или *null*, если пересечения нет.

[method:Boolean intersectsBox]([param:Box3 box])

[page:Box3 box] - [page:Box3] для интерсекта.

Возвращает true, если данный [page:Ray] пересекается с [page:Box3].

[method:Boolean intersectsPlane]([param:Plane plane])

[page:Plane plane] - [page:Plane] для пересечения.

Возвращает true, если данный [page:Ray] пересекается с [page:Plane].

[method:Boolean intersectsSphere]([param:Sphere sphere])

[page:Sphere sphere] - [page:Sphere] для пересечения.

Возвращает true, если данный [page:Ray] пересекается с [page:Sphere].

[method:Ray lookAt]([param:Vector3 v])

[page:Vector3 v] - [page:Vector3] для направления.

Регулирует направление луча так, чтобы он указывал на вектор в мировых координатах.

[method:Ray recast]([param:Float t])

[page:Float t] — Расстояние вдоль [page:Ray] для интерполяции.

Устанавливает свойства [page:.origin origin] и [page:.direction direction direction] этого луча, копируя значения из заданных объектов.

[method:Ray set]([param:Vector3 origin], [param:Vector3 direction])

[page:Vector3 origin] - [page:.origin origin] данного [page:Ray].
[page:Vector3 origin] - [page:.direction direction] данного [page:Ray]. Для правильной работы методов его необходимо нормализовать (с помощью [page:Vector3.normalize]).

Устанавливает свойства [page:.origin origin] и [page:.direction direction] этого луча, копируя значения из заданных объектов.
