[name]

Представляет сферическую гармонику третьего порядка (SH). Световые зонды используют этот класс для кодирования информации об освещении.

Конструктор

[name]()

Создает новый экземпляр [name].

Свойства

[property:Array coefficients]

Массив, содержащий (9) коэффициентов SH. Один коэффициент представлен как экземпляр [page:Vector3]..

Методы

[method:SphericalHarmonics3 add]([param:SphericalHarmonics3 sh])

[page:SphericalHarmonics3 sh] - SH, чтобы добавить.

Добавляет данную SH к данному экземпляру.

[method:SphericalHarmonics3 addScaledSH]([param:SphericalHarmonics3 sh], [param:Number scale])

[page:SphericalHarmonics3 sh] - SH, чтобы добавить.
[page:Number scale] - Масштабный коэффициент.

Удобный метод для одновременного выполнения [page:.add]() и [page:.scale]().

[method:SphericalHarmonics3 clone]()

Возвращает новый экземпляр [name] с равными коэффициентами.

[method:SphericalHarmonics3 copy]([param:SphericalHarmonics3 sh])

[page:SphericalHarmonics3 sh] - SH для копирования.

Копирует заданую SH в данный экземпляр.

[method:Boolean equals]([param:SphericalHarmonics3 sh])

[page:SphericalHarmonics3 sh] - SH для сравнения.

Возвращает true, если SH и данный экземпляр имеют одинаковые коэффициенты.

[method:SphericalHarmonics3 fromArray]([param:Array array], [param:Number offset])

[page:Array array] - Массив, содержащий номера коэффициентов SH.
[page:Number offset] - (необязательно) Смещение массива.

Устанавливает коэффициенты данного экземпляра из заданного массива.

[method:Vector3 getAt]([param:Vector3 normal], [param:Vector3 target])

[page:Vector3 normal] - Нормальный вектор (предполагается единичной длины).
[page:Vector3 target] - Вектор результата.

Возвращает освещенность в направлении заданной нормали.

[method:Vector3 getIrradianceAt]([param:Vector3 normal], [param:Vector3 target])

[page:Vector3 normal] - Вектор нормали(предполагается единичной длины).
[page:Vector3 target] - Результирующий вектор.

Возвращает irradiance(излучение) (radiance(освещенность), преобразованная с помощью косинусной доли) в направлении заданной нормали.

[method:SphericalHarmonics3 lerp]([param:SphericalHarmonics3 sh], [param:Number alpha])

[page:SphericalHarmonics3 sh] - SH для интерполяции.
[page:Number alpha] - Альфа-фактор.

Линейная интерполяция между заданной SH и данным экземпляром по заданному альфа-фактору.

[method:SphericalHarmonics3 scale]([param:Number scale])

[page:Number scale] - Масштабный коэффициент.

Масштабирует данную SH на заданный масштабный коэффициент.

[method:SphericalHarmonics3 set]([param:Array coefficients])

[page:Array coefficients] - Массив коэффициентов SH.

Устанавливает заданные коэффициенты SH для данного экземпляра.

[method:Array toArray]([param:Array array], [param:Number offset])

[page:Array array] - (необязательно) Целевой массив.
[page:Number offset] - (необязательно) Смещение массива.

Возвращает массив с коэффициентами или копирует их в предоставленный массив. Коэффициенты представляются в виде чисел.

[method:SphericalHarmonics3 zero]()

Устанавливает все коэффициенты SH в 0.

Статические методы

[method:void getBasisAt]([param:Vector3 normal], [param:Array shBasis])

[page:Vector3 normal] - вектор нормали (предполагается единичной длины).
[page:Array shBasis] - полученный базис SH.

Вычисляет базис SH для заданного вектора нормали.
