[name]

Двумерная поверхность, бесконечно простирающаяся в трехмерном пространстве, представленная в [link:http://mathworld.wolfram.com/HessianNormalForm.html Hessian normal form] нормальным вектором единичной длины и константой.

Конструктор

[name]([param:Vector3 normal], [param:Float constant])

[page:Vector3 normal] — (необязательно) единица длины [page:Vector3], определяющая нормаль плоскости. По умолчанию *(1, 0, 0)*.
[page:Float constant] — (необязательно) знаковое расстояние от начала координат до плоскости. По умолчанию *0*.

Свойства

[property:Vector3 normal]

[property:Float constant]

Methods

[method:Plane applyMatrix4]([param:Matrix4 matrix], [param:Matrix3 optionalNormalMatrix])

[page:Matrix4 matrix] — для применения [Page:Matrix4].
[page:Matrix3 optionalNormalMatrix] — (необязательно) предварительно вычисленная нормаль [Page:Matrix3] применяемой Matrix4.

Примените Matrix4 к плоскости. Матрица должна быть аффинным, однородным преобразованием.
Если поддерживается [page:Matrix3 optionalNormalMatrix], то она может быть создана следующим образом: const optionalNormalMatrix = new v3d.Matrix3().getNormalMatrix(matrix);

[method:Plane clone]()

Возвращает новую плоскость с теми же [page:.normal normal] и [page:.constant constant], что и данная.

[method:Vector3 coplanarPoint]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает [page:Vector3], компланарный плоскости, путем вычисления проекции вектора нормали в начале координат на плоскость.

[method:Plane copy]([param:Plane plane])

Копирует значения свойств [page:.normal normal] и [page:.constant constant] переданной плоскости в эту плоскость.

[method:Float distanceToPoint]([param:Vector3 point])

Возвращает знаковое расстояние от точки [page:Vector3] до плоскости.

[method:Float distanceToSphere]([param:Sphere sphere])

Возвращает знаковое расстояние от сферы [page:Sphere sphere] до плоскости.

[method:Boolean equals]([param:Plane plane])

Проверяет, равны ли две плоскости (их свойства [page:.normal normal] и [page:.constant constant] совпадают).

[method:Vector3 intersectLine]([param:Line3 line], [param:Vector3 target])

[page:Line3 line] - [page:Line3] для проверки пересечения.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает точку пересечения переданной прямой и плоскости. Возвращает неопределенное значение, если линия не пересекается. Возвращает начальную точку линии, если линия компланарна плоскости.

[method:Boolean intersectsBox]([param:Box3 box])

[page:Box3 box] - [page:Box3] для проверки пересечения.

Определяет, пересекает ли данная плоскость [page:Box3 box].

[method:Boolean intersectsLine]([param:Line3 line])

[page:Line3 line] - [page:Line3] для проверки пересечения.

Проверяет, пересекается ли отрезок прямой с плоскостью (проходит через нее).

[method:Boolean intersectsSphere]([param:Sphere sphere])

[page:Sphere sphere] - [page:Sphere] для проверки пересечения.

Определяет, пересекает ли данная плоскость [page:Sphere sphere].

[method:Plane negate]()

Отрицает как нормальный вектор, так и константу.

[method:Plane normalize]()

Нормализует вектор [page:.normal normal] и соответствующим образом корректирует значение [page:.constant constant].

[method:Vector3 projectPoint]([param:Vector3 point], [param:Vector3 target])

[page:Vector3 point] - [page:Vector3] для проецирования на плоскость.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Проецирует точку [page:Vector3] на плоскость.

[method:Plane set]([param:Vector3 normal], [param:Float constant])

[page:Vector3 normal] — единичная длина [page:Vector3], определяющая нормаль к плоскости.
[page:Float constant] — знаковое расстояние от начала координат до плоскости. По умолчанию *0*.

Устанавливает свойства [page:.normal normal] и [page:.constant constant] этой плоскости, копируя значения из заданной нормали.

[method:Plane setComponents]([param:Float x], [param:Float y], [param:Float z], [param:Float w])

[page:Float x] — x значение нормального вектора единичной длины.
[page:Float y] — значение y нормального вектора единичной длины.
[page:Float z] — значение z нормального вектора единичной длины.
[page:Float w] — значение свойства [page:.constant constant] плоскости.

Установите отдельные компоненты, определяющие плоскость.

[method:Plane setFromCoplanarPoints]([param:Vector3 a], [param:Vector3 b], [param:Vector3 c])

[page:Vector3 a] — первая точка на плоскости.
[page:Vector3 b] — вторая точка на плоскости.
[page:Vector3 c] — третья точка на плоскости.

Определяет плоскость на основе 3 заданных точек. Порядок предполагается против часовой стрелки и определяет направление [page:.normal normal].

[method:Plane setFromNormalAndCoplanarPoint]([param:Vector3 normal], [param:Vector3 point]) [param:Vector3 this]

[page:Vector3 normal] — единичная длина [page:Vector3], определяющая нормаль к плоскости.
[page:Vector3 point] - [page:Vector3]

Задает свойства плоскости, определяемые [page:Vector3 normal] и произвольной компланарной [page:Vector3 point].

[method:Plane translate]([param:Vector3 offset])

[page:Vector3 offset] — величина, на которую нужно переместить плоскость.

Переводит плоскость на расстояние, определяемое вектором [page:Vector3 offset].. Обратите внимание, что это влияет только на константу плоскости и не влияет на вектор нормали.
