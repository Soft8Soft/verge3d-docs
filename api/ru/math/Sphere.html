[name]

Сфера, определяемая центром и радиусом.

Конструктор

[name]([param:Vector3 center], [param:Float radius])

[page:Vector3 center] - центр сферы. По умолчанию это [page:Vector3] в точке (0, 0, 0)..
[page:Float radius] - радиус сферы. По умолчанию -1.

Создает новый [name].

Свойства

[property:Vector3 center]

[page:Vector3], определяющий центр сферы. По умолчанию (0, 0, 0)..

[property:Float radius]

Радиус сферы. По умолчанию равен 0.

Методы

[method:Sphere applyMatrix4]([param:Matrix4 matrix])

[page:Matrix4 matrix] - [Page:Matrix4] для применения

Преобразует эту сферу с помощью предоставленной [page:Matrix4].

[method:Vector3 clampPoint]([param:Vector3 point], [param:Vector3 target])

[page:Vector3 point] - [page:Vector3] Точка для ограничения.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Ограничивает точку внутри сферы. Если точка находится вне сферы, то она будет закреплена в ближайшей точке на краю сферы. Точки, уже находящиеся внутри сферы, не будут затронуты.

[method:Sphere clone]()

Возвращает новую сферу с теми же [page:.center center] и [page:.radius radius], что и данная.

[method:Boolean containsPoint]([param:Vector3 point])

[page:Vector3 point] - [page:Vector3] для проверки

Проверяет, содержит ли сфера предоставленную точку [page:Vector3 point], включая поверхность сферы.

[method:Sphere copy]([param:Sphere sphere])

Копирует значения свойств [page:.center center] и [page:.radius radius] переданной сферы в эту сферу.

[method:Float distanceToPoint]([param:Vector3 point])

Возвращает ближайшее расстояние от границы сферы до точки [page:Vector3]. Если сфера содержит точку, расстояние будет отрицательным.

[method:Boolean isEmpty]()

Проверяет, пуста ли сфера (радиус установлен в отрицательное число).
Сферы с радиусом 0 содержат только свою центральную точку и не считаются пустыми.

[method:Sphere makeEmpty]()

Сделайте сферу пустой, установив [page:.center center] в (0, 0, 0) и [page:.radius radius] в -1.

[method:Boolean equals]([param:Sphere sphere])

Проверяет, равны ли центры и радиусы двух сфер.

[method:Box3 getBoundingBox]([param:Box3 target])

[page:Box3 target] — результат будет скопирован в этот Box3.

Возвращает[link:https://en.wikipedia.org/wiki/Minimum_bounding_box Minimum Bounding Box] для сферы.

[method:Boolean intersectsBox]([param:Box3 box])

[page:Box3 box] - [page:Box3] для проверки пересечения.

Определяет, пересекает ли данная сфера заданный [page:Box3 box].

[method:Boolean intersectsPlane]([param:Plane plane])

[page:Plane plane] - Плоскость для проверки пересечения.

Определяет, пересекает ли данная сфера заданную [page:Plane plane].

[method:Boolean intersectsSphere]([param:Sphere sphere])

[page:Sphere sphere] - Sphere для проверки пересечения.

Проверяет, пересекаются ли две сферы.

[method:Sphere set]([param:Vector3 center], [param:Float radius])

[page:Vector3 center] - центр сферы.
[page:Float radius] - радиус сферы.

Устанавливает свойства [page:.center center] и [page:.radius radius] этой сферы.
Обратите внимание, что этот метод копирует только значения из заданного центра.

[method:Sphere setFromPoints]([param:Array points], [param:Vector3 optionalCenter])

[page:Array points] - [page:Array] из позиций [page:Vector3].
[page:Vector3 optionalCenter] - Необязательное [page:Vector3] положение для центра сферы.

Вычисляет минимальную ограничивающую сферу для массива [page:Array points]. Если задан [page:Vector3 optionalCenter], то он используется в качестве центра сферы. В противном случае вычисляется центр выровненного по оси ограничивающего поля, охватывающего [page:Array points].

[method:Sphere translate]([param:Vector3 offset])

Переместите центр сферы на заданное смещение [page:Vector3].
