[name]

Этот власс создан в основном для внутреннего использования в [page:DirectionalLightShadowCSM DirectionalLightShadowCSM] для расчета под-фрустумов.

Конструктор

[name]()

Создает новый [name].

Свойства

[property:Object vertices]

Содержит 8 точек, которые формируют фрустум.

[page:Array near] — массив [page:Vector3 точек] формирующие ближню плоскость отсечения.
[page:Array far] — массив [page:Vector3 точек] формирующие дальную плоскость отсечения.

Методы

[method:this setFromProjectionMatrix]([param:Matrix4 matrix], [param:Float maxFar])

[page:Matrix4 matrix] — матрициа проекции [page:Camera камеры] вида.
[page:Float maxFar] — растояние до дальней плоскости отсечения.

Вычисляет точки фрустума из матрицы проекции.

[method:this split]([param:Array breaks], [param:Array target], [param:Boolean fitToScene])

[page:Array breaks] — массив интервалов.
[page:Array target] — массив [page:CSMFrustum CSMFrustum], `target.length <= breaks.length` условие должно быть верным.
[page:Boolean fitToScene] — при значении *true*, использует метод *fit to scene*, иначе метод *fit to cascade*.

Расделяет этот фрустум на под-фрустумы.

[method:this toSpace]([param:Matrix4 matrix], [param:this target])

[page:Matrix4 matrix] — матрица применяемая к каждой точке фрустума.
[page:this target] — [page:CSMFrustum фрустум] в который будет записан результат.

Умножает каждую точку фрустума на matrix и записывает результат в target.
