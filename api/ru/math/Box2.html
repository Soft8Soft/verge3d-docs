[name]

Представляет выровненную по оси ограничительную рамку (AABB) в двумерном пространстве.

Конструктор

[name]([param:Vector2 min], [param:Vector2 max])

[page:Vector2 min] - (необязательно) [page:Vector2], представляющий нижнюю (x, y) границу куба. По умолчанию (+ бесконечность, + бесконечность).
[page:Vector2 max] - (необязательно) [page:Vector2], представляющий верхнюю (x, y) границу куба. По умолчанию (- Бесконечность, - Бесконечность)..

Создает [name], ограниченное min и max.

Свойства

[property:Vector2 min]

[page:Vector2] представляет собой нижнюю (x, y) границу куба.
По умолчанию (+ бесконечность, + бесконечность).

[property:Vector2 max]

[page:Vector2] представляет собой нижнюю верхнюю (x, y) границу куба.
По умолчанию (- Бесконечность, - Бесконечность).

Методы

[method:Vector2 clampPoint]([param:Vector2 point], [param:Vector2 target])

[page:Vector2 point] - зафиксировать [page:Vector2].
[page:Vector2 target] — результат будет скопирован в этот Vector2.

[link:https://en.wikipedia.org/wiki/Clamping_(graphics) Clamps] точка [page:Vector2 point] в границах этого поля.

[method:Box2 clone]()

Возвращает новый [page:Box2] с теми же [page:.min min] и [page:.max max], что и данный.

[method:Boolean containsBox]([param:Box2 box])

[page:Box2 box] - [page:Box2 Box2] для проверки на включение.

Возвращает true, если это поле включает в себя все поле [page:Box2 box]. Если это и [page:Box2 box] идентичны,
эта функция также возвращает true.

[method:Boolean containsPoint]([param:Vector2 point])

[page:Vector2 point] - [page:Vector2] для проверки на включение.

Возвращает true, если указанная точка [page:Vector2 point] лежит внутри или на границах данного поля.

[method:Box2 copy]([param:Box2 box])

Копирует значения [page:.min min] и [page:.max max] из [page:Box2 box] в этот box.

[method:Float distanceToPoint]([param:Vector2 point])

[page:Vector2 point] - [page:Vector2] для измерения расстояния.

Возвращает расстояние от любого края этого поля до указанной точки. Если точка [page:Vector2 point] лежит внутри этой области, расстояние будет равно 0.

[method:Boolean equals]([param:Box2 box])

[page:Box2 box] - Box для сравнения с этим.

Возвращает true, если у этого поля и [page:Box2 box] одинаковые нижняя и верхняя границы.

[method:Box2 expandByPoint]([param:Vector2 point])

[page:Vector2 point] - [page:Vector2], которые должны быть включены в поле.

Расширяет границы этой области, включая [page:Vector2 point].

[method:Box2 expandByScalar]([param:Float scalar])

[page:Float scalar] - Расстояние, на которое нужно увеличить box.

Расширяет каждое измерение поля на [page:Float scalar]. При отрицательном значении размеры ячейки сужаются.

[method:Box2 expandByVector]([param:Vector2 vector])

[page:Vector2 vector] - [page:Vector2] для расширения поля.

Равностороннее расширение этого поля на [page:Vector2 vector]. Ширина этого поля будет увеличена на x-компонент [page:Vector2 vector] в обоих направлениях. Высота этого поля будет увеличена на y-компонент [page:Vector2 vector] в обоих направлениях.

[method:Vector2 getCenter]([param:Vector2 target])

[page:Vector2 target] — результат будет скопирован в этот Vector2.

Возвращает центральную точку поля в виде [page:Vector2].

[method:Vector2 getParameter]([param:Vector2 point], [param:Vector2 target])

[page:Vector2 point] - [page:Vector2].
[page:Vector2 target] — результат будет скопирован в этот Vector2.

Возвращает точку как пропорцию ширины и высоты данного поля.

[method:Vector2 getSize]([param:Vector2 target])

[page:Vector2 target] — результат будет скопирован в этот Vector2.

Возвращает ширину и высоту этого поля.

[method:Box2 intersect]([param:Box2 box])

[page:Box2 box] - Box, с которым будет пересечения.

Возвращает пересечение этого и [page:Box2 box], устанавливая верхнюю границу этого бокса на меньшую из верхних границ двух боксов, а нижнюю границу этого бокса на большую из нижних границ двух боксов.

[method:Boolean intersectsBox]([param:Box2 box])

[page:Box2 box] - Box для проверки пересечения.

Определяет, пересекает ли данный бокс [page:Box2 box].

[method:Boolean isEmpty]()

Возвращает true, если эта область включает нулевые точки в своих границах.
Обратите внимание, что квадрат с равными нижней и верхней границами все еще включает одну точку, общую для обеих границ.

[method:Box2 makeEmpty]()

Делает этот box пустым.

[method:Box2 set]([param:Vector2 min], [param:Vector2 max])

[page:Vector2 min] - (требуется) [page:Vector2], представляющий нижнюю (x, y) границу бокса.
[page:Vector2 max] - (требуется) [page:Vector2], представляющий верхнюю (x, y) границу бокса.

Устанавливает нижнюю и верхнюю (x, y) границы данного бокса.
Обратите внимание, что этот метод только копирует значения из заданных объектов.

[method:Box2 setFromCenterAndSize]([param:Vector2 center], [param:Vector2 size])

[page:Vector2 center] - Желаемое положение центра бокса ([page:Vector2]).
[page:Vector2 size] - Желаемые размеры x и y бокса ([page:Vector2]).

Центрирует этот бокс по [page:Vector2 center] и устанавливает ширину и высоту этого бокса равными значениям, указанным в [page:Vector2 size].

[method:Box2 setFromPoints]([param:Array points])

[page:Array points] - Массив [page:Vector2 Vector2s], который будет содержать итоговый бокс.

Устанавливает верхнюю и нижнюю границы этого бокса, чтобы включить все точки из [page:Array points].

[method:Box2 translate]([param:Vector2 offset])

[page:Vector2 offset] - Направление и расстояние смещения.

Добавляет [page:Vector2 offset] к верхней и нижней границам этого бокса, эффективно перемещая этот бокс на [page:Vector2 offset] единиц в 2D пространстве.

[method:Box2 union]([param:Box2 box])

[page:Box2 box] - Бокс, который будет объединен с этим боксом.

Объединяет этот бокс с [page:Box2 box], устанавливая верхнюю границу этого бокса на большую из верхних границ двух боксов, а нижнюю границу этого бокса на меньшую из нижних границ двух боксов.
