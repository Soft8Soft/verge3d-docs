[name]

Класс, представляющий [link:http://en.wikipedia.org/wiki/Euler_angles Euler Angles].

Углы Эйлера описывают вращательное преобразование путем вращения объекта по различным осям на заданную величину по каждой оси и в заданном порядке осей.

Пример

const a = new v3d.Euler(0, 1, 1.57, 'XYZ'); const b = new v3d.Vector3(1, 0, 1); b.applyEuler(a);

Конструктор

[name]([param:Float x], [param:Float y], [param:Float z], [param:String order])

[page:Float x] — (необязательно) угол наклона оси x в радианах. По умолчанию *0*.
[page:Float y] — (необязательно) угол наклона оси y в радианах. По умолчанию *0*.
[page:Float z] — (необязательно) угол наклона оси z в радианах. По умолчанию *0*.
[page:String order] — (необязательно) строка, представляющая порядок применения вращений, по умолчанию 'XYZ' (должен быть в верхнем регистре).

Свойства

[property:String order]

Порядок применения вращений. По умолчанию - 'XYZ', что означает, что сначала объект будет повернут вокруг оси X, затем вокруг оси Y и, наконец, вокруг оси Z. Другими вариантами являются: 'YZX', 'ZXY', 'XZY', 'YXZ' и 'ZYX'. Они должны быть в верхнем регистре.

Verge3D использует intrinsic(внутренние углы) Тайт-Брайана. Это означает, что вращение выполняется относительно локальной системы координат. То есть, для порядка 'XYZ' вращение происходит сначала вокруг локальной оси X (которая совпадает с мировой осью X), затем вокруг локальной оси Y (которая теперь может отличаться от мировой оси Y), затем локальной оси Z (которая может отличаться от мировой оси Z).

[property:Float x]

Текущее значение компонента x.

[property:Float y]

Текущее значение компонента y.

[property:Float z]

Текущее значение компонента z.

Методы

[method:Euler copy]([param:Euler euler])

Копирует значение [page:Euler euler] в этот эйлер.

[method:Euler clone]()

Возвращает новый эйлер с теми же параметрами, что и данный.

[method:Boolean equals]([param:Euler euler])

Проверяет строгое равенство данного эйлера и [page:Euler euler].

[method:Boolean equalsEps]([param:Euler euler], [param:Number eps])

Проверяет равенство данного эйлера и заданного [page:Euler euler] с учетом заданного *eps* поля

[method:Euler fromArray]([param:Array array])

[page:Array array] длиной 3 или 4. Необязательный 4-й аргумент соответствует [page:.order order].

Присваивает данный эйлеров угол [page:.x x] массиву[0]. .
Присваивает данный эйлеров угол [page:.y y] массиву[1]. .
Присваивает данный эйлеров угол [page:.z z] массиву[2]. .
По желанию присваивает массиву [3] этот эйлеров угол [page:.order order].

[method:Euler makeAlternative]()

Меняет этот эйлер на альтернативный набор вращений в диапазоне [-π, π) соответствующий той же ориентации (например: от (0, 0, π) до (-π, -π, 0)).

[method:Euler reorder]([param:String newOrder])

Сброс эйлерова угла с новым порядком путем создания кватерниона из данного эйлерова угла, а затем установки данного эйлерова угла с кватернионом и новым порядком.

ПРЕДУПРЕЖДЕНИЕ: сбрасывает информацию о поворотах.

[method:Euler set]([param:Float x], [param:Float y], [param:Float z], [param:String order])

[page:.x x] — угол наклона оси x в радианах.
[page:.y y] — угол наклона оси y в радианах.
[page:.z z] — угол наклона оси z в радианах.
[page:.order order] — (необязательно) строка, представляющая порядок применения вращений.

Устанавливает углы этого эйлерова преобразования и, по желанию, [page:.order order].

[method:Euler setFromRotationMatrix]([param:Matrix4 m], [param:String order])

[page:Matrix4 m] — матрица [page:Matrix4] верхняя часть 3x3 которой является чистой матрицей вращения [link:https://en.wikipedia.org/wiki/Rotation_matrix rotation matrix] (т.е. без масштабирования).
[page:.order order] — (необязательно) строка, представляющая порядок применения вращений.
Устанавливает углы данного преобразования Эйлера от чистой матрицы вращения на основе ориентации, заданной порядком.

[method:Euler setFromQuaternion]([param:Quaternion q], [param:String order])

[page:Quaternion q] — нормализованный кватернион.
[page:.order order] — (необязательно) строка, представляющая порядок применения вращений.
Устанавливает углы данного преобразования Эйлера из нормализованного кватерниона на основе ориентации, указанной [page:.order order].

[method:Euler setFromVector3]([param:Vector3 vector], [param:String order])

[page:Vector3 vector] - [page:Vector3].
[page:.order order] — (необязательно) строка, представляющая порядок применения вращений.

Установите параметры [page:.x x], [page:.y y] и [page:.z z] и, по желанию, обновите [page:.order order].

[method:Array toArray]([param:Array array], [param:Integer offset])

[page:Array array] — (необязательно) массив для хранения эйлера.
[page:Integer offset] (необязательно) смещение в массиве.
Возвращает массив вида [[page:.x x], [page:.y y], [page:.z z], [page:.order order]].

[method:Vector3 toVector3]([param:Vector3 optionalResult])

[page:Vector3 optionalResult] — (необязательно) Если указано, результат будет скопирован в этот вектор, в противном случае будет создан новый.

Возвращает свойства Эйлера [page:.x x], [page:.y y] и [page:.z z] в виде [page:Vector3].
