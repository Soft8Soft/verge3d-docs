[name]

Отрезок геометрической линии, представленный начальной и конечной точкой.

Конструктор

[name]([param:Vector3 start], [param:Vector3 end])

[page:Vector3 start] - Начало отрезка линии. По умолчанию (0, 0, 0).
[page:Vector3 end] - Конец сегмента линии. По умолчанию (0, 0, 0).

Создает новый [name].

Свойства

[property:Vector3 start]

[page:Vector3] представляющий начальную точку линии.

[property:Vector3 end]

[page:Vector3] представляющий конечную точку линии.

Методы

[method:Line3 applyMatrix4]([param:Matrix4 matrix])

Применяет матричное преобразование к отрезку прямой.

[method:Vector3 at]([param:Float t], [param:Vector3 target])

[page:Float t] - Используйте значения 0-1, чтобы вернуть позицию вдоль сегмента линии.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает вектор в определенном положении вдоль линии. Когда [page:Float t] = 0, возвращается начальный вектор, а когда [page:Float t] = 1, возвращается конечный вектор.

[method:Line3 clone]()

Возвращает новый [page:Line3] с теми же векторами [page:.start start] и [page:.end end end], что и данный.

[method:Vector3 closestPointToPoint]([param:Vector3 point], [param:Boolean clampToLine], [param:Vector3 target])

[page:Vector3 point] - возвращает ближайшую точку на линии к данной точке.
[page:Boolean clampToLine] - привязывать ли возвращаемое значение к отрезку линии.
[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает точку замыкания на линии. Если [page:Boolean clampToLine] равно true, то возвращаемое значение будет привязано к отрезку линии.

[method:Float closestPointToPointParameter]([param:Vector3 point], [param:Boolean clampToLine])

[page:Vector3 point] - точка, для которой нужно вернуть параметр точки.
[page:Boolean clampToLine] - Ограничивать ли результат в диапазоне [0, 1]..

Возвращает параметр точки, основанный на ближайшей точке, спроецированной на сегмент линии. Если [page:Boolean clampToLine] равно true, то возвращаемое значение будет от 0 до 1.

[method:Line3 copy]([param:Line3 line])

Копирует векторы [page:.start start] и [page:.end end] переданной строки в эту строку.

[method:Vector3 delta]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает дельта-вектор сегмента линии (вектор [page:.end end] минус вектор [page:.start start]).

[method:Float distance]()

Возвращает Евклидово расстояние [link:https://en.wikipedia.org/wiki/Euclidean_distance Euclidean distance] (расстояние по прямой) между точками [page:.start start] и [page:.end end].

[method:Float distanceSq]()

Возвращает квадрат [link:https://en.wikipedia.org/wiki/Euclidean_distance Euclidean distance] (расстояние по прямой) между векторами [page:.start start] и [page:.end end].

[method:Boolean equals]([param:Line3 line])

[page:Line3 line] - [page:Line3] чтобы сравнить с этой.

Возвращает true, если обе точки линии [page:.start start] и [page:.end end end] равны.

[method:Vector3 getCenter]([param:Vector3 target])

[page:Vector3 target] — результат будет скопирован в этот Vector3.

Возвращает центр отрезка прямой.

[method:Line3 set]([param:Vector3 start], [param:Vector3 end])

[page:Vector3 start] - установить начальную точку [page:.start start point] строки.
[page:Vector3 end] - установить конечную точку [page:.end end point] строки.

Устанавливает начальное и конечное значения путем копирования предоставленных векторов.
