[name]

Простой класс для регистрации событий keyup/keydown и определения, нажата ли клавиша в данный момент.

Конструктор

[name]([page:HTMLDOMElement domElement])

[page:HTMLDOMElement domElement]: Элемент HTML для добавления слушателей событий.

Свойства

[property:Object isKeyDown]

Объект, ключами которого являются коды клавиш из словаря [page:Keys], а значениями - булевы флаги, указывающие, нажата ли в данный момент соответствующая клавиша.

[property:Function onKeyDown]

Обратный вызов, срабатывающий после события "keydown". Получает объект "event" в качестве параметра.

[property:Function onKeyUp]

Обратный вызов, срабатывающий после события "keyup". Получает объект "event" в качестве параметра.

Методы

[method:null dispose]()

Удалите все слушатели событий.
