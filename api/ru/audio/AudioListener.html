[name]

[name] представляет собой виртуальный [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioListener listener] из всех позиционных и непозиционных аудиоэффектов в сцене.
Приложение Verge3D обычно создает один экземпляр [name]. Это обязательный параметр-конструтор для сущностей аудио, таких как [page:Audio Audio] и [page:PositionalAudio PositionalAudio].
В большинстве случаев объект слушателя является дочерним объектом камеры, таким образом их положения в пространстве совпадают.

Пример

// create an AudioListener and add it to the camera const listener = new v3d.AudioListener(); camera.add(listener); // create a global audio source const sound = new v3d.Audio(listener); // load a sound and set it as the Audio object's buffer const audioLoader = new v3d.AudioLoader(); audioLoader.load('sounds/ambient.ogg', function(buffer) { sound.setBuffer(buffer); sound.setLoop(true); sound.setVolume(0.5); sound.play(); });

Конструктор

[name]()

Создает новый AudioListener.

Свойства

[property:AudioContext context]

[link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext] слушателя [page:AudioListener listener], заданного в конструкторе.

[property:GainNode gain]

Нода [link:https://developer.mozilla.org/en-US/docs/Web/API/GainNode GainNode] созданная с помощью [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext/createGain AudioContext.createGain]().

[property:AudioNode filter]

По умолчанию *null*.

[property:Number timeDelta]

Значение дельты времени для аудиоэлементов. Используется в контексте [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioParam/linearRampToValueAtTime AudioParam.linearRampToValueAtTimeDefault](). По умолчанию *0*.
