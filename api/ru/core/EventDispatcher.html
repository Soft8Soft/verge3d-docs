[name]

Обработка пользовательских событий «Ява Скрипта».

Пример

// добавление возможности отправки события class Car extends EventDispatcher { start() { this.dispatchEvent({ type: 'start', message: 'брум-брум!' }); } }; // обработка события const car = new Car(); car.addEventListener('start', function(event) { alert(event.message); }); car.start();

Конструктор

[name]()

Создает объект EventDispatcher.

Методы

[method:null addEventListener]([param:String type], [param:Function listener])

  • type — тип события, которое необходимо обработать.
  • listener — функция-обработчик, которая вызывается при наступлении события.

Добавляет обработчик события.

[method:Boolean hasEventListener]([param:String type], [param:Function listener])

  • type — тип события, которое необходимо обработать.
  • listener — функция-обработчик, которая вызывается при наступлении события.

Проверяет, имеется ли обработчик события данного типа.

[method:null removeEventListener]([param:String type], [param:Function listener])

  • type — тип событий, обрабатываемых функцией-обработчиком.
  • listener — удаляемая функция-обработчик.

Удаляет обработчик данного события.

[method:null dispatchEvent]([param:Object event])

event — отправляемое событие.

Отправка события.

Пазлы

«Вердж3Д» поддерживает ряд пазлов, реализующих похожую функциональность без необходимости писать код:

  • add event listener
  • remove event listener
  • dispatch event
