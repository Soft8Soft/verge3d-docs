Свойства

[property:Number meshPerAttribute]

Определяет, как часто должно повторяться значение этого буферного атрибута. Значение один означает, что каждое значение экземпляра атрибута используется для одного экземпляра. Значение два означает, что каждое значение используется для двух последовательных экземпляров (и так далее). По умолчанию *1*.
