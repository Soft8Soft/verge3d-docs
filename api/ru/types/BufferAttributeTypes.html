Типы BufferAttribute

В «Вердж3Д» доступно девять типов [page:BufferAttribute]. Они соответствуют [page:TypedArray типизированным массивам] «Ява Скрипта».

v3d.Float64BufferAttribute v3d.Float32BufferAttribute v3d.Float16BufferAttribute v3d.Uint32BufferAttribute v3d.Int32BufferAttribute v3d.Uint16BufferAttribute v3d.Int16BufferAttribute v3d.Uint8ClampedBufferAttribute v3d.Uint8BufferAttribute v3d.Int8BufferAttribute

Конструктор

Все вышеперечисленное вызывается одинаково.

TypedBufferAttribute([param:Array_or_Integer array], [param:Integer itemSize], [param:Boolean normalized])

array — это может быть типизированный или нетипизированный (обычный) массив или целочисленная длина. Значение массива будет преобразовано к указанному типу. Если указана длина, будет создан новый типизированный массив, инициализированный со всеми элементами, установленными в ноль.

itemSize — количество значений массива, которые должны быть связаны с конкретной вершиной.

normalized — (необязательно) указывает, как базовые данные в буфере отображаются на значения в коде GLSL.
