[name]

Объект-помощник для графического отображения ограничивающей рамки вокруг объекта с выравниванием по мировой оси. Фактическая ограничительная рамка обрабатывается в [page:Box3], это просто визуальный помощник для отладки. Его размер может быть автоматически изменен с помощью метода [page:BoxHelper.update], когда объект, на основе которого он создан, трансформируется. Обратите внимание, что для этого объект должен иметь [page:BufferGeometry], поэтому он не будет работать с [page:Sprite Sprites].

Пример

const sphere = new v3d.SphereGeometry(); const object = new v3d.Mesh(sphere, new v3d.MeshBasicMaterial(0xff0000)); const box = new v3d.BoxHelper(object, 0xffff00); app.scene.add(box);

Конструктор

[name]([param:Object3D object], [param:Color color])

[page:Object3D object] — (необ.) 3D-объект для отображения ограничивающей рамки, выровненной по мировой оси.
[page:Color color] — (необ.) шестнадцатеричное значение, определяющее цвет бокса. По умолчанию 0xffff00.

Создает новый вайрфреймовый бокс, который ограничивает заданный объект. Внутри используется [page:Box3.setFromObject] для вычисления размеров. Обратите внимание, что сюда входят все дочерние объекты.

Свойства

Общие свойства см. в базовом классе [page:LineSegments].

Методы

Общие методы см. в базовом классе [page:LineSegments].

[method:null update]()

Обновляет геометрию помощника в соответствии с размерами объекта, включая все дочерние объекты. См. [page:Box3.setFromObject].

[method:BoxHelper setFromObject]([param:Object3D object])

[page:Object3D object] — для создания помощника [page:Object3D].

Обновляет вайрфреймовый бокс для заданного объекта.
