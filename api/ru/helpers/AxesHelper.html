[name]

Объект оси для визуализации 3 осей простым способом.
Ось X имеет красный цвет. Ось Y - зеленый. Ось Z - синий.

Пример

const axesHelper = new v3d.AxesHelper(5); scene.add(axesHelper);

Конструктор

[name]([param:Number size])

[page:Number size] — (необ.) размер линий, представляющих оси. По умолчанию *1*.
