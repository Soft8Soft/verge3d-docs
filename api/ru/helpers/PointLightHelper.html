[page:Object3D] → [page:Mesh] →

[name]

Отображается вспомогательный объект, состоящий из сферического [page:Mesh] для визуализации [page:PointLight].

Пример

const pointLight = new v3d.PointLight(0xff0000, 1, 100); pointLight.position.set(10, 10, 10); app.scene.add(pointLight); const sphereSize = 1; const pointLightHelper = new v3d.PointLightHelper(pointLight, sphereSize); app.scene.add(pointLightHelper);

Конструктор

[name]([param:PointLight light], [param:Float sphereSize], [param:Hex color])

[page:PointLight light] — Свет, который необходимо визуализировать.

[page:Float sphereSize] — (необ.) Размер сферы помощника. По умолчанию *1*.

[page:Hex color] — (необ.) если это не задано, помощник примет цвет освещения.

Свойства

Общие свойства см. в базовом классе [page:Mesh].

[property:PointLight light]

[page:PointLight], который визуализируется.

[property:Object matrix]

Референс на pointLight's [page:Object3D.matrixWorld matrixWorld].

[property:Object matrixAutoUpdate]

См. [page:Object3D.matrixAutoUpdate]. Здесь установлено значение *false*, так как помощник использует pointLight's [page:Object3D.matrixWorld matrixWorld].

[property:hex color]

Параметр цвета, передаваемый в конструкторе. По умолчанию *undefined*. Если этот параметр изменить, цвет помощника будет обновлен при следующем вызове [page:.update update].

Методы

Общие методы см. в базовом классе [page:Mesh].

Утилизация помощника pointLightHelper.
