[name]

GridHelper - это объект для определения сеток. Сетки - это двумерные массивы линий.

Пример

const size = 10; const divisions = 10; const gridHelper = new v3d.GridHelper(size, divisions); app.scene.add(gridHelper);

Конструктор

[name]([param:number size], [param:Number divisions], [param:Color colorCenterLine], [param:Color colorGrid])

size — Размер сетки. По умолчанию - 10.
divisions — Количество делений в сетке. По умолчанию - 10.
colorCenterLine — Цвет центральной линии. Это может быть [page:Color], шестнадцатеричное значение и имя CSS-Color. По умолчанию 0x444444
colorGrid — Цвет линий сетки. Это может быть [page:Color], шестнадцатеричное значение и имя CSS-Color. По умолчанию 0x888888

Создает новый [name] с размером 'size' и разделенным на сегменты 'divisions' на каждой стороне. Цвета являются опциональными.
