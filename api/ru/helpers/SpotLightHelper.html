[page:Object3D] →

[name]

Здесь отображается вспомогательный объект в форме конуса для [page:SpotLight].

Пример

const spotLight = new v3d.SpotLight(0xffffff); spotLight.position.set(10, 10, 10); app.scene.add(spotLight); const spotLightHelper = new v3d.SpotLightHelper(spotLight); app.scene.add(spotLightHelper);

Конструктор

[name]([param:SpotLight light], [param:Hex color])

[page:SpotLight light] — [page:SpotLight] для визуализации.

[page:Hex color] — (необ.) если это не задано, помощник примет цвет света.

Свойства

Общие свойства см. в базовом классе [page:Object3D].

[property:LineSegments cone]

[page:LineSegments] используется для визуализации света.

[property:SpotLight light]

Референс на визуализируемый [page:SpotLight]..

[property:Object matrix]

Референс на spotLight's [page:Object3D.matrixWorld matrixWorld].

[property:Object matrixAutoUpdate]

См. [page:Object3D.matrixAutoUpdate]. Здесь установлено значение *false*, так как помощник использует spotLight's [page:Object3D.matrixWorld matrixWorld].

[property:hex color]

Параметр цвета, передаваемый в конструкторе. По умолчанию *undefined*. Если этот параметр изменить, цвет помощника будет обновлен при следующем вызове [page:.update update].

Методы

Общие методы см. в базовом классе [page:Object3D].

[method:null dispose]()

Утилизирует помощника света.

[method:null update]()

Обновляет помощника света.
