[name]

Объект-помощник для визуализации влияния [page:DirectionalLight] на сцену. Он состоит из плоскости и линии, представляющей положение и направление света.

Пример

const light = new v3d.DirectionalLight(0xFFFFFF); const helper = new v3d.DirectionalLightHelper(light, 5); app.scene.add(helper);

Конструктор

[name]([param:DirectionalLight light], [param:Number size], [param:Hex color])

[page:DirectionalLight light]-- Свет, который необходимо визуализировать.

[page:Number size] — (необ.) размеры плоскости. По умолчанию *1*.

[page:Hex color] — (необ.) если этот параметр не установлен, помощник примет цвет освещения.

Свойства

Общие свойства смотритеф в базовом классе [page:Object3D].

[property:Line lightPlane]

Содержит меш линии, показывающей расположение направленного света.

[property:DirectionalLight light]

Референс на визуализируемый [page:DirectionalLight directionalLight].

[property:Object matrix]

Референс на [page:Object3D.matrixWorld matrixWorld] освещения.

[property:Object matrixAutoUpdate]

См. [page:Object3D.matrixAutoUpdate]. Здесь установлено значение *false*, так как помощник использует [page:Object3D.matrixWorld matrixWorld] света.

[property:hex color]

Параметр цвета, передаваемый в конструкторе. По умолчанию *undefined*. Если этот параметр изменить, цвет помощника обновится при следующем вызове [page:.update update].

Методы

Общие свойства см. в базовом классе [page:Object3D].

[method:null dispose]()

Утилизация помощника.

[method:null update]()

Обновляет помощник в соответствии с положением и направлением визуализируемого [page:.light directionalLight].
