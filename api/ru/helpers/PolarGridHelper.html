[name]

PolarGridHelper - это объект для определения полярных сетей. Сети - это двумерные массивы линий.

Пример

const radius = 10; const radials = 16; const circles = 8; const divisions = 64; const helper = new v3d.PolarGridHelper(radius, radials, circles, divisions); app.scene.add(helper);

Конструктор

[name]([param:Number radius], [param:Number radials], [param:Number circles], [param:Number divisions], [param:Color color1], [param:Color color2])

radius — Радиус полярной сетки. Это может быть любое положительное число. По умолчанию - 10.
radials — Количество радиальных линий. Это может быть любое целое положительное число. По умолчанию - 16.
circles — Количество кругов. Это может быть любое целое положительное число. По умолчанию - 8.
divisions — Количество сегментов линии, используемых для каждой окружности. Это может быть любое целое положительное число, равное 3 или больше. По умолчанию - 64.
color1 — Первый цвет, используемый для элементов сетки. Это может быть [page:Color], шестнадцатеричное значение или имя CSS-Color. По умолчанию 0x444444
color2 — Второй цвет, используемый для элементов сетки. Это может быть [page:Color], шестнадцатеричное значение и имя CSS-Color. По умолчанию 0x888888

Создает новый [name] радиуса 'radius' с числом радиусов 'radials' и числом окружностей 'circles', где каждая окружность сглаживается в число сегментов линии 'divisions'. Цвета являются опциональными.
