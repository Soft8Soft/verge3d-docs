[name]

3Д-объект стрелки для визуализации направлений.

Пример

const dir = new v3d.Vector3(1, 2, 0); //normalize the direction vector (convert to vector of length 1) dir.normalize(); const origin = new v3d.Vector3(0, 0, 0); const length = 1; const hex = 0xffff00; const arrowHelper = new v3d.ArrowHelper(dir, origin, length, hex); app.scene.add(arrowHelper);

Конструктор

[name]([param:Vector3 dir], [param:Vector3 origin], [param:Number length], [param:Number hex], [param:Number headLength], [param:Number headWidth])

[page:Vector3 dir] — направление от начала координат. Должен быть единичным вектором.
[page:Vector3 origin] — точка, в которой начинается стрелка.
[page:Number length] — длина стрелки. По умолчанию *1*.
[page:Number hex] — шестнадцатеричное значение для определения цвета. По умолчанию 0xffff00.
[page:Number headLength] — длина наконечника стрелки. По умолчанию 0,2 * длина.
[page:Number headWidth] — ширина наконечника стрелки. По умолчанию 0,2 * headLength.

Свойства

Общие свойства см. в базовом классе [page:Object3D].

[property:Line line]

Содержит линейную часть стрелки.

[property:Mesh cone]

Содержит конусную часть стрелки.

Методы

Общие методы см. в базовом классе [page:Object3D].

[method:null setColor]([param:Color color])

color — Желаемый цвет.

Устанавливает цвет стрелки.

[method:null setLength]([param:Number length], [param:Number headLength], [param:Number headWidth])

length — Желаемая длина.
headLength — Длина наконечника стрелки.
headWidth — Ширина наконечника стрелки.

Устанавливает длину стрелки.

[method:null setDirection]([param:Vector3 dir])

dir — Желаемое направление. Должно быть единичным вектором.

Устанавливает направление стрелки.
