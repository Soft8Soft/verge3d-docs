[name]

Implements simple yet highly customizable preloader. Check out the Verge3D's standard app templates to see it in action.

Constructor

[name]([param:Object parameters])

[page:Object parameters] — object with properties defining the preloader's behavior.

The following is the only valid parameter.

[page:String container] — An element ID or an element used to store the created preloader.

Properties

[property:HTMLElement bar]

A <div> element used to display the preloader's bar.

[property:HTMLElement logo]

A <div> element used to display the preloader's logo.

[property:HTMLElement logoCont]

A <div> element used for the preloader's logo container.

[property:HTMLElement background]

A <div> element used for the preloader's background.

[property:HTMLElement container]

A container element passed to the constructor.

[property:Clock clock]

A [page:Clock clock] instance. Used internally.

Methods

Simple preloader overrides [page:Preloader Preloader] methods implementing image animation and cleanup logic.
