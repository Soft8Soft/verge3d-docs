[name]

A special mesh containing geometry for drawing lines. Intended to use with the [page:MeshLineMaterial] material. Capable of creating geometry from [page:BufferGeometry], for Array/Float32Array data use [page:MeshLine].

Constructor

[name]()

The constructor takes no parameters.

Properties

[property:BufferGeometry geometry]

The line's geometry to pass into the [page:Mesh] constructor. Created after calling the [page:.fromBufferGeometry fromBufferGeometry] method.

Methods

[method:undefined fromBufferGeometry]([param:BufferGeometry geometry])

Build the line's geometry from the given [page:BufferGeometry] data.
