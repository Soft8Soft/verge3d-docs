[name]

A special mesh containing geometry for drawing lines. Intended to use with the [page:MeshLineMaterial] material. Capable of creating geometry from Array/Float32Array data, for [page:BufferGeometry] use [page:MeshLineIndexed] instead.

Constructor

[name]()

The constructor takes no parameters.

Properties

[property:BufferGeometry geometry]

The line's geometry to pass into the [page:Mesh] constructor. Created after calling the [page:.setGeometry setGeometry] method.

Methods

[method:undefined setGeometry]([param:Array|Float32Array geometry], [param:Function widthCallback])

Build the line's geometry from the given [page:BufferGeometry] or Array/Float32Array data. `widthCallback` can be used to control the width at each point along the line. Its parameter `p` varies from `0` to `1` along the line:

line.setGeometry(geometry, function(p) { return p; }); [sourceHint]
