[name]

A class containing useful utility functions for your 3D apps.

Methods

[method:undefined addToAppList]([param:App app])

Add `app` to the global `v3d.apps` array. Normally, this method is executed by the [page:App App constructor].

[method:Object getPageParams]([param:Window windowObj])

Parse page GET parameters and return object with parsed key-value data.

[method:Boolean isJS]([param:String url])

Check if the `url` parameter represents a JavaScript file.

[method:Boolean isXML]([param:String url])

Check if the `url` parameter represents an XML file.

[method:undefined loadScript]([param:String url], [param:HTMLElement appendElem], [param:Function callback])

url — Loaded script URL.
appendElem — The HTML element to append loaded script to.
callback — load event listener.

Load and execute the given JavaScript file.

[method:undefined printPerformanceInfo]([param:App scope], [param:Float delta])

Collect and print performance info profile. See [page:App.printPerformanceInfo] for more info.

[method:undefined registerServiceKeys]([param:App app])

Register the service keys (like the triple tilda for the service menu) for the running application.

[method:undefined removeFromAppList]([param:App app])

Remove `app` from the global `v3d.apps` array. Normally, this method is executed by [page:App.dispose].

[method:undefined unregisterServiceKeys]([param:App app])

Unregister the service keys for the running application.

[method:String updatePageParam]([param:String url], [param:String param], [param:AnyType paramVal])

Update GET parameter for the passed `url` and return an updated URL.
