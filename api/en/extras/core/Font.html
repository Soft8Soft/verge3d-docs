[name]

Create a set of [page:Shape Shapes] representing a font loaded in JSON format.

Constructor

[name]([param:Object data])

data — JSON typeface data representing the font.

This constructor creates a new [name], which is an array of [page:Shape Shapes].

Properties

[property:Object data]

The JSON typeface data passed in the constructor.

Methods

[method:undefined generateShapes]([param:String text], [param:Float size], [param:String alignX], [param:String alignY], [param:Float lineHeight], [param:Boolean scaledEmSize])

[page:String text] — string of text.
[page:Float size] — (optional) scale for the [page:Shape Shapes]. Default is `100`.
[page:Float size] — (optional) scale for the [page:Shape Shapes]. Default is `100`.
[page:String alignX] — (optional) horizontal text alignment. Default is `"left"`.
[page:String alignY] — (optional) vertical text alignment. Default is `"topBaseline"`.
[page:Float lineHeight] — (optional) line height. Default is `-1` for auto.
[page:Boolean scaledEmSize] — (optional) reduce font height to match font bounding box, not ascender-descender value taken from the font. Default is `false`.

Creates an array of [page:Shape Shapes] representing the text in the font.
