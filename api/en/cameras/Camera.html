[name]

Abstract base class for cameras. This class should always be inherited when you build a new camera.

Constructor

[name]()

Creates a new [name]. Note that this class is not intended to be called directly; you probably want a [page:PerspectiveCamera] or [page:OrthographicCamera] instead.

Properties

See the base [page:Object3D] class for common properties.

[property:ControlSettings controlSettings]

Camera control settings used to assign application [page:App.controls controls].

[property:Boolean ignoreShadowLayers]

Ignore camera layers when rendering objects into the shadow map. Used internally by SSR pass. Default is `false`.

[property:Boolean isCamera]

Read-only flag to check if a given object is of type [name].

[property:Layers layers]

The [page:Layers layers] that the camera is a member of. This is an inherited property from [page:Object3D].

Objects must share at least one layer with the camera to be seen by the camera.

[property:Matrix4 matrixWorldInverse]

This is the inverse of `matrixWorld`. `matrixWorld` contains the matrix which has the world transform of the camera.

[property:Matrix4 projectionMatrix]

This is the matrix which contains the projection.

[property:Matrix4 projectionMatrixInverse]

The inverse of `projectionMatrix`.

[property:Object viewportFit]

An object with viewport fit settings:

[page:Cameras type]
Viewport fit type, see [page:Cameras here] for more info.
[page:Float initialAspect]
Initial viewport aspect ratio.
