[name]

An object which represents clipping plane.

Clipping in Verge3D is quite complex as you require several objects to render it correctly:

  • Instance of [name].
  • One or many [page:Mesh] objects you need to clip.
  • Two stencil (front/back) meshes per each clipped mesh. These meshes share the geometry of the original mesh but have the special materials based on [page:MeshBasicMaterial].
  • Optional object with planar geometry which represents filled cross-section. Contrary to stencil meshes, this object is parented to [name].

Stencil and cross-section objects are marked by the special `.isAuxClippingMesh = true` flag.

Set to 'ClippingPlaneObject', this can be used to find all clipping planes in a scene.
