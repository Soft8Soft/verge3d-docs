[name]

An 3D arrow object for visualizing directions.

Code Example

const dir = new v3d.Vector3(1, 2, 0); // normalize the direction vector (convert to vector of length 1) dir.normalize(); const origin = new v3d.Vector3(0, 0, 0); const length = 1; const hex = 0xffff00; const arrowHelper = new v3d.ArrowHelper(dir, origin, length, hex); scene.add(arrowHelper);

Constructor

[name]([param:Vector3 dir], [param:Vector3 origin], [param:Float length], [param:Color|Integer|String color], [param:Float headLength], [param:Float headWidth])

[page:Vector3 dir] — direction from origin. Must be a unit vector.
[page:Vector3 origin] — Point at which the arrow starts.
[page:Float length] — length of the arrow. Default is `1`.
[page:Integer color] — hexadecimal value to define color. Default is `0xffff00` (yellow).
[page:Float headLength] — The length of the head of the arrow. Default is `0.2 * length`.
[page:Float headWidth] — The width of the head of the arrow. Default is `0.2 * headLength`.
