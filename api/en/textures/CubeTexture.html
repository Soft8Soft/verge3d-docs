[name]

Creates a cube texture made up of six images.

Code Example

const loader = new v3d.CubeTextureLoader(); loader.setPath('textures/cube/pisa/'); const textureCube = loader.load([ 'px.png', 'nx.png', 'py.png', 'ny.png', 'pz.png', 'nz.png' ]); const material = new v3d.MeshBasicMaterial({ color: 0xffffff, envMap: textureCube });

Constructor

[name]([param:Array images], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Constant format], [param:Constant type], [param:Integer anisotropy], [param:Constant encoding])

CubeTexture is almost equivalent in functionality and usage to [page:Texture]. The only differences are that the images are an array of 6 images as opposed to a single image, and the mapping options are [page:Textures v3d.CubeReflectionMapping] (default) or [page:Textures v3d.CubeRefractionMapping]

Properties

See the base [page:Texture Texture] class for common properties.

[property:Boolean flipY]

If set to `true`, the texture is flipped along the vertical axis when uploaded to the GPU. Default is `false`.

[property:Boolean isCubeTexture]

Read-only flag to check if a given object is of type [name].
