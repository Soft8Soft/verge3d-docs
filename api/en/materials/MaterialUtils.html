[name]

A module containing useful utility methods to work with Verge3D materials.

Methods

[method:undefined replaceTexture]([param:Material mat], [param:Texture oldTex], [param:Texture newTex])

  • [page:Material mat] — material instance.
  • [page:Texture oldTex] — texture to be replaced.
  • [page:Texture newTex] — a new texture.

Replace one texture by another in the given material. Currently this method supports only [page:MeshNodeMaterial] and [page:MeshStandardMaterial] materials.

[method:undefined disposeTextures]([param:Material mat])

  • [page:Material mat] — material instance.

Dispose all [page:Texture textures] found in the given material. Currently this method supports only [page:MeshNodeMaterial] and [page:MeshStandardMaterial] materials.

[method:DiGraph createDiffuseMaterial]([param:String name], [param:Color color], [param:Float opacity], [param:String profile])

  • [page:String name] — material name.
  • [page:Color color] — material diffuse color.
  • [page:Float opacity] — material opacity.
  • [page:String profile] — material profile, specifies which nodes will be used to compose the node graph. One of `"blender"`, `"max"`, `"maya"`.

Generate node-based diffuse (lambert) material with the given `color` and `opacity` params.

[method:MeshNodeMaterial createEnvironmentMaterial]([param:String name], [param:Texture envTex], [param:Texture|Color bkgTexOrColor])

  • [page:String name] — material name.
  • [page:Texture envTex] — texture used for environment lighting.
  • bkgTexOrColor — [page:Texture texture] or [page:Color color] used for background.

Create node-based material which can be used for environment lighting and scene background.

[method:DiGraph createStandardPBRMaterial]([param:String profile], [param:Object params])

  • [page:String profile] — material profile, specifies which nodes will be used to compose the node graph. One of `"blender"`, `"max"`, `"maya"`.
  • [page:Object params] — (optional) additional material properties.

Generate node-based standard PBR material with the given set of parameters:

[page:Color color]
Material base color. Default is `'white'`.
[page:Float opacity]
Material opacity value. Default is `1.0` (opaque).
[page:Float metalness]
Material metalness value. Default is `0` (non-metallic).
[page:Float roughness]
Material roughness value. Default is `1`.
[page:Color emissive]
Material emissive color. Default is `'black'`.
[page:Float emissiveIntensity]
Material emissive color intensity. Default is `1`.
