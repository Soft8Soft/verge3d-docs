[name]

Class for storing and applying camera control settings.

Code Example

const aspect = app.container.offsetWidth / app.container.offsetHeight; const camera = new v3d.PerspectiveCamera(45, aspect, 1, 1000); camera.position.z = -5; camera.controlSettings = new v3d.ControlSettings('ORBIT'); app.scene.add(camera); // sets active camera and assigns app controls app.setCamera(camera);

Constructor

[name]([param:String type], [param:Boolean enablePan], [param:Float rotateSpeed], [param:Float moveSpeed])

  • [page:String type] — controls type: `'ORBIT'`, `'FIRST_PERSON'`, or `'FLYING'`. Optional, default is `'ORBIT'`.
  • [page:Boolean enablePan] — create camera controls with panning enabled. Optional, default is `'true'`.
  • [page:Float rotateSpeed] — camera rotation speed. Optional, default is `1`.
  • [page:Float moveSpeed] — camera movement speed. Optional, default is `1`.

Create a new object with camera control settings.

Properties

[property:Boolean enableKeys]

Enable or disable the use of keyboard controls. Default is `true`.

[property:Boolean enablePan]

Enable or disable camera panning. Default is `true`.

[property:Boolean enableRotate]

Enable or disable horizontal and vertical rotation of the camera. Default is `true`.

[property:Boolean enableZoom]

Enable or disable zooming (dollying) of the camera.

[property:Float moveSpeed]

Movements speed factor. Upon [page:.assignToControls assigning] settings to actual controls object, this value affects both panning and zooming speeds. Default is `1`.

[property:Boolean orbitEnableTurnover]

Enable or disable moving the camera over the head. Default is `false`.

[property:Float orbitMaxAzimuthAngle]

How far you can orbit horizontally, upper limit. Range is `-Math.PI` to `Math.PI` (or `Infinity` for no limit) and default is `Infinity`;

[property:Float orbitMaxDistance]

How far you can dolly out ([page:PerspectiveCamera] only). Default is `Infinity`.

[property:Float orbitMaxPolarAngle]

How far you can orbit vertically, upper limit. Range is `0` to `Math.PI` radians, and default is `Math.PI`.

[property:Float orbitMaxZoom]

How far you can zoom out ([page:OrthographicCamera] only). Default is Infinity.

[property:Float orbitMinAzimuthAngle]

How far you can orbit horizontally, lower limit. Range is `-Math.PI` to `Math.PI` (or `-Infinity` for no limit) and default is `-Infinity`;

[property:Float orbitMinDistance]

How far you can dolly in ([page:PerspectiveCamera] only). Default is `0`.

[property:Float orbitMinPolarAngle]

How far you can orbit vertically, lower limit. Range is `0` to `Math.PI` radians, and default is `0`.

[property:Float orbitMinZoom]

How far you can zoom in ([page:OrthographicCamera] only). Default is `0`.

[property:Float rotateSpeed]

Rotations speed factor. Default is `1`.

[property:String type]

Controls type: `'ORBIT'`, `'FIRST_PERSON'`, or `'FLYING'`.
