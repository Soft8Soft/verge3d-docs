[name]

This class is mainly intended for use internally by [page:DirectionalLightShadowCSM DirectionalLightShadowCSM] for calculating subfrustums.

Constructor

[name]()

Creates a new [name].

Properties

[property:Object vertices]

Contains 8 points forming the frustum.

[page:Array near] — array of [page:Vector3 points] forming the near plane.
[page:Array far] — array of [page:Vector3 points] forming the far plane.

Methods

[method:this setFromProjectionMatrix]([param:Matrix4 matrix], [param:Float maxFar])

[page:Matrix4 matrix] — view [page:Camera camera]'s projection matrix.
[page:Float maxFar] — frustum far plane distance.

Sets frustum points from projection matrix.

[method:this split]([param:Array breaks], [param:Array target], [param:Boolean fitToScene])

[page:Array breaks] — array of split intervals.
[page:Array target] — array of [page:CSMFrustum subfrustums], should be `target.length <= breaks.length`.
[page:Boolean fitToScene] — if `true`, uses `fit to scene`, else `fit to cascade` method.

Splits this frustum into subfrustums.

[method:this toSpace]([param:Matrix4 matrix], [param:this target])

[page:Matrix4 matrix] — matrix applied to each frustum point.
[page:this target] — [page:CSMFrustum frustum] to which the results will be written.

Multiplies each frustum point by a matrix and write the results to the target.
