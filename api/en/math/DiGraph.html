[name]

A directed acyclic graph (DAG). In Verge3D this structure is used to define the hierarchy of node-based [page:MeshNodeMaterial materials].

Constructor

[name]([param:Integer V])

Create a new graph of the size `V`.

Properties

[property:Integer V]

Graph size.

[property:Array adjList]

Graph adjacency list.

[property:Array nodeInfo]

Data attached to graph nodes.

[property:Object edges]

Graph edges.

Methods

[method:undefined addNode]([param:Object nodeInfo])

Add a new node to the graph and attach [page:.nodeInfo] data to it.

[method:undefined removeAll]()

Remove all nodes and edges from the graph.

[method:undefined addEdge]([param:Edge edge])

Add an edge to the graph.

[method:Edge addNewEdge]([param:Integer v], [param:Integer w])

Add a new edge to the graph which connects nodes `v` and `w`.

[method:undefined removeEdge]([param:Integer v], [param:Integer w])

Remove an edge from the graph which connects nodes `v` and `w`.

[method:Edge edge]([param:Integer v], [param:Integer w])

Return an edge which connects nodes `v` and `w`. Return `null` if no such edge is found.

[method:Integer adj]([param:Integer v])

Return node adjacent to node `v`.

[method:Object node]([param:Integer v])

Return node info data for node `v`.

[method:Integer nodeINdex]([param:Object nodeInfo])

Return the index of the node which has the passed [page:.nodeInfo] data.

[method:DiGraph reverse]()

Return a new graph which has all edges reversed. The source graph remains intact.

[method:DiGraph len]()

Return graph length.

[method:Array findInEdges]([param:Integer v])

Return input edges for node `v`.

[method:Array findOutEdges]([param:Integer v])

Return output edges for node `v`.

[method:undefined disconnect]([param:Integer v], [param:Boolean removeNode])

Disconnect node `v` from the graph. If `removeNode=false` (default) leave the disconnected node inside the graph. If `removeNode=true` remove the node (with its data) as well.

[method:undefined insert]([param:DiGraph G], [param:Array edgesInG], [param:Array edgesOutG], [param:Function copyNodeCb], [param:Function copyEdgeCb])

Insert a `G` subgraph into the graph. The `edgesInG` and `edgesOutG` arrays represent input and output edges which will be used to connect the subgraph. Optional callbacks `copyNodeCb` and `copyEdgeCb` are executed per node/edge inserted to the graph.

[method:undefined vis]([param:Function labelCallback])

Convert the graph to the DOT (Graphviz) format and print to the browser console. Optional `labelCallback` is used to assign labels for the printed graph nodes. It accepts node index and node info as parameters.
