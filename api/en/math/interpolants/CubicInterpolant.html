[name]

Performs cubic interpoloation for the input arrays.

Code Example

const interpolant = new v3d.[name]( new Float32Array(2), new Float32Array(2), 1, new Float32Array(1) ); interpolant.evaluate(0.5);

Constructor

[name](parameterPositions, sampleValues, sampleSize, resultBuffer)

parameterPositions — array of positions
sampleValues — array of samples
sampleSize — number of samples
resultBuffer — buffer to store the interpolation results.
