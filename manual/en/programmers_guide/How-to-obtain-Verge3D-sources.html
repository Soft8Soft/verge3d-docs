How to Obtain Verge3D Sources

The complete Verge3D source code is provided as part of Verge3D Developer Kit package. This package is available free of charge to all Verge3D Enterprise customers. See this page for instructions on how to obtain and install it.

If you experience difficulties with using any of the API methods, feel free to consult with the Verge3D developers and community on the Soft8Soft [link:https://www.soft8soft.com/forums/ forums].
