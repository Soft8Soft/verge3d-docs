Диспетчер приложений Диспетчер приложений — удобный инструмент позволяющий:
  • Создание новых приложений на основе шаблонов.
  • Запуск приложений и программ для 3Д-моделирования («Блендер», «3дс Макс», «Майя»).
  • Настройку внешнего и метаданных приложений.
  • Создание устанавливаемых прогрессивных веб-приложений (PWA).
  • Редактирование и запуск блоков визуальной логики (пазлов).
  • Обновление приложений до новых версий «Вердж3Д».
  • Публикация приложений в «Вердж3Д Нетворк».
  • Экспорт и импорт приложений с помощью «Зип»-архивов.
  • Управление лицензионными ключами.
  • Доступ к документации и каналам поддержки «Вердж3Д».
Диспетчер приложений доступен прямо из «Блендера», «3дс Макса» или «Майи». Кроме того, его можно запустить с помощью иконки на рабочем столе (в ОС «Виндоус»), либо с помощью командной строки (в любой системе).

Программирование

Функциональность приложения может быть расширена с помощью «Ява Скрипта», путем изменения исходников или интеграции кода, созданного в редакторе пазлов.

«Вердж3Д» также предоставляет программный интерфейс (API), частично совместимый с фреймворком «Три Джей Эс», что упрощает портирование имеющегося кода.
Публикация приложения Вы можете распространять свой контент, используя:
  • Ваш собственный сервер.
  • Любой веб-хостинг (Yandex Cloud, Reg.ru, «Спринтхост», Beget, Timeweb и т.д.).
  • «Вердж3Д Нетворк».
  • С помощью SCORM-совместимой системы электронного обучения («Модл», «Чамило», «SCORM клауд», «Канвас», «Ай спринг»).
  • В виде мобильного приложения через магазины «Ру стор», «Гугл плей» и «Апп стор».
  • В виде десктопного приложения с помощью фреймворка «Электрон».
Пользовательский интерфейс и HTML Пользовательский интерфейс 3Д-приложений может быть создан с помощью:
  • Трёхмерных моделей.
  • HTML-пазлов.
  • ВИЗИВИГ редакторов, например «Вебфлоу».
  • Самописного HTML/CSS-кода.
Серверные функции и сеть «Вердж3Д» содержит плагин для «Вордпресса», самой популярной системы управления контентом. Плагин включает в себя следующие функции:
  • Размещение и публикация приложений (встраивание).
  • Постоянное хранилище файлов.
  • Интеграция с «Ву Коммерц».
  • Полностью настраиваемая система электронной коммерции для обработки заказов клиентов.
В дополнение к этому, редактор визуальной логики поддерживает блоки для отправки и получения данных по сети в формате JSON, CSV или обычного текста.
Постпроцессинговые эффекты Следующие эффекты можно включить в пакете моделирования:
  • Эмбиент окклюжн (GTAO) — смотрите настройки в «Блендере» / «3дс Максе» / «Майе»
С использованием пазлов можно включить:
  • Размытие
  • Блум
  • Яркость/контраст
  • Глубина резкости
  • Ч/б
  • Контур
  • SSR (рефракция и отражение)
  • Суперсемплинговый анти-алиасинг (SSAA)
Электронная коммерция

«Вердж3Д» включает в себя следующие возможности по созданию 3Д-приложений для электронной коммерции:

  • Платежи «Пейпал» с использованием пазлов.
  • Интеграция с системой «Ву Коммерц».

Кроме того, «Вердж3Д» включает в себя полноценную систему электронной коммерции на базе плагина для системы управления контентом «Вордпресс», позволяющую

  • Принимать заказы с помощью форм или низкоуровневого REST API.
  • Возможность создавать конфигурируемые продукты с неограниченным числом вариаций.
  • Уведомления по электронной почте о новых, обновлённых и исполненных заказах.
  • Система расчёта цен.
  • Генерирование счетов на оплату и инвойсов.
  • Продажа скачиваемых продуктов.
  • Оплата с помощью «Пейпал».
  • Отправка форм с произвольным набором данных.

Кроме того, в вашем распоряжении более 300 блоков визуального программирования, с помощью которых можно создавать интерактивные 3Д-магазины, конфигураторы, презентации, промо-контент и браузерные игры.

Системы, включённые в поставку «Вердж3Д», никак не ограничивают ваши возможности по созданию собственных визуализаций для 3Д-коммерции (в том числе с использованием виртуальной и дополненной реальности). В базовом исполнении это могут быть обычные крутящиеся модели товаров, интегрированные в стороннюю систему электронной коммерции (в частности, «Ву Коммерц»).

Электронное обучение В составе «Вердж3Д» имеется следующий набор функций для создания систем электронного обучения:
  • Генератор курсов в формате SCORM 1.2 (SCORM 1.2), которые могут быть запущены в любой совместимой системе электронного обучения: «Модл» (Moodle), «Чамило» (Chamilo), «SCORM Клауд» (SCORM Cloud), «Канвас» (CANVAS), «Ай Спринг» (iSpring).
  • Пазлы для SCORM 1.2.
  • Более 300 других пазлов для реализации интерактивных обучающих материалов.
  • Поддержка устройств на базе «Хром ОС» для создания и запуска 3Д приложений.
Виртуальная реальность

Поддерживаются следующие устройства виртуальной реальности:

  • «Мета Квест»
  • «Эйч-Ти-Си Вайв»

В целом, любое устройство, поддерживающее стандарт WebXR должно работать в «Вердж3Д».
Дополненная реальность

Поддерживается в «Хроме» для «Андроида» (требуется мобильное устройство, поддерживающее ARCore, также известное как Google Play Services для AR) и Mozilla WebXR Viewer на «Ай ОС».

Также вы можете использовать упрощённый режим дополненной реальности на устройствах «Айос»/«Аайпад» с использованием формата USDZ.

Шейдеры

Пользовательские шейдеры могут быть добавлены с использованием OpenGL ES Shading Language 2.0 (WebGL 1.0) или 3.0 (WebGL 2.0).

Кроме того, имеется возможность экспорта и конвертации шейдеров для процедурных текстур в формате OSL.

Физика Физический модуль обеспечивает обнаружение столкновений, динамическое перемещение объектов, поддержку персонажей и транспортных средств, имитацию пружин, веревок и ткани. Физические симуляции можно производить с помощью пазлов или «Ява Скрипта».
Звук «Вердж3Д» поддерживает следующие подсистемы аудио (с помощью звуковых пазлов или с «Ява Скрипта»).
  • Эффективная подсистема, основанная на HTML5-audio, для проигрывания длительной фоновой музыки.
  • Подсистема, основанная на WebAudio, для проигрывания коротких звуков с минимальной задержкой.
  • Синтезатор, способный проигрывать больше 100 музыкальных инструментов, ударных, а также проигрывать MIDI-файлы.
В дополнение, вы можете программировать сложные звуковые эффекты с помощью «Ява Скрипта».
Видео Видео текстуры можно назначать и воспроизводить с помощью специальных пазлов или «Ява Скрипта».

Демо и примеры В состав «Вердж3Д» входит ряд полнофункциональных демо-приложений, доступных совершенно бесплатно в магазине ассетов:
  • «Аркадные гонки» — гонки с реалистичной физикой.
  • «Дополненная реальность» — базовое демо-приложение для дополненной реальности.
  • «Мясорубка» — показывает работу секущих плоскостей, используемых для показа внутренней структуры инженерно-технических изделий.
  • «Фоторамка» — виртуальная фоторамка с возможностью загрузки, хранения и публикации изображений.
  • «Электронное обучение» — простое приложение для электронного обучения.
  • «Приключения фермера» — аркадная браузерная игра.
  • Firstperson Camera — управление камерой от первого лица.
  • «Глобальное освещение» — реалистичная визуализация интерьера (используются кубические карты отражений и источники освещения в виде области).
  • «glTF» — демонстрация материалов совместимых с glTF 2.0.
  • «Промышленный робот» — интерактивная симуляция робота. Поддерживается режим виртуальной и дополненной реальности.
  • «Ювелирный конфигуратор» — продвинутый ювелирный конфигуратор для электронной коммерции.
  • «Загружаем-выгружаем» — пример использования динамической загрузки ассетов.
  • «Параметрические модели» — 3Д-конфигуратор параметрических моделей.
  • «Кресло» — 3Д-конфигуратор и готовый к использованию шаблон интернет-магазина.
  • «Кольцо» — простой 3Д-конфигуратор ювелирных изделий с возможностью онлайн-заказа.
  • «Скутер» — мощный 3Д-конфигуратор с реалистичными материалами и широким выбором.
  • «Скролл-анимация» — демонстрация возможностей современного прокручиваемого сайта с настраиваемой 3Д-моделью.
  • «Простой конфигуратор» — простое обучающее демо для электронной коммерции.
  • «Спиннер» — простая презентация продукта с изменяемыми материалами и анимацией.
  • «Швейцарский нож» — продвинутое обучающее демо-приложение электронной коммерции.
  • «Подогреватель чайников» — необычное интерактивное 3Д-приложение созданное при помощи пазлов.
  • «Телевизор» — пример использования видеотекстур.
  • «Виртуальная реальность» — простое приложение с возможностью перемещения по сцене в режиме виртуальной реальности.

Управление камерой Эти режимы управления камерой доступны по умолчанию:
  • Орбитальные контролы — привязка камеры к определенному объекту в сцене.
  • Полёт — свободное перемещение камеры в сцене.
  • FPS (шутер от первого лица) — управление камерой от первого лица.
  • Контрол виртуальной реальности — специализирвоанный режим камеры для использования в виртуальной реальности.
