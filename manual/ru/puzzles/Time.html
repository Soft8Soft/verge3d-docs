Пазлы времени (Time)

Эти пазлы отсчитывают время и запускают события, основанные на времени.

Описание пазлов

[anchor:after]

after

Ожидает заданное количество времени, а затем запускает пазлы, помещенные в слот "do".

[anchor:every]

every

Ожидает определенное количество времени, а затем запускает пазлы, помещенные в слот "do". Затем повторяется.

[anchor:every_frame]

every frame

Запускает пазлы, помещенные в слот "do" каждый кадр рендеринга (обычно со скоростью 60 кадров в секунду).

[anchor:elapsed_delta]

elapsed delta

Выводит количество времени (в секундах), прошедшее с предыдущего кадра рендеринга. Может использоваться с пазлом"every frame" для реализации независимой от кадров анимации.

[anchor:elapsed_total]

elapsed total

Выводит количество времени (в секундах), прошедшее с запуска приложения. Может использоваться с пазлом "every frame" для реализации процедурной анимации и различных визуальных эффектов.

[anchor:set_timer]

set timer

Запускает пазлы, помещенные в слот "do", после заданной задержки. Повторяет это действие количество раз, указанное в слоте "repeat", каждый раз после той же задержки. Специальный идентификатор может быть использован для отмены или прекращения действия таймера.

[anchor:remove_timer]

remove timer

Удаляет предварительно установленный таймер, используя его ID.

[anchor:system_date_time]

system date / time

Возвращает системную дату и время. Включите флажок GMT, чтобы получить время зоны UTC/GMT вместо местного часового пояса.
