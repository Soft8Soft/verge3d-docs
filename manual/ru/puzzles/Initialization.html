Перечень пазлов

[anchor:configure_application]

configure application

Позволяет задать параметры инициализации приложения, включая некоторые параметры создания контекста WebGL.

compressed assets
Заставляет приложение загружать сжатую сцену в формате .xz вместо обычного .gltf (см. Asset Compression для большей информации).
default fullscreen button
Включает стандартную кнопку полноэкранного режима, расположенную в правом верхнем углу.
transparent background
Делает фон прозрачным, чтобы основные части веб-страницы были видны через канвас WebGL. Включите этот параметр, чтобы встроить 3Д-контент на веб-страницу или использовать функцию дополненной реальности.
enable screenshots
Позволяет снимать скриншоты окна рендеринга (устанавливает свойство `preserveDrawingBuffer` контекста WebGL в `true`). Включите этот параметр, если собираетесь использовать пазл take screenshot.
fade annotations
Делает аннотации полупрозрачными, когда они загораживаются объектами сцены.
[anchor:setup_preloader]

setup preloader + percentage

Удаляет прелоадер по умолчанию и передает обратные вызовы событий для обработки событий прогресса прелоадера.

В следующем примере пользовательский прелоадер представлен с контейнером HTML-элементом (например, <div>) с идентификатором "preloader". Внутри него находится другой элемент с идентификатором "loading progress" (например, <span>), в котором выводится процент загрузки с помощью пазла *percentage*. Кроме того, ширина третьего элемента (например, <div>) динамически изменяется для графической визуализации прогресса загрузки.

[anchor:load_font]

load font

Загружает шрифт по указанному УРЛ-адресу и делает его доступным через CSS-правило *font-family*.

Вы можете использовать этот шрифт в HTML пазлах и генерировать текстовые текстуры на его основе.
