Пазлы Отладки

Эти пазлы можно использовать для получения и вывода значений переменных, измерения производительности и получения информации о GPU.

Перечень пазлов

[anchor:print_to_console]

print to console

Выводит данные любого типа в консоль браузера. Последнюю обычно можно открыть клавишей F12 (Хром, Файрфокс в Виндоус, Линукс). На Mac используйте меню View > Developer > JavaScript Console (Option-Cmd-J) в Хром или меню Develop > Show Error Console (Option-Cmd-C) в Сафари.

[anchor:print_performance_info]

print performance info

Записывает профиль производительности в течение 1 секунды и выводит его в консоль браузера. Последнюю обычно можно открыть клавишей F12 (Хром, Файрфокс в Виндоус, Линукс). На Mac используйте меню View > Developer > JavaScript Console (Option-Cmd-J) в Хром или меню Develop > Show Error Console (Option-Cmd-C) в Сафари.

[anchor:get_GPU]

get GPU

Выводит информацию о GPU пользователя - производитель, например: NVIDIA Corporation, Apple Inc., Qualcomm, и модель GPU, например: GeForce GTX 1060 3GB/PCIe/SSE2, Apple A9 GPU, Adreno (TM) 330.
