Пазлы селекторов (Selectors)

Пазлы из этой категории используются для выбора 3Д-объектов, групп, анимационных клипов и материалов для вставки в другие пазлы.

Перечень пазлов

[anchor:select_object]

select object

Выпадающий список, содержит алфавитный список всех объектов в сцене.

[anchor:all_objects]

all objects

Обозначает все объекты в сцене, чтобы операция могла быть применена к ним в массовом порядке.

Этот пазл не итерируемый, в частности он не может быть подан на вход пазла for each in list. Чтобы сделать проход по списку объектов, используйте пазл get objects from

[anchor:select_group]

select group

Выпадающий список содержит алфавитный список всех групп объектов, представленных в сцене. Операция может быть применена к группе объектов в большом количестве.

Что на самом деле обозначается понятием "группа" зависит от пакета моделирования. В «Блендере» это коллекция/collection объектов (самая верхняя коллекция "Scene Collection" не отображается в пазлах):

В «3дс Максе» это либо группа/group ("Furniture"), либо выбранное множество/selection set ("Furniture2"):

В «Майе» это либо группа/group ("Furniture"), либо множество/set ("Furniture2"):

Этот пазл не итерируемый, в частности он не может быть подан на вход пазла for each in list. Чтобы сделать проход по списку объектов, используйте пазл get objects from

[anchor:select_animation]

select animation

Выпадающий список содержит алфавитный список всех анимационных клипов, представленных в сцене. Клип анимации называется в соответствии с объектом, к которому применяется анимация в «Блендере», «3дс Максе» или «Майе».

[anchor:select_material]

select material

Выпадающий список содержит алфавитный список всех материалов, которые присутствуют в сцене.

[anchor:get_entity_by_name]

get by name

Выбирает сущность (объект, группу, анимационный клип или материал) по имени.

Наиболее часто используется совместно с динамической загрузкой сцен. На этапе когда сцена не загружена, вы не можете использовать обычные селекторы для выбора объектов, поскольку они ещё не присутствуют на сцене.
