Ключи формы (морфинг) / Блендер

Ключи формы (в «Вердж3Д» носят название морфинг) это функция, позволяющая изменять геометрию объекта в реальном времени. Наиболее распространенным вариантом использования ключей формы является создание параметрических объектов, у которых могут изменяться размеры, форма, толщина, отверстия, контуры и др.

Настройка

Ключи формы экспортируются из «Блендера» по умолчанию. В «Вердж3Д» поддерживаются только ключи формы типа *Relative*.

[anchor:animation]

Анимация

Чтобы анимировать ключи формы в «Вердж3Д», добавьте ключевые кадры для их параметра Value. Анимация ключей формы работает точно также, как и анимация других параметров. Вы можете воспроизводить анимацию при запуске приложения, либо использовать для этого анимационные пазлы.
