Анимация / Блендер

Объектная анимация

Ключевые кадры анимации могут быть добавлены в «Блендер» стандартным путём, через меню Object → Animation → Insert Keyframe... (или при нажатии горячей клавиши *I*), после этого, на временной шкале выбирается другой кадр, объект перемещается/вращается/масштабируется и добавляется новый ключевой кадр для этих изменений.

Ознакомьтесь со следующим обучающим видео:

[anchor:Skeletal_Animation]

Скелетная анимация

Скелетная анимация объекта в «Блендере» осуществляется с использованием арматуры. «Вердж3Д» поддерживает максимум 4 веса на одну кость (взаимодействовать с костью будут только 4 веса с самым большим значением влияния, остальные будут проигнорированы). Кости могут быть анимированы в *Pose Mode* (Ctrl+Tab), используя меню Pose → Animation → Insert Keyframe... (горячая клавиша *I*), после чего, на временной шкале выбирается другой кадр, кость перемещается/вращается/масштабируется и с данными изменениями добавляется новый ключевой кадр.

[anchor:Shape_Key_Animation]

Анимация ключей формы (морфинг-анимация)

В «Блендере» вы можете добавить ключи формы к объекту стандартным путём (для получения более подробной информации пройдите по ссылке).

Значения ключей формы могут быть анимированы в Блендер нажатием ПКМ на Value и выборе *Insert Keyframe* (или при наведении на него курсора и нажатии горячей клавиши *I*), затем выбирается другой кадр на временной шкале, в поле ввода устанавливается новое значение Value и добавляется новый анимационный ключ для этого значения.

[anchor:Material_Animation]

Анимация материалов

В отличие от «Блендера», в «Вердж3Д» можно анимировать параметр материала, только подключив к нему ноду *Value* или *RGB*.

Значения цветов заданные этими нодами могут быть анимированы в «Блендере» нажатием ПКМ на соответсвующую ноду (или при наведении на него курсора и нажатии горячей клавиши *I*), затем выбирается другой кадр на временной шкале, устанавливается новое значение или цвет, которым добавляется анимационный ключ.

[anchor:light_animation]

Анимация источников освещения

«Вердж3Д» поддерживает анимацию интенсивности источников освещения (носит название *Power* для точечных источников, прожекторов и источников-областей и *Strength* для солнечных источников).

Также вы можете управлять источниками процедурно, с использованием пазла set light param.

[anchor:Path_Animation]

Анимация пути

Чтобы проанимировать движение объекта вдоль кривой вам понадобится добавить на объект ограничитель Follow Path. Далее нужно выбрать объект *path* (кривую) и открыть панель *Object Data Properties*, на которой расположена панель *Path Animation*. Здесь необходимо выставить кейфреймы анимации для параметра *Evaluation Time*.

В настоящий момент *twist method* может быть только *Z-Up*.

[anchor:Procedural_Animation]

Процедурная анимация

Вместо того, чтобы проигрывать анимацию созданную в «Блендере», вы можете использовать пазлы или «Ява Скрипт» чтобы изменить координаты объектов и другие параметры. В частности, для этого хорошо подходит пазл animate param.

Параметры воспроизведения

[anchor:Verge3D_Per_Object_Settings]

Настройки «Вердж3Д» на объектах

Каждому объекту в «Блендере» назначаются настройки «Вердж3Д», включая настройки воспроизведения анимации.

Настройки анимации:

Auto Start
Отвечает за начало проигрывания анимации при загрузке сцены.
Loop Mode
Режимы воспроизведения: *Repeat* — проигрывает анимацию от начала до конца, затем повторяет, *Once* — проигрывает один раз, затем останавливается, *Ping Pong* — проигрывает от начала до конца, затем воспроизводит анимацию в обратную сторону.
Repeat Infinitely
Если необходимо, чтобы анимация объекта проигрывалась бесконечно. Действительно только для режимов *Repeat* и *Ping Pong*.
Repeat Count
Отвечает за количество раз, которое будет проиграна анимация. Действительно только для *Repeat* и *Ping Pong*, если режим *Repeat Infinitely* отключен.
Offset
Какое количество кадров должно быть пропущено перед началом воспроизведения анимации.
[anchor:Interpolation]

Интерполяция

Меню переключения интерполяции между ключевыми кадрами вызывается нажатием ПКМ в окне временной шкалы в «Блендере» (горячая клавиша *T*).

«Вердж3Д» поддерживает все режимы интерполяции доступные в «Блендере».

[anchor:Frame_Rate]

Частота кадров

Значение частоты смены кадров (по умолчанию 24 кадра в секунду) влияет на скорость воспроизведения и может быть изменено во вкладке *Output Properties*.

[anchor:Animation_Control_with_Puzzles]

Управление анимацией с помощью пазлов

Вы можете использовать пазлы анимации для воспроизведения предварительно созданной анимации (для получения более подробной информации пройдите по ссылке).

Анимация также может быть реализована с использованием других пазлов — смотрите разделы Камера и свет и Время.
