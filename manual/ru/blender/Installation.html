Необходимые условия

«Вердж3Д» может работать поверх «Блендера», пакета 3Д-моделирования и анимации с открытым исходным кодом, который необходимо установить заранее (поддерживаются версии 3.4-4.4). Вы можете бесплатно загрузить «Блендер» с официального сайта.

[anchor:Downloading_Verge3D]

Загрузка «Вердж3Д»

Загрузите «Вердж3Д для Блендера» для своей операционной системы (.exe-установщик для «Виндоус» или .xz-архив для «Мак ОС»/«Линукс»/«Хром ОС»).

Установка в «Мак ОС» и «Линукс»

Распакуйте архив с программой «Вердж3Д» (имеет расширение .xz) в каталог по вашему выбору. Убедитесь, что у вас достаточно разрешений на чтение и запись в эту папку. Распаковка в системный каталог не рекомендуется. Распакованная на данном этапе папка будет именоваться в настоящей документации как установочная папка «Вердж3Д».

В «Блендере» вызовите окно *Preferences*, выбрав соответствующую опцию в меню *Edit*.

Следующие шаги будут зависеть от используемой версии «Блендера».

«Блендер» 4.2 или более поздняя

Перейдите на вкладку *Get Extensions*, вызовите меню *Repositories*. В открывшемся диалоге нажмите кнопку *Add* (+) чтобы добавить новый репозиторий.

Для этого выберите *Add Local Repository*.

В открывшемся диалоге укажите имя репозитория "Soft8Soft", включите *Custom Directory*, а затем укажите папку "addons", расположенную внутри папки с распакованным «Вердж3Д». Нажмите на кнопку *Create*.

Указывайте папку внимательно! Получившийся путь должен заканчиваться на "addons", а не на "addons/verge3d".

Выберите *View Details* из меню дополнения (галочка *v*) чтобы перейти к его настройкам (либо найдите аддон «Вердж3Д» на вкладке *Add-ons*).

Включите аддон «Вердж3Д», а затем закройте окно *Blender Preferences*.

Установка завершена!

«Блендера» 4.1 или более ранняя

На вкладке *File Paths* найдите панель *Scripts Directories*. В ней нажмите на кнопку *Add* (или +) и укажите папку, в которую был распакован «Вердж3Д». Закройте окно *Preferences* и перезапустите «Блендер».

Снова откройте окно *Preferences* и перейдите на вкладку *Add-ons*. Найдите и включите аддон «Вердж3Д», после чего закройте окно.

Установка завершена!

Установка «Блендера» в «Хром ОС»

Чтобы установить «Блендер» в «Хром ОС» запустите терминал:

Вызовите следующую команду чтобы обновить репозиторий:

sudo apt update

Далее установите пакет «Блендера» с необходимыми зависимостями:

sudo apt install blender

«Вердж3Д» требует для работы установленную библиотеку *numpy* (используется во время экспорта), а также утилиту *zenity* (для работы диалогов диспетчера приложений). Оба пакета устанавливаются аналогично:

sudo apt install python3-numpy zenity

«Блендер» установлен! Вы можете запускать его с помощью ярлыка как любое другое приложение для «Хром ОС»:

«Блендер» идущий в поставке с «Хром ОС» может быть несколько устаревшим (например в «Хром ОС» 121+ используется версия 3.4.1). Если вам нужен более свежий «Блендер», скачайте его с сайта blender.org и установите в какой-нибудь папке внутри «Линукс»-окружения.

Установка «Вердж3Д»

Откройте .xz-архив «Вердж3Д» и распакуйте его в каталог по вашему выбору. Однако, чтобы упростить настройку путей в «Блендере», рекомендуется поместить *verge3d_blender* в папку *Linux files*:

В «Блендере» вызовете окно *Preferences* выбрав соответствующий пункт меню *Edit*, затем перейдите во вкладку *File Paths*:

На вкладке *File Paths* найдите поле *Scripts* и установите его в путь к распакованной папке «Вердж3Д». Закройте окно *Preferences* и перезапустите «Блендер».

Откройте окно *Preferences* снова и перейдите во вкладку *Add-ons*. Найдите и включите аддон «Вердж3Д», после чего закройте окно настроек.

Нажмите на кнопке *App Manager* и проследуйте к следующему этапу.

[anchor:installation_chromeos_first_run]

Первый запуск

При первом запуске показывается окно с заставкой с предложением выбрать путь к папке «Вердж3Д»-приложений. Вы можете оставить предложенный по умолчанию путь, в этом случае приложения будут создаваться внутри «Линукс»-окружения:

Выбрав путь к папке приложений, нажмите *Apply*. Установка завершена!

В качестве альтернативы, если вы хотите держать ваши приложения в самой «Хром ОС», сделайте следующее. Создайте любую папку внутри папки *Files*, кликните на ней правой кнопкой мыши и выберите пункт меню *Share with Linux*. Подобным образом вы можете пробросить внутрь окружения «Линукса» удалённую папку из «Гугл Драйва» или папку расположенную в локальной сети «Виндоус».

Проброшенная папка будет доступна в папке /mnt/chromeos/... «Линукс»-окружения:

Возникает ошибка «ModuleNotFoundError: No module named 'numpy'»

В момент активации плагина «Вердж3Д» в «Линуксе» или «Хром ОС», вы можете увидеть следующее сообщение, говорящее об отсутствии модуля *numpy*:

Чтобы решить эту проблему, установите пакет *numpy* с помощью пакетного менеджера вашего дистрибутива, например в «Убунту»:

sudo apt install python3-numpy

После чего активируйте аддон «Вердж3Д» повторно.
