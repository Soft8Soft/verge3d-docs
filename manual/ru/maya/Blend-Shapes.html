Blend Shapes (морфинг) / Майя

Морфинг это функция, позволяющая изменять объекты в режиме реального времени. Наиболее распространенным вариантом использования этой функции является лицевая анимация. Морфинг также хорошо подходит для создания параметрических объектов, которые могут интерактивно изменять свои размеры, форму, толщину, округлости, профиль и т.д.

Настройка

В «Майе» откройте Shape Editor и настройте цели для своих объектов. Обратите внимание, что из-за аппаратных ограничений вы можете создать только один морфинг-объект (с несколькими целями).

[anchor:animating]

Анимация морфинга

«Вердж3Д» поддерживает анимацию ключевых кадров морфинга. Смотрите раздел «Анимация» для получения более подробной информации.

[anchor:controlling_with_puzzles]

Управление с помощью пазлов

Для управления морфингом в реальном времени есть два пазла: get morph factor и set morph factor.
