Использование библиотеки материалов / Майя

Библиотека материалов «Вердж3Д», это коллекция готовых к использованию материалов, которые вы можете назначить своей модели в «Майе». Ниже, вы можете ознакомиться с процессом установки и эффективным методом использования данных материалов.

[contents] [anchor:installing]

Установка

В магазине «Софт Эйт Софт» доступны следующие библиотеки материалов:

  • Базовая
  • Промышленная

Просто распакуйте архив с пакетом библиотек в любую папку на вашем компьютере. Внутри этой папки находятся файлы *mb* (по одному файлу на материал). Текстуры, используемые в материалах, находятся в папке *material_maps*. Папка *environments* содержит 3 варианта HDR-изображений, которые вы можете использовать для окружения (каждый вариант представлен в трех разрешениях: 1К, 2К и 4К).

[anchor:adding_materials_to_project]

Добавление материалов к проекту

Для того, чтобы использовать материал из библиотеки в проекте, вызовите команду File → Import в окне редактора Hypershade:

Перейдите в папку библиотеки, выберите файл *mb*, укажите *Load no references* в *Load Settings*, затем нажмите *Import*:

После того как материал импортирован, вы должны иметь возможность назначить его модели. Выберите модель, щелкните правой кнопкой мыши, затем Assign Existing Material → Имя:

Последним шагом является удаление неиспользуемых ссылок на библиотеку материалов:

Настройка окружения

Без настройки окружения материалы будут выглядеть темными и не отражающими. Окружение уже настроено в проекте куба «Вердж3Д» по умолчанию, который вы можете использовать в качестве основы для собственного приложения.

В качестве альтернативы вы можете настроить среду с нуля, как показано здесь. Для этого можно использовать любую из HDR-текстур в папке *environments* пакета материалов.

[anchor:uv_mapping]

UV Mapping

Большинство материалов в библиотеке требуют, чтобы модели были отображены в ультрафиолете. Для достижения наилучших результатов следует развернуть модели.

Перемещение текстур в папку приложений

Для обеспечения целостности вашего приложения (важно, если вы передаете исходники проекта третьим лицам), вы можете сохранить текстуры, используемые импортированными материалами, в папке приложения, как описано ниже.

  1. Откройте *File Path Editor* в «Майе»:
  2. Найдите текстуры, которые используются импортированным материалом. Скопируйте эти текстуры из *material_maps* в папку приложения:
  3. Выберите эти материалы в редакторе, затем нажмите на кнопку *Repath Files*. Появится новое диалоговое окно, укажите *New path* к директории приложения, включите опцию *Repath Currently Resolved Files*, затем нажмите *Repath*:

Вы можете проверить целостность вашего приложения, переместив его папку в другое место в вашей файловой системе. Откройте файл *mb* (или *ma*) и убедитесь, что не возникло ошибок с отсутствующими ассетами.
