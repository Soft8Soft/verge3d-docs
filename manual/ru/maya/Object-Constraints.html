Данная страница содержит информацию об ограничителях объектов «Майи», которые можно использовать в «Вердж3Д».

[contents] [anchor:Constraints_and_Settings]

Перечень поддерживаемых ограничителей

Ниже перечислены объектные ограничители «Майи», которые могут быть использованы в «Вердж3Д». Параметры поддерживаемые «Вердж3Д» выделены зеленым цветом.

[anchor:parent_constraint]

Parent

Этот ограничитель задаёт отношение "родитель/потомок" между объектами.

[anchor:point_constraint]

Point

Этот ограничитель заставляет объект, к которому он применяется, иметь то же местоположение, что и его цель.

[anchor:orient_constraint]

Orient

Этот ограничитель заставляет объект, к которому он применяется, соответствовать вращению его цели.

[anchor:scale_constraint]

Масштаб

Этот ограничитель заставляет объект, к которому он применяется, соответствовать масштабу цели.

[anchor:aim_constraint]

Aim

Этот ограничитель применяет вращение к своему владельцу, так что он всегда направляет заданную ось *Aim* на свою цель, а другая ось *Up* постоянно поддерживается максимально выровненной с глобальной осью Z. Таким образом, с помощью этого ограничителя можно реализовать билбординг (псевдотрёхмерность).

«Вердж3Д» поддерживает только выровненные по оси векторы *Aim* и *Up*, такие как [1,0,0] или [0,-1,0].

[anchor:motion_path_constraint]

Motion Path

Этот ограничитель заставляет двигаться объект вдоль кривой.

1. «Вердж3Д» поддерживает только *Vector* в качестве значения настройки *World Up Type*.
2. Настройка *follow* должна быть включена, в противном случае ограничитель не будет экспортироваться.

[anchor:use_with_puzzles]

Использование с пазлами

Все назначенные в «Майе» ограничители могут быть удалены, отключены и включены с помощью пазлов.

Кроме того, ограничители *Point/Orient/Scale* могут быть назначены во время работы приложения с помощью пазлов.
