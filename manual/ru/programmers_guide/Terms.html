Словарь терминов компьютерной графики и программирования для всемирной паутины

Ниже приведён список русскоязычных терминов и определений, используемых в компьютерной графике и программировании для всемирной паутины.

2Д
относящийся к двумерному пространству. Модель мира, в которой имеется всего два измерения: длина, ширина или иначе — плоская модель мира.
3Д
относящийся к трёхмерному пространству. Модель мира, в которой имеется три измерения: длина, ширина, высота (глубина) или иначе — объёмная модель мира.
3Д-паутина
набор технологий всемирной паутины и сопутствующего программного обеспечения для создания и отображения трёхмерной графики, в том числе с использованием устройств виртуальной и дополненной реальности. Включает в себя: ПГБ, ПГП, ФПГБ, «Вердж3Д» и др.
3дс Макс
коммерческий пакет для создания трёхмерной графики, разрабатываемый компанией «Автодеск».
Автодеск
крупнейшая компания-разработчик инженерного и графического программного обеспечения из США.
Автокад
профессиональная система автоматизированного проектирования (САПР) и черчения. Разрабатывается компанией «Автодеск», США.
АЛЗМ
алгоритм Лемпеля-Зива-Маркова. Высокоэффективный алгоритм сжатия (компрессии) данных, а также формат файлов, сжатых с помощью этого алгоритма. Разработан российским программистом Игорем Павловым. В частности, данный алгоритм используется в движке «Вердж3Д» для сжатия геометрии и метаданных трёхмерных сцен, с целью ускорения загрузки приложений.
Арнольд
профессиональный движок для рендеринга (рендер), использующий трассировку лучей для получения фотореалистичных изображений. Разрабатывается компанией «Автодеск», входит в состав ряда продуктов компании: «3дс Макс», «Майя». Также имеется отдельная платная версия.
АУУВ, «Куда»
архитектура унифицированных устройств вычисления. Архитектура графических процессоров и программный интерфейс для высокопроизводительных параллельных вычислений, разработанный компанией «Нвидиа». На текущий момент поддерживается исключительно на оборудовании компании «Нвидиа».
БААА
быстрый аппроксимирующий анти-алиасинг. Основанный на постобработке алгоритм сглаживания 3Д-визуализации, разработанный компанией «Нвидиа».
Блендер
пакет трёхмерного моделирования, анимации и рендеринга с открытым исходным кодом. Распространяется свободно.
Буллет
программная библиотека для проведения физических симуляций (физический движок) с открытым исходным кодом. Основные сферы применения — разработка компьютерных игр и приложений виртуальной реальности.
Вавилон, Вавилон-Джей-Эс
открытый движок для создания трёхмерных визуализаций во всемирной паутине, разрабатывается компанией «Майкрософт», США.
Веб
жаргонное название всемирной паутины.
Вебфлоу
визуальный редактор, использующийся для создания паутинных страниц без программирования и вёрстки. Разрабатывается одноименной компанией «Вебфлоу», США.
Вердж3Д
программное обеспечение для создания трёхмерных приложений, работающих во всемирной паутине. Используется для создания магазинов, конфигураторов товаров, 3Д-презентаций, обучающих курсов, научных приложений, приложений виртуальной и дополненной реальности, браузерных игр и др.
ВДД
имеющий высокий динамический диапазон. В контексте компьютерной графики обозначает текстуру или рендер с увеличенным диапазоном цветопередачи (больше 8-бит на каждый канал цвета).
Вирей
профессиональный движок для рендеринга (рендер), использующий трассировку лучей для получения фотореалистичных изображений. Разрабатывается компанией «Хаос Груп», Болгария.
Вордпресс
открытая система управления контентом, написанная на языке ПГТ. Используется для создания сайтов любой сложности, на сегодняшний день является крупнейшей в мире по числу установок.
Ву Коммерц
система электронной коммерции, выполненная в виде плагина для системы управления контентом «Вордпресс». Является наиболее популярным решением для создания магазинов во всемирной паутине.
ВШЯ
высокоуровневый шейдерный язык. Шейдерный язык, используем совместно с «Директ Икс».
Вьюпорт 2.0
движок для рендеринга (рендер), используемый в программе «Майя» для отрисовки содержимого вьюпорта. Обладает высокой производительностью, позволяя получать результат рендеринга в реальном времени.
ГИМП
свободно-распространяемая программа для редактирования изображений.
Директ 3Д
составная часть «Директ Икс», программный интерфейс (ИПП) для создания трёхмерной графики для ОС «Виндоус».
Директ Икс
набор программных интерфейсов для создания компьютерных игр и других мультимедийных приложений для операционной системы «Виндоус».
ЗРЗ
значения, разделённые запятыми. Простой формат для хранения табличных данных. Строки таблицы представляют собой строчки текстового файла, колонки разбиваются с помощью запятых (реже других символов).
Иви
открытый движок для рендеринга (рендер), используемый в программе «Блендер» для отрисовки содержимого вьюпорта. Обладает высокой производительностью, позволяя получать результат рендеринга в реальном времени.
Иллюстратор
редактор векторной графики, разрабатываемый компанией «Адоби», США.
ИПП
интерфейс программирования приложений, то же что и «программный интерфейс», способ взаимодействия программы с другой программой или библиотекой.
Кубомапа
жаргонное название кубической текстуры (кубической карты), то есть текстуры, используемой для наложения отражений на трёхмерные объекты.
Конфигурация, цена, предложение
вид специализированного программного обеспечения, используемого в сфере электронной коммерции. Сочетает в себе функции конфигурирования продуктов под запросы покупателя, расчёта итоговой цены и формирования коммерческих предложений.
Кордова
открытый инструмент для преобразования паутинных приложений в кроссплатформенные мобильные приложения, работающие на платформах «Ай ОС» и «Андроид».
КТС
язык каскадных таблиц стилей. Используется совместно с ЯРГТ для задания внешнего вида паутинных страниц.
Майя
пакет трёхмерного моделирования, анимации и рендеринга от компании «Автодеск».
Меш
описание геометрии объекта в виде вершин, рёбер и граней. Зачастую является синонимом слов «геометрический объект» или «полигональный объект».
МСАА
мультисемпловый анти-алиасинг. Техника сглаживания 3Д-визуализации, использующая возможности графического оборудования по рендерингу геометрии на суб-пиксельном уровне.
МССИОС
модель ссылок на совместно используемые объекты содержимого. Стандарт систем электронного обучения, а также формат обучающих курсов, совместимых с данными системами.
НВГП
неспециализированные вычисления на графических процессорах. Техника использования графических процессоров видеокарт для математических вычислений общего назначения. Одной из технологий НВГП является АУУВ, разработанная компанией «Нвидиа» .
Ноуд, Ноуд-Джей-Эс
платформа для запуска языка программирования «Ява Скрипт» за пределами паутинного браузера. Позволяет создавать десктопные и серверные приложения с использованием данного языка.
НОЯ
нотация объектов «Ява Скрипта». Простой формат для хранения и обмена структурированными данными. Основные сферы использования: паутинное программирование, хранение настроек приложений, 3Д-паутина (в частности, формат ФПГБ был разработан на базе НОЯ).
ОГБ
открытая графическая библиотека. Стандарт и программный интерфейс (ИПП) для создания приложений, работающих с трёхмерной графикой.
ОШЯ
открытый шейдерный язык. Используется в движках трассировки лучшей.
ОЯВ
открытый язык вычислений. Открытый и кроссплатформенный программный интерфейс для высокопроизводительных параллельных вычислений. Является открытым аналогом технологии АУУВ.
Паутинное Аудио
технология работы со звуком с использованием обозревателей всемирной паутины. Используется для игр, интерактивных визуализаций, виртуальной реальности, а также для профессионального ПО для обработки звука. Поддерживается во всех современных обозревателях.
ПГБ
паутинная графическая библиотека. Стандарт рендеринга трёхмерной графики во всемирной паутине. Поддерживается во всех современных обозревателях.
ПГП
паутинный графический процессор. Экспериментальный стандарт рендеринга трёхмерной графики в интернете, идущий на смену ПГБ.
Питон
многофункциональный язык программирования. Основные сферы применения: паутинная разработка, написание встроенных скриптов для ряда приложений, научные и инженерные расчёты, анализ данных.
Полигон
треугольник, четырёхугольник или многоугольник, используемый для описания геометрии объектов (мешей).
Программный интерфейс
то же что и «ИПП», способ взаимодействия программы с другой программой или библиотекой.
ПРС, ПРС ИПП
передача репрезентативного состояния, набор правил взаимодействия программы с сервером. Чаще всего подразумевает набор специализированных УРЛ-ссылок, формирующий ИПП-сервера, а также формат передачи данных между клиентом и сервером на основе НОЯ.
ПСР
паутинная смешанная реальность. Технология взаимодействия с устройствами виртуальной и дополненной реальности из обозревателей всемирной паутины.
ПГТ
препроцессор гипертекста. Популярный серверный язык программирования, используемый для разработки узлов всемирной паутины.
Растеризация
процесс преобразования векторной или полигональной графики в растровое (состоящее из пикселей) изображение.
Рейтрейсинг
см. «трассировка лучей».
Рендер, рендерер
программное обеспечение для осуществления рендеринга, т.е. отрисовки трёхмерных моделей.
Рендеринг
процесс визуализации трёхмерной модели с помощью специализированной программы-рендера.
Сабстанс Пейнтер
программного обеспечение для текстурирования трёхмерных моделей. Разрабатывается компанией «Адоби».
Сайклз
движок для рендеринга (рендер), использующий трассировку лучей для получения фотореалистичных изображений. Распространяется свободно, входит в состав пакета 3Д-моделирования «Блендер».
СГ
«Си» для графики, высокоуровненый шейдерный язык, разработанный компанией «Нвидиа».
СДД, «Коллада»
совместная дизайнерская деятельность. Открытый формат хранения и обмена трёхмерными данными. В настоящее время вытесняется форматами ФПГБ и УОС.
Синема 4Д
коммерческий пакет для создания трёхмерной графики, разрабатываемый компанией «Максон», Германия.
Сканлайн
устаревший движок для рендеринга, поставляемый в составе пакета моделирования «3дс Макс». В настоящее время заменяется рендером «Арнольд».
Спекуляр, спекулярное отражение
зеркальное отражение, блик. Специфичный термин, используемый в компьютерной графике для обозначения отражения, противоположного по смыслу диффузному.
Трассировка лучей
или рейтрейсинг, метод визуализации моделей (рендеринг), основанный на симуляции взаимодействия поверхности с отдельными лучами света. Позволяет получить реалистичное изображение с качеством приближенным к фотографии (фотореализм). Недостаток — высокая вычислительная сложность алгоритмов, приводящая к низкой производительности рендеров, использующих трассировку лучей.
Три-Джей-Эс
открытый движок для создания трёхмерных визуализаций в интернете.
МПН
менеджер пакетов «Ноуд-Джей-Эс». Пакетный менеджер и коллекция готовых пакетов для разработчиков приложений на платформе «Ноуд-Джей-Эс».
УОС, УОСЗ
унифицированное описание сцен, унифицированное описание сцен в формате «Зип». Формат файлов, используемый для хранения 3Д-графики. Файлы в этом формате могут содержать геометрию, положение объектов, структуру сцен, анимацию, данные камеры, источники света, материалы, текстуры и другую информацию. Создан компанией «Пиксар», среди прочего используется на устройствах «Эпл» («Айфон», «Айпад») для отображения моделей в режиме дополненной реальности. Формат УОСЗ отличается от базового варианта УОС компоновкой всех содержащихся данных в едином файле, упакованном с помощью алгоритма компрессии «Зип» (отсюда и последняя буква в названии). Файлы в форматах УОС и УОСМ имеют расширения .usd и .usdz соответственно.
УРИ
унифицированный ресурсный идентификатор — строка идентифицирующая некоторый ресурс во всемирной паутине, чаще всего представляет собой ссылку (УРЛ).
УРЛ
унифицированный ресурсный локатор — ссылка на какой-либо ресурс во всемирной паутине.
ФКР
физически корректный рендеринг, набор алгоритмов для симуляции особенностей распространения света. Позволяет добиться высокой степени реализма изображения.
Фотошоп
программа для редактирования изображений от компании «Адоби».
ФПГБ
формат передачи графической библиотеки. Открытый формат файлов, используемый для хранения и передачи 3Д-графики. Файлы в этом формате могут содержать геометрию, положение объектов, структуру сцен, анимацию, данные камеры, источники света, материалы, текстуры и другую информацию. Формат создан с расчётом на применение во всемирной паутине. Файлы в формате ФПГБ имеют расширения .gltf/.bin или .glb.
ФПД
формат переносимых документов. Популярный формат распространения электронных документов, разработанный компанией «Адоби». Файлы данного формата имеют расширение .pdf.
Фрустум
усечённая пирамида обзора. Определяет пространство, видимое из конкретной камеры.
Фрустум каллинг
техника оптимизации рендеринга. Заключается в отсечении геометрии, не попадающей в пирамиду обзора камеры.
ЧВТП
«что видишь, то и получишь». Техника создания контента, при которой разрабатываемый продукт выглядит максимально приближенным к конечному результату. Термин обычно используется в контексте программного обеспечения для создания документов, презентаций и страниц всемирной паутины. Примеры ЧВТП-редакторов: «Майкрософт Офис», «Либреофис», «Вебфлоу».
шейдер
короткая программа выполняющаяся на графическом процессоре. Шейдеры чаще всего пишутся с использованием специализированных языков программирования, таких как ШЯГБ (язык шейдеров ОГБ), ВШЯ (язык шейдеров «Директ Икс»), ОШЯ (открытый язык для рендеров, использующих трассировку лучей) и др.
ШПЛ
шутер от первого лица, жанр компьютерных игр в которых управление камерой производится от первого лица. Термин применяется также и вне контекста компьютерных игр (в частности, в 3Д-паутинных приложениях) для обозначения аналогичного способа управления камерой.
ШЯГБ
шейдерный язык графической библиотеки. Шейдерный язык, используемый совместно с ОГБ.
Электрон
открытое программное обеспечение для преобразования паутинных приложений в кросс-платформенные десктопные приложения.
Эмбиент окклюжн
техника улучшающая реалистичность рендеринга посредством дополнительного затенения от непрямых источников. Степень этого затенения определяется тем, насколько видима рисуемая часть объекта, если смотреть не из камеры, а со стороны источников освещения.
Юнити
игровой движок разрабатываемый компанией «Юнити Технолоджис», США. Основное применение — создание игр для мобильных устройств.
Ява Скрипт
язык программирования, используемый для создания приложений для всемирной паутины.
ЯРГТ
язык разметки гипертекста. Используется для определения содержимого и внешнего вида страниц во всемирной паутине. Файлы данного формата имеют расширение .html или .htm.
