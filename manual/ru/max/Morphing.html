Морфинг / 3дс Макс

Морфинг — это функция, позволяющая изменять форму объектов в режиме реального времени. Наиболее распространенный вариант использования этой функции - лицевая анимация. Морфинг также хорошо подходит для создания параметрических объектов, которые могут интерактивно изменять свои размеры, форму, толщину, отверстия, профили и т.д.

Настройка

В «3дс Максе» добавьте модификатор Morpher и настройте каналы для целей морфа.

Учтите, что если канал морфинга не содержит целевого объекта, он не будет экспортирован.

[anchor:animating]

Анимация морфинга

«Вердж3Д» поддерживает ключевую анимацию весов каналов морфинга. Читайте раздел «Анимация», чтобы получить подробную информацию.

[anchor:controlling_with_puzzles]

Управление с помощью пазлов

Для манипулирования объектами морфа в реальном времени есть два пазла: get morph factor и set morph factor. Более подробную информацию смотрите в демонстрации [demoLink:parametric_models/index Параметрические модели].
