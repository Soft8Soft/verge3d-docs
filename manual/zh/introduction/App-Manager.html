应用管理器

Verge3D的应用管理器（App Manager）很容易操作，可用于创建和执行Verge3D应用的各种操作。

内容

[contents] [anchor:Running_App_Manager]

运行应用管理器

应用管理器（App Manager）以本地运行的Web服务器的方式实现，界面显示为类似于Wi-Fi路由器管理面板形式的网站。

默认情况下，应用管理器的地址为 localhost:8668 ， 您可以进行导航、刷新、书签等方式操作，服务器可以通过多种方式启动（见下文）。

[anchor:From_Blender_Max_Maya]

A. 从Blender、3ds Max或Maya中启动

若您已为3D软件安装了Verge3D插件，那启动它们时，会自动启动本地Web服务器。单击App Manager按钮（Blender），或从菜单选项中选择App Manager，都会在网页中显示应用管理器。

如果Web服务器碰巧因为某些原因关闭（如您关闭了3D软件的第一个实例），那么此操作也会将其重新启动。

[anchor:Using_Desktop_Application]

B. 从桌面快捷方式启动

即使没有安装Blender、3ds Max或Maya，也可以启动本地web服务器。只需单击Verge3D安装程序创建的桌面快捷方式图标（仅限Windows），App Manager页面将立即显示在web浏览器中。

如果Web浏览器已经在运行（例如您已经从3D编辑器启动了它），那么点击桌面快捷方式只会在浏览器中打开应用管理器。

[anchor:From_Command_Line]

C. 用命令行启动

安装Python 3，比如从 微软商店 。当您安装完成，启动 *命令提示符* 并输入：

cd verge3d_blenders

（或安装Verge3D的任何路径），接着对用于Blender版本的类型：

python manager\server.py BLENDER

对用于3ds Max的版本：

python manager\server.py MAX

或用于Maya的版本：

python manager\server.py MAYA

最后，打开浏览器，并访问此页面 localhost:8668 。macOS或Linux用户可以使用相同的命令，但需要使用正斜杠。

主页应用列表

启动或刷新主页时，应用管理器会扫描安装目录中 *applications* 文件夹目录，并显示出所有找到的文件夹。

对于列表中的每个应用，应用管理器都会显示在其文件夹下找到的所有.html、gltf/glb和3D文件（.blend、.max、.ma/mb）等。单击这些文件的图标，可以查看这些文件。

对于文件夹中包含较多文件的应用，单击其名称附近的三角形图标，可以展开查看应用文件细节。

可以使用右上角的搜索工具栏对应用进行过滤。

[anchor:Creating_Applications]

创建新应用

使用左侧栏目上的新建按钮，即可为Verge3D创建一个新应用。

您可以输入新应用的名称，然后从官方提供的模板来创建应用。默认有4套模板可控选用："Standard Light(标准明调模板)", "Standard Dark(标准暗调模板)"，"Blank Scene(空模板)", 以及"Code-Based(基于代码的模板)"。您也可以 创建和添加更多模板 以供选择。

您还可以在应用中包含可选模块：启用TrueType字体转换器(Font Convertor)，纹理解压缩模块（Texture Decompressor），以及在应用中包含物理引擎(Physics)。

新应用创建完成后，将显示一个链接，可使用该链接进行跳转到应用页面（或者您也可以直接将其关闭）。

新应用将可在主应用列表中显现。

发布应用

点击主页“操作”列中的发布按钮，即可发布对应的应用：

... 或者您也可以在应用详情页面的右侧点击拼图按钮：

此操作将开始将应用上传到Verge3D Network（可取消的操作），从而使其可以在线访问。

如果上传失败并显示消息 "The difference between your local system time and the server time is too large(本地系统时间与服务器时间之间的差异太大)"，则应更正计算机上的时间设置并重试。

您可以点击社交网络的图标进行分享，或者用手机扫描二维码在移动端查看应用。

已上传的文件，可以在 Network Directory 中查看或者删除。

[anchor:Upload_Archive]

上传应用压缩包

在应用详情页面上时，可使用右侧栏上的按钮上传应用文件夹的zip压缩包：

该操作将压缩您的应用文件夹，并将压缩包上传到Verge3D Network（可取消操作），以便可与您的同事、客户或Verge3D开发人员共享项目。

如果上传失败并显示消息 "The difference between your local system time and the server time is too large(本地系统时间与服务器时间之间的差异太大)"，则应更正计算机上的时间设置并重试。

上传完成后，您可从弹出窗口中复制zip压缩包的网络路径。

已上传的文件，可以在 Network Directory 中查看或者删除。

[anchor:Templates]

模板

[anchor:Whats_a_Template]

什么是模板

当创建一个新应用时，您可以选择一个模板来作为基础创建应用。默认有4套模板可控选用："Standard Light(标准明调模板)", "Standard Dark(标准暗调模板)"，"Blank Scene(空模板)", 以及"Code-Based(基于代码的模板)"。

这些模板的源文件可以在模板文件夹中找到，如"verge3d_blender\manager\templates"。您可以修改这些文件或依次创建一个在项目中使用的全新模板。

在templates文件夹中，还有一个隐藏的模板叫做 "Embeddable" ， 您可以用它来创建 React.js 或 Vue.js 应用。

有很多可能用来修改和改进模板的方法，您可以通过编辑嵌入的文件来改变布局，使用一些Java脚本代码或库，替换UI元素或预加载器，或添加一些默认包含在新应用中的拼图。

[anchor:Creating_Templates]

创建模板

点击左边栏的 *Settings* 按钮，然后选择 *Templates* 面板：

在显示的面板中,点击 + ，输入新模板的名称(例如，"My Template")然后点击 *Apply(应用)*：

这样，每次创建新应用时，都会显示您创建的自定义模板了：

默认情况下，使用您刚才自定义模板创建的新应用会是一个空文件夹，因为您还没有在其中提供文件。最简单的模板创建方法是直接复制一个现有模板作为基础（例如"Standard Dark"，标准暗调模板)，然后重命名文件夹为您刚才新建的模板名称（例如，"My Template")。

管理网络目录

点击左侧栏的云朵图标，即可打开 Verge3D Network 目录：

网络目录列出了您之前上传的所有文件、目录或 ZIP 存档：

在此页面上，您可以查看项目的大小以及资产上传到 Verge3D 网络的日期/时间（当地时区）。 对于目录，日期/时间信息对应于 最近上传 的子元素（文件或其他目录）。

General选项卡

Applications Folder
使用指定的绝对路径（例如，C:\Users\Alex\my_verge），用作存放应用的目录。
Notify about Verge3D updates
Verge3D主程序更新提醒：当Verge3D主程序发布新版本时，应用管理器会显示更新提醒（该选项默认开启）。
Upload app sources (models, puzzles, backups)
上传应用源文件（模型、拼图、备份数据）： 发布 功能通常只会上传在线运行应用所必须的文件，所以默认情况下会忽略一些文件，包括.blend, .max, .ma/.mb, visual_logic.xml, 以及v3d_app_data文件夹内的内容。如果你需要与同事共享这些文件，可以开启此选项。
Enable external server interface
启用外部服务器访问地址：使用本地Wi-Fi地址访问地址，代替默认使用的 localhost:8668。您可以从操作系统的网络属性首选项中查看该地址：Win 10, Win 7, Mac, Linux.开启后即可通过IP方式访问应用管理器了，例如：192.168.1.15:8668
Verge3D Network cache age (minutes)
Verge3D Network缓存更新时间（分钟）：设置Verge3D Network刷新已上传文件缓存的时间。需要在开发中快速迭代时，可将此值设置为0（立即刷新）。

UI选项卡

Select Theme
切换主题：为应用管理器和拼图编辑器切换“浅色”或“深色”模式外观。

Templates选项卡

Puzzles选项卡

Enable Performance Mode
启用性能模式

实验功能：拼图编辑器将使用一些优化措施，旨在提高拖动和缩放性能。 用户体验可能因特定基于Chromium的浏览器而异（如Google Chrome、MS Edge、Brave等）， 其中Chromium的浏览器改善最大， 而像Firefox和Safari这样的浏览器效果较小。 如果您的拼图设置已经过多， 以至于导致明显的性能问题， 则此功能可以帮助解决。
