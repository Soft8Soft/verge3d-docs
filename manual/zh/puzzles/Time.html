时间｜Time

此类拼图用于计算时间并触发基于时间的事件。

拼图参考

[anchor:after]

after(之后)

等待指定数量的时间，然后触发放在 "do" 插槽中的拼图。

[anchor:every]

every(每)

等待指定数量的时间，然后触发放在 "do" 插槽中的拼图。然后重复进行。

[anchor:every_frame]

every frame(每帧)

每渲染一帧（通常以每秒60帧的速度）就触发一次 "do" 插槽中的拼图。

[anchor:elapsed_delta]

elapsed delta(时间增量)

输出距离被执行的渲染帧到现在的时间增量（以秒为单位）。可以和 "every frame(每一帧)" 拼图一起使用，可以直接在拼图编辑器里创建动画。

[anchor:elapsed_total]

elapsed total(总耗时)

输出从应用开始运行到当前所经过的时间量（秒）。可以和 "every frame(每一帧)" 拼图一起使用，实现流程性动画和各种视觉效果。

[anchor:set_timer]

set timer(设置定时器)

触发放置在 "do" 插槽中的拼图，并在指定的延迟后触发。 在相同的延时后重复此动作若干次， 次数是在 "repeat" 中槽中指定的。 指定的ID可以用来覆盖或取消定时器。

[anchor:remove_timer]

remove timer(移除定时器)

通过ID删除一个先前设置的定时器。

[anchor:system_date_time]

system date / time(系统日期/时间)

返回系统日期和时间。启用GMT复选框可获得UTC/GMT区域的时间，而非本地时区时间。
