Debug拼图

这些拼图可以用来检索和打印变量值，测量性能，以及检索GPU信息。

拼图参考

[anchor:print_to_console]

print to console(打印到控制台)

打印任何类型的数据到浏览器控制台。浏览器控制台通常可以用F12键（Linux和Windows上的Chrome与火狐 ）打开。在Mac电脑中，Chrome使用 View > Developer > JavaScript控制台菜单（快捷键：Option-Cmd-J）开启，Safari需使用 Developer > Show Error Console菜单（快捷键：Option-Cmd-C）开启浏览器控制台。

[anchor:print_performance_info]

print performance info(打印性能信息)

记录1秒钟内的性能概况，并将其打印到浏览器控制台。浏览器控制台通常可以用F12键（Linux和Windows上的Chrome与火狐 ）打开。在Mac电脑中，Chrome使用 View > Developer > JavaScript控制台菜单（快捷键：Option-Cmd-J）开启，Safari需使用 Developer > Show Error Console菜单（快捷键：Option-Cmd-C）开启浏览器控制台。

[anchor:get_GPU]

get GPU(获取GPU信息)

输出用户的GPU信息——例如厂商信息：NVIDIA公司、苹果公司、高通公司，以及GPU的型号，例如：GeForce GTX 1060 3GB/PCIe/SSE2, Apple A9 GPU, Adreno (TM) 330.
