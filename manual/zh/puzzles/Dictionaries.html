此类拼图允许您有效地存储、传输和检索各种数据。特别是，字典对于存储多个参数，从JavaScript中大量传递参数，以及通过网络传递参数非常有用。

拼图参考

[anchor:create_empty_dict]

create empty dict(创建空字典)

返回一个不包含任何数据记录的字典。

在 JavaScript 中，一个 dictionary 是由一个没有任何属性的 *Object* 表示的，也就是 { } 。

[anchor:create_dict_with]

create dict with（使用...创建字典）

返回具有给定键和值的字典。

Keys/values 键值可用列表：

或多行文本，其中每行代表一个键或值：

[anchor:dict_set_key]

dict set key(在字典中设置键)

设置项目，使其与字典中的指定键相关联。这个键必须是一个文本，而分配的值可以是任何类型（文本、数字、列表、其他字典等）。

[anchor:dict_get_key]

dict get key(在字典中获取键)

返回与字典中指定键相关的项目。

[anchor:get_keys]

get keys(获取键的列表)

返回一个字典中所有键的列表。

[anchor:dict_check_key]

dict check key(在字典中检查键)

检查一个指定的键是否存在于一个字典中，并返回布尔类型的结果 *true* 或 *false*。

[anchor:dict_remove_key]

dict remove key(从字典中移除键)

删除与字典中指定键相关的项目，并删除键本身。

[anchor:is_empty]

is empty(为空)

检查指定的字典是否不包含键，并返回布尔类型的结果 *true* 或 *false*。
