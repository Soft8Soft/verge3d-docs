选择器

此类拼图是用来选择三维物体、组、动画片段和材质，以匹配于其他拼图。

拼图参考

[anchor:select_object]

select object(选择对象)

下拉菜单列表里按字母顺序排列了场景中的所有物体。

[anchor:all_objects]

all objects(所有对象)

表示场景中的所有物体，可用于执行批量操作。不可遍历。

[anchor:select_group]

select group(选择组)

该下拉菜单包含一个按字母顺序排列的场景中所有对象组的列表。对组的操作可被批量应用到组内全部对象上。不可遍历。

[anchor:select_animation]

select animation(选择动画)

下拉菜单包含一个按字母顺序排列的场景中的所有动画片断的列表。动画片段名是依照在Blender、3ds Max或Maya中应用动画的对象来命名的。

[anchor:select_material]

select material(选择材质)

该下拉列表包含了一个按字母顺序排列的场景中存在的所有材质的列表。

[anchor:get_entity_by_name]

按名称获取

选择由其名称指定的实体（对象、组、动画片段或材质）。

此拼图最常用于动态场景加载情境中，此时您无法在选择器的列表中找到被加载的实体的名称。
